Reports for Access Gateway Product Category : Net Capture Report : Reports by Destination and User
Reports by Destination and User
 
To view the following granular reports of a particular destination and user, go to Reports → Net Capture → Top Destinations → Destination → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Clientip
•
Top Liveips
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientip widget
Widget report displays number of connections per client IP by the selected destination and user.
 
View the report from Reports → Net Capture → Top Destinations → Destination → Top Users → User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of Top Destination Ports for selected client IP, user and destination, drill down by clicking the client IP in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected destination and user.
View the report from Reports → Net Capture → Top Destinations → Destination → Top Users → User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the list of Top Destination Ports for selected live IP, user and destination drill down by clicking the live IP address in the graph or hyperlink in the table.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected destination and user.
.
View the report from Reports → Net Capture → Top Destinations → Destination → Top Users → User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of Top Destination Ports for selected MAC address, destination and user, drill down by clicking the MAC address in the graph or hyperlink in the table.
