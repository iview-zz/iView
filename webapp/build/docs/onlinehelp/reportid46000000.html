Reports for UTM Product Category : Web Usage : Top Applications
Top Applications
Report displays list of top applications along with the number of connections that generate the most traffic for various domains, users, contents, URLs and categories.
 
View the report from Reports →Web Usage → Top Applications.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transferred per application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To defined an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular application drill down by clicking application in the graph or the application hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Web Users
•
Top Contents
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
