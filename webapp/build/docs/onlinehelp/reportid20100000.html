Reports for UTM Product Category : Application Based Usage Report : Top Applications
Top Applications
Report displays list of top applications along with the number of connections that generate the most traffic for various users, destinations and hosts.
 
View the report from Reports → Application Based Usage → Top Applications.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
Application Group: Displays name of the application group as defined in Cyberoam iView. If application group is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified applications and group unassigned applications please refer to Add Custom Application
•
Connections: Number of connections
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular application, drill down by clicking application name in the graph or the application name hyperlink in the table:
•
Top Users
•
Top Destinations
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
