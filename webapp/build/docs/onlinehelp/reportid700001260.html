Reports for Web Server Product Category : Apache : Top Content Types
Top Content Types
Report displays list of top content types along with the number of connections that generate the most traffic for various clients, servers, referrers, response codes, user agents and URLs.
 
View the report from Reports → Apache → Top Content Types.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transfer per content type while tabular report contains following information:
•
Content Type: Type of content
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transfer
To view the following granular reports of a particular content type, drill down by clicking content type in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
