Reports for UTM Product Category : Web Usage : Reports by Application, Category and Domain
Reports by Application, Category and Domain
To view the following granular reports of a particular application, category and domain, go to Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Domains widget and drill down by clicking the domain name in the graph or hyperlink in the table.
•
Top Users
•
Top Contents
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the domain, category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Domains widget→ Domain Name.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred through the user
 
To view the list of URLs for selected user, domain, category and application drill down by clicking the user name in the graph or hyperlink in the table:
Top Contents widget
Widget report displays amount of data transferred and number of connections content type by the domain, category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Domains widget→ Domain Name
 
Bar graph displays amount of data transferred by each content type while tabular report contains following information:
•
Content: Type of the content e.g. audio, video, text etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of users for selected content, domain, category and application drill down by clicking the user name in the graph or hyperlink in the table.
