Reports for Access Gateway Product Category : Net Capture Report : Reports by Clientip
Reports by Clientip
To view the following granular reports of a particular Destination, go to Reports → Net Capture → Top Clientips and drill down by clicking client IP address in the graph or the Destination hyperlink in the table:
•
Top Users
•
Top MAC Addresses
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user for the selected client IP.
 
View the report from Reports → Net Capture → Top Clientips → Client IP.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular client IP and user, go to Reports → Net Capture → Top Clientips → Clientip → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top MAC Addresses
•
Top Liveips
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP.
 
View the report from Reports → Net Capture → Top Clientips → Client IP.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of users for particular client IP and MAC address, go to Reports → Net Capture → Top Clientips → Clientip → Top MAC Address widget and drill down by clicking the MAC address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected client IP.
 
View the report from Reports → Net Capture → Top Clientips → Client IP.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections to Live IP address
 
To view the list of users for particular client IP and live IP, go to Reports → Net Capture → Top Clientips → Clientip → Top Liveips widget and drill down by clicking the live IP in the graph or hyperlink in the table.
