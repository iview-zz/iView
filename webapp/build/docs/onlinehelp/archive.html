System : Archives : Archive Files
Archive Files
Archive logs are collection of historical records, which are the initial line of forensic investigation. Cyberoam iView retains archive log data for the configured period. Data Retention period can be configured from the System → Configuration→ Data Management page. For further details refer to Data Management section
Use System → Archives → Archive Files page to view archived log files generated by Cyberoam iView:
 
Screen Components:
•
Date: Date of archive logs.
•
File Details: Cyberoam iView stores archived data for a specified day in four parts and every part contains data for 6 hours. This column displays file for individual part with respective data size. Cyberoam iView can display archive files for maximum of 15 days per page.
•
Total Size : Size of archive data available for the specified day.
•
Action: Action that can be performed on archived data.
o
Load: Load archived file from your local drive to the Cyberoam iView database.
o
Unload: Unload archived file from Cyberoam iView database.
o
Search: Perform a refined search based on multiple criteria.
o
Backup: Take backup of selected file on the machine on which iView is installed.
 
Use the System → Archives → Archive Files page to
•
Load Archive Files
•
Search in Archive Files
Load Archive Files
 
•
Super Admin or Admin privilege required to load archived file
 
1.
Go to System → Archives → Archive Files
2.
Select one or multiple network devices from the device drop down given at top left.
3.
Select date range from the given calendar
4.
Click OK to save the selection.
5.
Select check box against file(s) and click Load to upload the file in Cyberoam iView database. This process may take some time depending on the size of data
•
The check box will be disabled once the file is uploaded to the Cyberoam iView database
