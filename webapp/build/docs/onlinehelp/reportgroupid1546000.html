Reports by Application Group, User (Upload) and Application (Upload)
To view the following reports of a particular user, application group and application, go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Application (Upload) widget → Application.
•
Top Destination (Upload)
•
Top Host (Upload)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destination (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected user, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Application (Upload) widget → Application.
 
Bar graph displays amount of data uploaded to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
Top Host (Upload) widget
Widget report displays amount of data uploaded and number of connections per host by the selected user, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Application (Upload) widget → Application.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
