Reports for UTM Product Category : Web Usage : Reports by Category and User
Reports by Category and User
To view the following granular reports of a particular user and category, go to Reports → Web Usage → Top Categories → Category Name → Top Users widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Domains
•
Top Contents
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the selected user and category.
 
View the report from Reports → Web Usage→ Top Categories → Category Name → Top Users widget →User.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs from where the data is transferred through the selected domain, user and category drill down by clicking the domain name in the graph or hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the selected user and category.
 
View the report from Reports → Web Usage→ Top Categories → Category Name → Top Users widget → User.
 
Bar graph displays amount of data transferred by each contetnt type while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of domains from where the data is transferred through the selected content, user and category, drill down by clicking the content type in the graph or hyperlink in the table.
