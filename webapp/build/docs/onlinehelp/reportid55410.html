Reports for UTM Product Category : Mail Usage : Report by Application and Host
Report by Application and Host
Report displays amount of data transferred by selected host and application.
 
View the report from Reports → Mail Usage → Top Mail Applications →Application →Top Hosts → Host.
 
Bar graph displays amount of data transferred by each sender correspondeing to selected host and application while tabular report contains following information:
•
Time: Time in YYYY: MM: DD HH:MM:SS format
•
Sender: Email ID of the sender
•
Recipient: Colon (: ) separated email IDs of recipient(s)
•
Subject : Subject line of the email
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Destination: IP address of the destination
•
Size: Size of the email
