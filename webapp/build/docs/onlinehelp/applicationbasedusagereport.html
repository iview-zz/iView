Application Based Usage Report
Application based usage reports provide snapshot of application based traffic through your network . It helps identify high volume traffic generators who are affecting overall network traffic. It provides statistics based on traffic generated by applications.
 
These reports can help determining traffic flow issues and provide a basis for fine-tuning the configuration to better control traffic flow.
 
View application based usage reports from Reports → Application Based Usage.
 
It enables to view traffic generated by:
