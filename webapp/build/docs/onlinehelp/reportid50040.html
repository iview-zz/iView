Web Search Results by URL and Group
The report displays number of hits and amount of data transferred for the selected URL and group along with web site name and URL path.
1.
To view report go to Search → Web Surfing Reports
2.
Specify search parameters as below:
o
Search Type: URL
o
Search For: Group
o
Group Name
o
Domain/URL
 
Bar graph displays user group wise number of connections while tabular report contains following information:
•
Group Name: Group name of the user group as defined in the monitored device. If group is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user group
•
Site Name: URL of the website visited by the user group
•
URL Path: Complete URL path of the visited page. The complete URL path is available on tool tip also
•
Connections: Number of connections to the user group
•
Bytes Amount of data transferred
