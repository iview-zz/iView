Reports for UTM Product Category : Blocked Web Attempts : Top Denied Domains
Top Denied Domains
Report displays a list of top denied domains that various users tried to access and the number of connections over each domain.
 
View the report from Reports → Blocked Web Attempts →Top Denied Domains.
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Bar graph displays list of domains while tabular report contains following information:
•
Domain: Name of the domain
•
Connections: Number of Connections
 
To view the following granular reports under a particular domain, drill down by clicking the bar against domain name in the graph or the domain name hyperlink in the table:
•
Top Users
