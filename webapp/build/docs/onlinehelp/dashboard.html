Dashboards
Cyberoam iView displays Dashboard for UTM product category as soon as you logon to the Web Admin Console.
To view dashboard for other product category one needs to select product category from drop down provided on top left.
 
Dashboard provides a summary view of web and mail traffic including what is happening on the network, such as top attacks or top spammers.
 
It also provides the current resource usage - CPU, Disk, Memory as well total events received by Cyberoam iView from each device.
 
By default, Cyberoam iView gives following dashboards:
•
Main Dashboard: Provides overview of network traffic for all monitored devices of selected product category.
•
Device Dashboard : Provides overview of network traffic of selected UTM device
•
User Dashboard : Provides overview of Internet behavior of selected user
•
Source Host Dashboard: Provides overview of traffic generated by selected source host
•
Email Address Dashboard: Provides account of internet activities conducted by selected email address
•
Cyberoam iView Dashboard: Provides overview of all the important parameters of Cyberoam- iView
 
To return to the Main Dashboard from any other page of the Web Admin console, click “Home” link provided in Admin Tool bar.
 
Common Components
 
Navigation Menu
 
Navigation Pane on the leftmost side provides access to various configuration pages through various menu items.
To view page associate with menu, click menu item.
 
Admin Tool Bar
 
A bar that includes collection of following links provides access to most common and often used functions like:
•
Home: Click to return to Cyberoam iView main dashboard
•
Help: Click to access context sensitive online help
•
Feedback: Click to send feedback via email
•
Logout: Click to log out from Cyberoam iView
 
Date and Time Selection
 
Use option on top most left corner to select dashboard date and time. By default, current date’s dashboard will be displayed.
 
Button Bar
 
A bar that includes a collection of buttons provides an easy way to perform tasks like add or delete on clicking them.
Bar appears at the top left hand corner of the Information Area of every page.
