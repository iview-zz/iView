Reports by Application Group, User (Download) and Application (Download)
To view the following reports of a particular user, application group and application, go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Download) widget →User →Top Application (Download) widget → Application.
•
Top Destination (Download)
•
Top Host (Download)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destination (Download) widget
Widget report displays amount of data downloaded and number of connections per destination by the selected user, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Download) widget →User →Top Application (Download) widget → Application.
 
Bar graph displays amount of data downloaded to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data downloaded
Top Host (Download) widget
Widget report displays amount of data downloaded and number of connections per host by the selected user, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Download) widget →User →Top Application (Download) widget → Application.
 
Bar graph displays amount of data downloaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data downloaded
