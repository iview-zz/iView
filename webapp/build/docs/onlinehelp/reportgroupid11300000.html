Top Viruses widget
Report displays list of top viruses along with the number of connections.
View report from Reports →Virus →Web Viruses.
 
Report is displayed as graph as well as in tabular format.
 
Bar graph displays number of connections per virus while tabular report contains following information:
•
Virus: Name of the virus identified by monitored device
•
Connections: Number of connections to the virus
 
To view the following granular reports of a particular virus drill down by clicking virus name in the graph or the virus name hyperlink in the table:
•
Top Domains
•
Top Users
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Domains widget
Report displays list of top domains along with the number of connections.
View report from Reports →Virus →Web Viruses.
 
Report is displayed as graph as well as in tabular format.
 
Bar graph displays number of connections per domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
 
To view the following granular reports of a particular domain drill down by clicking domain name in the graph or the domain name hyperlink in the table:
•
Top Viruses
•
Top Users
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Users widget
Report displays list of top users along with the number of connections.
View report from Reports →Virus →Web Viruses.
 
Report is displayed as graph as well as in tabular format.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view the following granular reports of a particular user drill down by clicking user name in the graph or the user name hyperlink in the table:
•
Top Viruses
•
Top Domains
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
