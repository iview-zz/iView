Reports for UTM Product Category : Source Host Based Usage Report : Top Hosts
Top Hosts
Report displays list of top hosts along with the number of connections that generate the most traffic for various users, destinations and application group.
 
View the report from Reports → Source Host Based Usage → Top Hosts.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred by each application group while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user, drill down by clicking application group name in the graph or the application group hyperlink in the table:
•
Top Application Groups
•
Top Destinations
•
Top Users
•
Top Rules
•
Top Application Groups (Upload)
•
Top Application Groups (Download)
•
Top Destinations (Upload)
•
Top Destinations (Download)
•
Top Users (Upload)
•
Top Users (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
