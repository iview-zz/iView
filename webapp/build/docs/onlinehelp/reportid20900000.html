Top Hosts per Application Group (Download)
Report displays list of top hosts along with the application group and number of connections that generate the most traffic for various applications, destinations and users.
 
View the report from Reports → Application Based Usage → Top Hosts per Application Group (Download).
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
Host: IP address of the host
•
Application Group: Displays name of the application group as defined in Cyberoam iView. If application group is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified applications and group unassigned applications please refer to Add Custom Application
•
Connections: Number of connections to the host
•
Bytes: Amount of data downloaded
 
To view the following granular reports of a particular host, drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Applications (Download)
•
Top Destinations (Download)
•
Top Users (Download)
