Reports for Access Gateway Product Category : Web Surfing Logger Report : Reports by Site
Reports by Site
To view the following granular reports of a particular site, go to Reports → Web Surfing Logger → Top Sites and drill down by clicking URL name or the IP address in the graph or the site hyperlink in the table:
•
Top Users
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user for the selected site.
 
View the report from Reports → Web Surfing Logger → Top Sites → Site.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular site and user, go to Reports → Web Surfing Logger → Top Sites → Site → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Clientip
•
Top Liveips
•
Top MAC Address
Top Liveips widget
Widget report displays number of connections per live IP by the selected site.
 
View the report from Reports → Web Surfing Logger → Top Sites → Site.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections to Live IP address
 
To view the list of users for particular site and live IP, go to Reports → Web Surfing Logger → Top Sites → Site → Top Liveips widget and drill down by clicking the live IP in the graph or hyperlink in the table.
