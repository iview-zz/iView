Top Destinations by Destination Port, User, Client IP and MAC Address
Report displays a list of destinations along with the number of connections to each destination for the selected destination port, user, client IP and MAC address.
 
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Users widget→ User→Top Clientips widget→ Client IP →Top MAC Addresses widget→ MAC Address.
