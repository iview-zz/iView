Reports for Access Gateway Product Category : Net Capture Report : Reports by Destination Port, Destination and Live IP
Reports by Destination Port, Destination and Live IP
To view the following granular reports of a particular destination port, destination and live IP go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Liveips Widget and drill down by clicking the live IP in the graph or hyperlink in the table.
•
Top Clientips
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected destination, destination port live IP.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Liveips Widget→ Live IP.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of MAC addresses and list of users for selected client IP, live IP, destination port and destination drill down by clicking the live IP in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user by the selected destination port, destination and live IP.
.
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget→ Destination→ Top Liveips widget→ Live IP.
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the list of MAC addresses and list of client IPs for selected user, live IP, destination port and destination drill down by clicking the username in the graph or hyperlink in the table.
