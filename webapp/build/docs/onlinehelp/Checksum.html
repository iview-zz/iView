System : Configuration : Logs
Logs
•
Super Admin or Admin privilege required to access and manage Logs page.
 
To achieve compliance requirement of some geographical region, Cyberoam iView now provides MD5 sum for DHCP and Web Usage log files. It ensures integrity of log data, which means the log files are intact and log data is not manipulated.
Use System → Configuration → Logs to configure MD5 checksum generation for DHCP and Web Usage logs.
 
 
1.
Go to System → Configuration → Logs to configure MD5 checksum generation.
2.
Enable MD5 Checksum module for DHCP and Web Usage.
3.
Click Save to save changes.
