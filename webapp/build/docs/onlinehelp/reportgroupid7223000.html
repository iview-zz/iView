Reports for UTM Product Category : Blocked Attempts : Reports by Application Group, User and Host
Reports by Application Group, User and Host
To view the following reports of a particular host, application group and user, go to Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Users widget→ Top Hosts widget → Host.
•
Top Applications
•
Top Destinations
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected host, user and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Users widget→ Top Hosts widget → Host.
 
Bar graph displaysnumber of connections to each application while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
To view the list of destinations for the selected application, host, user and application group drill down by clicking the application in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected destination, user and application group.
 
View the report from Reports → Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Users widget→ Top Hosts widget → Host.
 
Bar graph displays number of connections to each destination while tabular report contains following information:
•
Destination: IP address of the Destination
•
Connections: Number of connections to the destination
 
To view the list of applications for the selected destination, host, user and application group drill down by clicking the destination in the graph or hyperlink in the table.
