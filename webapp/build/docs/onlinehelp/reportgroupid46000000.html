Reports for UTM Product Category : Web Usage : Reports by Application
Reports by Application
To view the following granular reports of a particular application, go to Reports → Web Usage → Top Applications and drill down by clicking application in the graph or the application hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Web Users
•
Top Contents
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories widget
Widget report displays number of connections and amount of data transferred per category for the selected application.
 
View the report from Reports →Web Usage → Top Applications →Application.
 
Bar graph displays number of connections and amount of data transferred per category whille tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view the following granular reports of each category drill down by clicking the category name in the graph or hyperlink in the table:
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected category and application
•
Top Users i.e. number of connections and amount of data transferred per user for the selected category and application
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected category and application
Top Domains widget
Widget report displays number of connections and amount of data transferred per domain for the selected application.
 
View the report from Reports →Web Usage → Top Applications →Application.
 
Bar graph displays number of connections and amount of data transferred per domain whille tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred
 
To view the following granular reports of each domain drill down by clicking the domain name in the graph or hyperlink in the table:
•
Top Users i.e. number of connections and amount of data transferred per user for the selected domain and application
•
Top Contents i.e. number of connections and amount of data transferred content wise by the selected domain and application
Top Web Users widget
Widget report displays number of connections and amount of data transferred per user for the selected application.
 
View the report from Reports →Web Usage → Top Applications →Application.
 
Bar graph displays number of connections and amount of data transferred per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the following granular reports of each user drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user and application
•
Top Domains i.e. number of connections and amount of data transferred per domain for the selected user and application
•
Top Contents i.e. number of connections and amount of data transferred content wise by the selected user and application
•
Top URLs i.e. number of connections and amount of data transferred per URL for the selected user and application
Top Contents widget
Widget report displays number of connections and amount of data transferred content wise for the selected application.
 
View the report from Reports →Web Usage → Top Applications →Application.
 
Bar graph displays number of connections and amount of data transferred content wise whille tabular report contains following information:
•
Content: Type of the content e.g. audio, video, text etc
•
Connections: Number of connections to the content
•
Bytes: Amount of data transferred
 
To view the following granular reports of each content type drill down by clicking the content type in the graph or hyperlink in the table:
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected content and application
•
Top Users i.e. number of connections and amount of data transferred per user by the selected content and application
•
Top Categories i.e. number of connections and amount of data transferred per category by the selected content and application
