Reports for UTM Product Category : Web Usage : Top Users by Category Type, Category, Domain and Content
Top Users by Category Type, Category, Domain and Content
Report displays a list of URLs, the number of connections to each URL and the amount of data transferred through the selected content, domain, category and by the category type.
 
View the report from Reports → Web Usage → Top Category Types → Category Type → Top Categories widget → Category → Top Domains → Domain → Top Contents widget → Content.
 
To view the list of URLs for the selected user, content, domain, category and category type drill down by clicking the user hyperlink in the table.
