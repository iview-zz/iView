Reports for UTM Product Category : FTP Usage : Top Files Downloaded via FTP
Top Files Downloaded via FTP
Report displays the list of top files uploaded via various FTP servers, hosts, and users.
 
View the report from Reports → FTP Usage →Top Files downloaded via FTP.
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of files downloaded via FTP while tabular report contains following information:
•
File: Name of the file which has been downloaded
•
Connections: Number of Connections
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular file, drill down by clicking the bar against file name in the graph or the file name hyperlink in the table:
•
Top Servers
•
Top FTP Hosts
•
Top FTP Users
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
