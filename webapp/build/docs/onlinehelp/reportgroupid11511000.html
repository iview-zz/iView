Reports by Virus (Upload) and Server
To view the following granular reports of a particular virus (upload) and server, go to Reports → Virus →FTP Viruses → Virus →Top Viruses (Upload) widget →Viruses →Top Servers widget and drill down by clicking server in the graph or the server hyperlink in the table:
•
Top Hosts
•
Top Files
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Hosts widget
Widget report displays number of connections per host for the selected virus and server.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Servers widget → Server.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view host details for the selected host, server and virus (upload), drill down by clicking the host in the graph or hyperlink in the table:
Top Files widget

View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Servers widget → Server.
 
Bar graph displays number of connections per file whille tabular report contains following information:
•
File: Name of the file
•
Connections: Number of connections to the file
 
To view file details for the selected file, server and virus (upload), drill down by clicking the file in the graph or hyperlink in the table:
Top Users widget
Widget report displays number of connections per user for the selected virus and server.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Servers widget → Server.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the file
