Reports for UTM Product Category : Web Usage : Reports by Category Type, Category and Domain
Reports by Category Type, Category and Domain
To view the following granular reports of a particular category type, category and domain go to Reports → Web Usage → Top Category Types → Category Type → Top Categories widget→ Category→ Top Domains and drill down by clicking domain hyperlink in the table:
•
Top Users
•
Top Contents
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections and amount of data transferred per user for the selected domain, category type and category.
 
View the report from Reports → Web Usage → Top Category Types → Category Type → Top Categories widget→ Category→ Top Domains → Domain.
 
Bar graph displays amount of data transferred per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the list of URLs for the selected user, domain, category and category type drill down by clicking the User hyperlink in the table.
Top Contents widget
Widget report displays number of connections and amount of data transferred content wise for the selected domain, category type and category.
 
View the report from Reports → Web Usage → Top Category Types → Category Type → Top Categories widget→ Category→ Top Domains → Domain
 
Bar graph displays amount of data transferred content wise while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of users for the selected content, domain, category and category type drill down by clicking the content type hyperlink in the table.
