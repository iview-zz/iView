Reports for UTM Product Category : Web Usage : Reports by Application, Content and User
Reports by Application, Content and User
To view the following granular reports of a particular application, content and user, go to Reports → Web Usage→ Top Applications → Application → Top Contents widget → Content Type → Top Users widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Domains
•
Top Categories
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the user, content and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Contents widget → Content Type → Top Users widget→ User.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs for selected domain, user, content, and application drill down by clicking the domain name in the graph or hyperlink in the table.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the user, content, and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Contents widget → Content Type → Top Users widget→ User.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the list of domains for selected category, user, contentand application drill down by clicking the category name in the graph or hyperlink in the table.
