Reports for EPS Product Category : USB Control
USB Control
 
USB control reports provide snapshot of USB usage in the organization. The administrator can view the in-out flow of data by USB devices. These reports are important to prevent data leakage in any organization.
USB control reports provide USB usage statistics based on users, USB devices, hosts and actions.
 
View USB control reports from Reports → USB Control.
It enables to view traffic generated by:
•
Top Host IP Address
•
Top Users
•
Action Taken
•
Top USB Serial Number
