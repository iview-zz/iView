Reports for UTM Product Category : Spam : Top Spam Senders
Top Spam Senders
Report displays list of top spam senders along with the number of connections that generate the most traffic for various spam recipients, users, destinations, source hosts and applications.
 
View the report from Reports →Spam→ Top Spam Senders.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per spam sender while tabular report contains following information:
•
Sender: Email ID of the sender
•
Connections: Number of connections to the sender
 
To view the following granular reports of a particular spam sender, drill down by clicking sender email in the graph or the sender email hyperlink in the table:
•
Top Spam Recipients
•
Top Source Hosts
•
Top Destinations
•
Top Users
•
Top Applications
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
