Reports for Web Server Product Category : Apache : Reports by Server and Content Type
Reports by Server and Content Type
To view the following granular reports of a particular server and content type go to Reports → Apache → Top Servers→ Server → Top Content Types widget and drill down by clicking the content type in the graph or the hyperlink in the table:
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Clients widget
Widget report displays number of connections and amount of data transfer per client by the selected server and content type.
 
View the report from Reports → Apache → Top Server → Server → Top Content Types widget → Content Type.
 
Bar graph displays amount of data transfer per client while tabular report contains following information:
•
Client IP: IP address of the client
•
Connections: Number of connections to the client
•
Bytes: Amount of data transfer
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected server and content type.
 
View the report from Reports → Apache → Top Server → Server → Top Content Types widget → Content Type.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Referrers widget
Widget report displays number of connections and amount of data transfer per referrer by the selected server and content type.
 
View the report from Reports → Apache → Top Server → Server → Top Content Types widget → Content Type.
 
Bar graph displays amount of data transfer per referrer while tabular report contains following information:
•
Referrer: URL of the referrrer
•
Connections: Number of connections to the referrer
•
Bytes: Amount of data transfer
 
To view the list of URLs for the selected referrer, content type and server, click the referrer in the graph or hyperlink in the table.

View the report from Reports → Apache → Top Server → Server → Top Content Types widget → Content Type.
 
Bar graph displays amount of data transfer per user agent while tabular report contains following information:
•
User Agent: URL of the user agent
•

Bytes: Amount of data transfer
 
To view the list of URLs for the selected user agent, content type and server, click the user agent in the graph or hyperlink in the table.
Top Response Codes widget
Widget report displays number of connections and amount of data transfer per response code by the selected server and content type.
 
View the report from Reports → Apache → Top Server → Server → Top Content Types widget → Content Type.
 
Bar graph displays amount of data transfer per response code while tabular report contains following information:
•
Response Code: Response code number
•
Connections: Number of connections to the response code
•
Bytes: Amount of data transfer
 
To view the list of URLs for the selected response code, content type and server click the response code in the graph or hyperlink in the table.
