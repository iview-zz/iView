Top Attackers by Attacks and Victims
Report displays top attackers who launched the attack with particular victim.
 
View the report from Reports → Attacks → Top Victims → Top Attacks → Attack.
 
The bar graph displays number of connections and IP address of the attackers while tabular report contains following information:
•
Attacker: IP address of the attacker
•
User: Username of the attacker. If username is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Application: Application through which attack is launched. If application is not identified by iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Severity: Severity of the attack by attacker
•
Action: Number of times action is performed
•
Connections: Number of Connections by the victim
