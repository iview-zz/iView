Reports for UTM Product Category : Firewall Rules Report : Accept Report –Destination Wise
Accept Report –Destination Wise
Report displays number of connections per host for selected destination.
 
View the report from Reports → Firewall Rules Based Based Usage → Top Accept Rules- Destination Wise→ Destination.
 
The bar graph displays amount of data transferred to the selected destination by various hosts. The table contains following information:
•
Host : IP address of the host
•
Application: Name of application. If application is not identified by Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
User: Username of the user. If User is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Connections: Number of connections per host
•
Bytes: Amount of data transferred by selected host to the selected destination by various hosts
