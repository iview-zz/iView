Reports by Virus (Upload) and User
To view the following granular reports of a particular virus (upload) and user, go to Reports → Virus →FTP Viruses → Virus →Top Viruses (Upload) widget →Viruses →Top Users widget and drill down by clicking username in the graph or the user hyperlink in the table:
•
Top Servers
•
Top Files
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections per server for the selected virus and user.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Users widget → User.
 
Bar graph displays number of connections per server whille tabular report contains following information:
•
Server: IP address of the server
•
Connections: Number of connections to the server
 
To view server details for the selected server, user and virus (upload), drill down by clicking the server in the graph or hyperlink in the table.
Top Files widget
Widget report displays number of connections per file for the selected virus and user.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Users widget → User.
 
Bar graph displays number of connections per file whille tabular report contains following information:

File: Name of the file

Connections: Number of connections to the file
 
To view file details for the selected file, user and virus (upload), drill down by clicking the file in the graph or hyperlink in the table.
