Reports for UTM Product Category : Virus : Report by Domain and User
Report by Domain and User
Report displays number of connections per URL for the selected user and domain.
 
View report from Reports → Virus → Web Viruses → Top Domains widget → Domain Name→ Top Users widget → User.
 
Bar graph displays number of connections per URL whille tabular report contains following information:
•
URL: URL name or IP address of URL
•
Virus: Name of the virus identified by monitored device
•
Host: IP address of the host
•
Connections: Number of connections to the URL
