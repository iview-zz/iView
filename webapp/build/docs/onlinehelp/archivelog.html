System : Archives : Archive Files : Load Archive Files
Search in Archive Files
•
Super Admin or Admin privilege required to search in archived file
•
Loading of appropriate archive file is required
 
1.
Go to System → Archives → Archive Files
2.
Click Search to perform search in uploaded archived file
3.
In Advanced Search section, select Match all of the following to get search result based on all criteria or Match any of the following to get search result based on any of the criterion
4.
Select criteria and enter respective values to perform search on archive logs.
Search Criteria for UTM
•
Protocol
•
Source
•
Destination
•
User
•
URL
•
Data Sent (in Bytes)
•
Data Received (in Bytes)
•
Rule
 
Search Result Format for UTM
•
Formatted Logs: Displays logs in comprehensive format
•
Raw Logs : Displays logs in syslog format
•
Date Time: Displays date and time for the log.
•
Device Name: Displays device name.
•
Firewall Rule ID: Displays firewall rule ID as configured in the device.
•
Username: Displays name of the user as defined in the device.
•
URL: Displays IP address or URL name accessed by the user.
•
Source IP: Displays source IP address.
•
Destination IP: Displays destination IP address
•
Protocol: Displays protocol number.
•
Sent Bytes: Displays number of bytes sent.
•
Received Bytes: Displays number of bytes received.

5.
Click Add Criteria to add new criteria to refine current search.
6.
Click Remove Criteria to remove criteria to refine current search
•
Blank fields in result show unavailability of the data.
Unload Archived Files
 
•
Super Admin or Admin privilege required to unload archived files
•
Loading of appropriate archive file is required
 
1.
Go to System → Archives → Archive Files
2.
Click Unload to unload all the loaded files.
•
Unload operation will unload all the loaded files. User will not have option to unload individual file.
•
Please note that unloading file does not delete the data from the Cyberoam iView.
