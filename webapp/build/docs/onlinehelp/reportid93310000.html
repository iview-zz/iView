Top Attacks by Applications and Attackers
Report displays top attacks launched by the attacker under particular application.
 
View the report from Reports → Attacks → Top Attackers → Top Applications → Application.
 
The bar graph displays number of connections and type of the attacks while tabular report contains following information:
•
Attack: Name of the attack
•
Victim: IP address of the victim
•
User: Username of the attacker. If username is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Severity: Severity of the attack by application
•
Action: Number of times action is performed
•
Connections: Number of Connections by the attacker
