Reports by File Download via FTP
To view the following granular reports of a particular file, go to Reports → FTP Usage → Files downloaded via FTP and drill down by clicking the bar against file name in the graph or the file hyperlink in the table:
•
Top Servers
•
Top FTP Hosts
•
Top FTP Users
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays amount of data downloaded through each server in the form of the selected file.
 
View the report from Reports → FTP Usage →Top Files downloaded via FTP → File.
 
Bar graph displays amount of data downloaded through server while tabular report contains following information:
•
Server: Server IP address through which the file is downloaded
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of hosts and users who downloaded the data to that particular server for the file, drill down by clicking the server in the graph or hyperlink in the table.
Top FTP Hosts widget
Widget report displays amount of data downloaded through each host in the form of the selected file.
 
View the report from Reports → FTP Usage →Top Files downloaded via FTP → File.
 
Bar graph displays amount of data downloaded by each host while tabular report contains following information:
•
Host: Hosts through which file is downloaded
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of servers and users who downloaded the data to that particular host and file, drill down by clicking the host in the graph or hyperlink in the table.
Top FTP Users widget
Widget report displays amount of data downloaded by users in form of that particular file.
 
View the report from Reports → FTP Usage →Top Files downloaded via FTP → File.
 
Bar graph displays amount of data downloaded by the users in the group while tabular report contains following information:
•
Users: Top users who have downloaded the file
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of hosts and servers who downloaded the data through that particular user for the file, drill down by clicking the user name in the graph or hyperlink in the table.
