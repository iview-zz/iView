Reports by Virus (Upload) and Host
To view the following granular reports of a particular virus (upload) and host, go to Reports → Virus →FTP Viruses → Virus →Top Viruses (Upload) widget →Viruses →Top Hosts widget and drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Servers
•
Top Files
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections per server for the selected virus and host.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Hosts widget → Host.
 
Bar graph displays number of connections per server whille tabular report contains following information:
•
Server: IP address of the server
•
Connections: Number of connections to the server
 
To view server details for the selected server, host and virus (upload), drill down by clicking the server in the graph or hyperlink in the table:
Top Files widget

View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Hosts widget → Host.
 
Bar graph displays number of connections per file whille tabular report contains following information:
•
File: Name of the file
•
Connections: Number of connections to the file
 
To view file details for the selected file, host and virus (upload), drill down by clicking the file in the graph or hyperlink in the table:
Top Users widget
Widget report displays number of connections per user for the selected virus and host.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Hosts widget → Host.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
