Mail Search Report
The report displays amount of data transferred for the selected protocol, user type, email address and subject line.
1.
To view report go to Search → Mail Usage
2.
Specify search parameters as below:
o
Protocol: SMTP/POP3/IMAP/Any
o
User Type: Recipient/Sender/Any
o
Email Address
o
Subject
 
Tabular report contains following information:
•
From: From email ID
•
To: To email ID
•
Subject: Subject line of the email
•
Protocol: Protocol name
•
Source IP: Source IP address of the email
•
Destination IP: Destination IP address of the email
•
Bytes: Amount of data transferred
