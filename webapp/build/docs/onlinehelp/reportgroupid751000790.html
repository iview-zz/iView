Reports for Access Gateway Product Category : Net Capture Report : Reports by Destination Ports
Reports by Destination Ports
 
To view the following granular reports of a destination port, go to Reports → Net Capture → Top Destination Ports and drill down by clicking destination port in the graph or the destination port hyperlink in the table:
•
Top Destinations
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destinations widget
Widget report displays number of connections per destination for the selected destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections
 
To view the following granular reports of a particular destination port and destination, go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destination widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Users
•
Top Clientips
•
Top Liveips
Top Users widget
Widget report displays number of connections per user by the destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port.
 
Bar graph displays connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular destination port and user, go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget and drill down by clicking the username in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
•
Top MAC Addresses
•
Top Destinations
