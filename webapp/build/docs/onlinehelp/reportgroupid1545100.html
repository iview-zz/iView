Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group, Application (Download) and Destination (Download)
Reports by Application Group, Application (Download) and Destination (Download)
To view the following reports of a particular destination, application group and application, go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications (Download) widget →Application →Top Destination (Download) widget →Destination.
•
Top User (Download)
•
Top Host (Download)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top User (Download) widget
Widget report displays amount of data downloaded and number of connections per user by the selected destination, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications (Download) widget →Application →Top Destination (Download) widget →Destination.
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data downloaded
Top Host (Download) widget
Widget report displays amount of data downloaded and number of connections per host by the selected destination, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications (Download) widget →Application →Top Destination (Download) widget →Destination.
 
Bar graph displays amount of data downloaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data downloaded
