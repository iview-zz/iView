Bookmark Management
 
Bookmark management allows the user to create bookmark of any Cyberoam iView report at any level of drill-down. It provides administrator with great level of network visibility based on any criterion. E.g. the administrator can monitor web usage of a particular user by creating bookmark of user based web usage report.
Every bookmark should be a part of a defined bookmark group and every bookmark can be sent to specified email address(s) in the form of report notification.
Use System → Configuration → Bookmark Management to create bookmark groups in Cyberoam iView.
 
Screen Components:
•
Bookmark Groups: Name of the bookmark group.
•
Add Button: Click to add a new bookmark group.
•
Delete Icon: Click to delete a bookmark group.
Use the System → Configuration → Bookmark Management to
•
Add Bookmark Group
•
Delete Bookmark Group
