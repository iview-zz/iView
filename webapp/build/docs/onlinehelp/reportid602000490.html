Reports for EPS Product Category : USB Control : Top USB Serial Number
Top USB Serial Number
Report displays list of top USB serial numbers along with the number of connections. Report can be drilled down for serial number to view top host IP addresses, top users and top USB serial numbers.
 
View the report from Reports → USB Control → Top USB Serial Number.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per serial number while tabular report contains following information:
•
Top USB Serial Number: Displays serial number of USB
•
Connections: Number of connections to the USB
 
To view the following granular reports of a serial number drilled down by serial number in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Top Users
•
Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
