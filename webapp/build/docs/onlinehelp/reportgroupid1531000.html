Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group and User
Reports by Application Group and User
To view the following granular reports of a particular application group and user, go to Reports → Source Host Based Usage → Top Application Group → Application Group → Top Users widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Applications
•
Top Destinations
•
Top Hosts
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays amount of data transferred and number of connections per application by the selected user and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group → Top Users widget→ User.
 
Bar graph displays amount of data transferred by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the list of destinations and list of hosts through which the data is transferred by the selected user, application and application group, drill down by clicking the application in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays amount of data transferred and number of connections per destination by the selected user and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users widget→User.
 
Bar graph displays amount of data transferred to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data transferred
 
To view the list of applications and list of hosts through which the data is transferred by the selected destination, user and application group, drill down by clicking the destination in the graph or hyperlink in the table.
Top Hosts Widget
Widget report displays amount of data transferred and number of connections per host by the selected user and application group.
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users widget →User.
 
Bar graph displays amount of data transferred through each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view the list of destinations and list of applications through which the data is transferred by the selected host, user and application group, drill down by clicking the host in the graph or hyperlink in the table.
