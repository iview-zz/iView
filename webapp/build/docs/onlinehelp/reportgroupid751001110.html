Reports by Destination Port, User and Destination
 
To view the following granular reports of a particular destination port, user and destination go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget → User → Top Destination widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected destination, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User→ Top Destinations widget→ Destination.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of live IPs for selected destination port, user, destination and client IP drill down by clicking the client IP in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected destination, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port→ Top Users widget→ User→ Top Destinations widget→ Destination.
 
To view the list of client IPs for selected destination port, user, destination and live IP drill down by clicking the live IP in the graph or hyperlink in the table.
