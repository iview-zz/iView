Reports for UTM Product Category : Source Host Based Usage Report : Top Rules
Top Rules
Top Rules report provides firewall rule wise traffic information from the firewall rule.
 
View the reports from Reports → Firewall Rules Based Usage → Top Rules.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
The bar graph displays amount of data transferred by each firewall rule while tabular report contains following information
•
Rule ID: Firewall rule ID as defined in the monitored device. If firewall rule ID is not defined in the monitored device then it will display firewall policy name.
•
Connections: Number of connections per firewall rule
•
Bytes: Amount of data transferred through firewall rule
 
To view detail Report for particular firewall rule, drill down by clicking the graph or the Rule ID hyperlink.
