Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group
Reports by Application Group
To view the following granular reports of a particular application group, go to Reports → Source Host Based Usage → Top Application Groups and drill down by clicking application group name in the graph or the application group hyperlink in the table:
•
Top Applications
•
Top Users
•
Top Destinations
•
Top Hosts
•
Top Rules
•
Top Applications (Upload)
•
Top Applications (Download)
•
Top Users (Upload)
•
Top Users (Download)
•
Top Destinations (Upload)
•
Top Destinations(Download)
•
Top Hosts(Upload)
•
Top Hosts(Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays amount of data transferred and number of connections per application of the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group.
 
Bar graph displays amount of data transferred by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the following granular reports of each application, drill down by clicking the application name in the graph or hyperlink in the table:
•
Top Users i.e. amount of data transferred and number of connections per user for selected application of the application group
•
Top Destinations i.e. amount of data transferred and number of connections per destination by the selected application of the application group
•
Top Hosts i.e. amount of data transferred and number of connections per host by the application of the application group
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Users widget
Widget report displays amount of data transferred and number of connections per user for the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group→ Application Group.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the following granular reports of each user, drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Applications i.e. amount of data transferred and number of connections per applications by the user for the selected application group
•
Top Destinations i.e. amount of data transferred and number of connections per destination by the user for the selected application group
•
Top Hosts i.e. amount of data transferred and number of connections per host by the user for the selected application group
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
