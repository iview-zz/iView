Reports for EPS Product Category : Web Report : Reports by Web Site
Reports by Web Site
To view the following granular reports of a particular web site, go to Reports → Web Report → Web Sites Visited and drill down by clicking URL in the graph or the hyperlink in the table:
•
Top Web Host
•
Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Web Host widget
Widget report displays number of connections per web host by the selected web site.
 
View the report from Reports →Web Report → Web Sites Visited→ Web Site.
 
Bar graph displays number of connections per web host while tabular report contains following information:
•
Web Host: IP address of the web host
•
Connections: Number of connections to the web host.
 
To view list of actions for the selected web host and web site drill down by clicking host in the graph or the hyperlink in the table.
Action Taken widget
Widget report displays number of connections per action by the selected web site.
View the report from Reports →Web Report → Web Sites Visited→ Web Site.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action: Action taken by the EPS product. Possible actions are
o
Allowed
o
Blocked
•
Connections: Number of connections to the action.
 
To view list of web hosts for the selected web site and action drill down by clicking action in the graph or the hyperlink in the table.
