Trend Reports : EPS Trend : File Antivirus
File Antivirus
 
File Antivirus trend report provides statistics based on events related to virus-infected files. The report displays number of events for each time interval along with event type.
 
View the report from Trend Reports → EPS Trend →File Antivirus.
 
Bar graph displays number of event per time interval while tabular report contains following information:
•
Time: Time of the event in YYYY-MM-DD HH:MM::SS format
•
Event Type: Type of event
•
Event: Number of events
