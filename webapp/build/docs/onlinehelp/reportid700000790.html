Reports for Web Server Product Category : Apache : Top Referrers
Top Referrers
 
Report displays list of top referrers along with the number of connections that generate the most traffic for various clients, servers, URLs, response codes, content types and user agents.
 
View the report from Reports → Apache → Top Referrers.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transfer per referrer while tabular report contains following information:
•
Referrer: URL of the referrer.
•
Connections: Number of connections
•
Bytes: Amount of data transfer
 
To view the following granular reports of a particular referrer, drill down by clicking referrer URL in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top URLs
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
