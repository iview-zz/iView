Reports by Firewall Rule
Report displays amount of data transfer by host for the selected firewall rule.
 
View the report from Reports → Firewall Rules Based Usage → Top Rules→ Rule ID.
The bar graph displays amount of data transferred by host through firewall rule while tabular report contains following information:
•
Host : IP address of the host
•
Application: Name of application. If application is not identified by iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
User: Username of the user. If User is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Destination: IP address of destination
•
Connections: Number of connections per host
•
Count: Number of times the rule has been applied
