Reports by Destination Port, Destination, Live IP and User
To view the following granular reports of a particular destination port, destination and live IP go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Liveips Widget→ Liveips → Top Users Widget and drill down by clicking the user in the graph or hyperlink in the table.
•
Top Clienips
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected destination, destination port, live IP and user.
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Destinations widget→ Destination→Top Liveips widget→ Live IP →Top Users widget→ User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected destination, destination port, live IP and user.
 
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Destinations widget→ Destination→Top Liveips widget→ Live IP →Top Users widget→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
