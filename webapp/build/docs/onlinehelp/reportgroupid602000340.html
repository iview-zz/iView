Reports for EPS Product Category : USB Control : Reports by Action Taken
Reports by Action Taken
To view the following granular reports of a particular action, go to Reports → USB Control → Action Taken and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Top Users
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected action.
 
View the report from Reports → USB Control → Action Taken→ Action.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the following granular reports for selected action and host, click the host IP address in the graph or hyperlink in the table.
•
Top Users
•
Top USB Serial Number
Top Users widget
Widget report displays number of connections per user by the selected action.
 
View the report from Reports → USB Control → Action Taken→ Action.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the following granular reports for selected user and host, click the user in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Top USB Serial Number
Top USB Serial Number widget
Widget report displays number of connection per USB serial number by the selected action.
 
View the report from Reports → USB Control → Action Taken→ Action.
 
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the following granular reports for selected USB serial number and action, click the USB serial number in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Top Users
