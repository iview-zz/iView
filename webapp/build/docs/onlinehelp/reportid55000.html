Reports for UTM Product Category : Mail Usage : Top Mail Applications
Top Mail Applications
Report displays list of top email applications along with the number of connections that generate the most traffic for various senders, recipients, users, destinations and hosts.
 
View the report from Reports →Mail Usage → Top Mail Applications.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application or group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
To view the following granular reports of a particular application, drill down by clicking application in the graph or the application hyperlink in the table:
•
Top Senders
•
Top Recipients
•
Top Users
•
Top Hosts
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
