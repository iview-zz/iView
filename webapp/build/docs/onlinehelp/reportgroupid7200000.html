Reports for UTM Product Category : Blocked Attempts : Reports by Denied Application Group
Reports by Denied Application Group
To view the following granular reports of a particular application group, go to Reports → Blocked Attempts → Top Denied Application Groups and drill down by clicking application group in the graph or the application group hyperlink in the table:
•
Top Applications
•
Top Users
•
Top Destinations
•
Top Hosts
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application for the selected application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups→ Application Group.
 
Bar graph displays number of connections per application whille tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the following granular reports of each application, drill down by clicking the application name in the graph or hyperlink in the table:
•
Top Users i.e. number of connections per users by the selected application and application group
•
Top Destinations i.e. number of connections per destination by the selected application and application group
•
Top Hosts i.e. number of connections per host by the selected application and application group
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
Top Users widget
Widget report displays number of connections per user for the selected application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups→ Application Group.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view the following granular reports of each host, drill down by clicking the host in the graph or hyperlink in the table:
•
Top Applications i.e. number of connections per application by the selected user and application group
•
Top Destinations i.e. number of connections per destination by the selected user and application group
•
Top Hosts i.e. number of connections per host by the selected user and application group
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
Top Destinations widget
Widget report displays number of connections per destination for the selected application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups→ Application Group.
 
Bar graph displays number of connections per destination whille tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
To view the following granular reports of each destination, drill down by clicking the destination in the graph or hyperlink in the table:
•
Top Applications i.e. number of connections per application by the selected application group and destination
•
Top Users i.e. number of connections per users by the selected destination and application group
•
Top Hosts i.e. number of connections per host by the selected destination and application group
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.

Connections: Number of connections to the host

Top Applications i.e. number of connections per application by the selected host and application group
•
Top Destinations i.e. number of connections per destination by the selected host and application group
•
Top Users i.e. number of connections per user by the selected host and application group
