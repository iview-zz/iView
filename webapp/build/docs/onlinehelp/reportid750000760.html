Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Sites by MAC Address, Clientip, User and Liveip
Top Sites by MAC Address, Clientip, User and Liveip
Report displays a list of web sites along with the number of hits to each website for the selected client IP, MAC address, user and live IP.
 
View the report from Reports → Web Surfing Logger → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users→ User → Top Liveips → Live IP.
 
To view the list of URLs for selected client IP, MAC address, user, live IP and site drill down by clicking the site name or IP address in the graph or hyperlink in the table.
