Reports for UTM Product Category : Firewall Rules Report
Firewall Rules Report
Firewall Rules report provides firewall rule wise traffic information from the firewall rule. It provides traffic statistics based on firewall rules associated with various users, hosts and applications.
 
View the Firewall Rules reports from Reports → Firewall Rule Based Usage
 
It enables to view traffic generated by:
•
Top Accept Rules
•
Top Deny Rule
•
Top Accept Rules- Application Group wise
•
Top Deny Rules- Application Group wise
•
Top Accept Rules- Host wise
•
Top Deny Rules- Host wise
•
Top Accept Rules- Destination wise
•
Top Deny Rules- Destination wise
