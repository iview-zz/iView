Reports for Web Server Product Category : Apache : Reports by Response Code
Reports by Response Code
To view the following granular reports of a particular response code, go to Reports → Apache → Top Response Codes and drill down by clicking response code in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected response code.
 
View the report from Reports → Apache → Top Response Codes → Response Code.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
To view the list of URLs and list of content types for the selected response code and server, click the server IP in the graph or hyperlink in the table
