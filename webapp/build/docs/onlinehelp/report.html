Reports
Common screen components to generate and view the reports:
 
•
Device Selection

Select one or multiple devices or device groups from the “Select Device” drop down given at top left corner of the screen.

Click OK to save the selection.
 
•
Date and Time Selection

Use to select report date and time. By default, current date’s report will be displayed.

Click Go to generate report for selected device(s) or device groups(s) for selected date and time.
 
•
Report View Customization

Select number of rows of the report to be displayed per page from the given drop down. A page can have minimum 5 and maximum 25 rows.

Click the toggle link ‘Hide Graph’ or ‘Hide Table’ as per requirement.

For more than 10 results, no graphs will be displayed.
 
•
Page Controls
Every report will display first page of the report along with total number of pages available for the report.
Use following controls to navigate report pages:

: Navigate to next page of report

: Navigate to previous page

: Navigate to the first page

: Navigate to the last page of the report
 
•
Reports Navigation
Cyberoam iView provides granular reports to get in-depth visibility of network activities. Top-level reports are the summary reports which can be drilled down to detailed reports.
 
Drilldown reports helps to investigate the information more deeply and view the Information with a different perspective.
 
Bar graph and Tabular report data provide the navigation hyperlink to view the drilldown reports. The drilldown reports page displays multiple reports in Widget format. Widget can be closed using by clicking button while minimized by clicking button.
 
You need to refresh the page to retrieve closed report widget.

•
DNS Resolver
Cyberoam iView comes with an inbuilt DNS query resolver to resolve an IP address for domain name because at times report only shows IP address of the host, domain or URL.
Click against IP address in report to get domain name for that IP address.
Local DNS of Cyberoam iView will be used to resolve the DNS query.
 
•
PDF Converter
Cyberoam iView allows exporting generated reports in PDF format. Click given at the top right of the report to export the report in to industry standard PDF format.
 
•
MS Excel Converter
Cyberoam iView allows exporting generated reports in MS Excel format. Click given at the top right of the report to export the report in to MS Excel format.
 
•
Bookmark
Cyberoam iView provides facility to create bookmark of a report at any level of drill-down.
Click given at the top right of the report to create bookmark of the report. The created bookmark will be displayed in left navigation menu under Bookmarks head.
