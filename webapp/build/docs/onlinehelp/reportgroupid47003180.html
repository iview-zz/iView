Reports for UTM Product Category : Web Usage : Reports by Category Type
Reports by Category Type
To view the following granular reports of a particular web category type, go to Reports → Web Usage → Top Category Types and drill down by clicking category type hyperlink in the table:
•
Top Categories
•
Top Users
•
Top Web User Groups
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays number of connections and amount of data transferred per category for the selected category type.
 
View the report from Reports → Web Usage → Top Category Types→ Category Type.
 
Bar graph displays amount of data transferred per category while tabular report contains following information:
•
Category: Name of the category as defined in monitored device
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view list of domains for the selected category and category type, drill down by clicking the category type hyperlink in the table.
Top Users widget
Widget report displays number of connections and amount of data transferred per user for the selected category type.
 
View the report from Reports → Web Usage → Top Category Types →Category Type.
 
Bar graph displays amount of data transferred per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view following granular reports for the selected user and category type, drill down by clicking the category type hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user and category type
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected user and category type
•
Top Contents i.e. number of connections and amount of data transferred per content type by the selected user and category type
Top Web User Groups widget
Widget report displays number of connections and amount of data transferred per user group for the selected category type.
 
View the report from Reports → Web Usage → Top Category Types →Category Type.
 
Bar graph displays amount of data transferred per user group while tabular report contains following information:
•
User Group: Name of the user group as defined in the monitored device. If the user group is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user group
•
Bytes: Amount of data transferred
 
To view list of users for the selected user group and category type, drill down by clicking the user group hyperlink in the table.
