Reports for Access Gateway Product Category : Web Surfing Logger Report : Reports by User
Reports by User
 
To view the following granular reports of a particular user, go to Reports → Web Surfing Logger → Top Users and drill down by clicking user name in the graph or the user name hyperlink in the table:
•
Top Sites
•
Top Clientips
•
Top Liveips
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Sites widget
Widget report displays number of connections per web site by the selected user.
 
View the report from Reports → Web Surfing Logger → Top Users → User.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
 
To view the list of Top URLs click the site name in the graph or hyperlink in the table.
Top Clientips widget
Widget report displays number of connections per client IP by the selected user.
 
View the report from Reports → Web Surfing Logger → Top Users → User.
 
Bar graph displays connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of Top Sites per client IP click the client IP in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected user.
 
View the report from Reports → Web Surfing Logger → Top Users → User.
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections to Live IP address
 
To view the list of Top Sites per Live IP click the live IP in the graph or hyperlink in the table.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected user.
View the report from Reports → Web Surfing Logger → Top Users → User.
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of Top Sites for selected user and MAC address, click the MAC address in the graph or hyperlink in the table.
