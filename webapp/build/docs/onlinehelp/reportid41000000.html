Reports for UTM Product Category : Web Usage : Top Web Users
Top Web Users
Report displays list of top web users along with the number of connections that generate the most traffic for various applications, hosts, destinations, domains and categories.
 
View the report from Reports →Web Usage → Top Web Users.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transferred per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user drill down by clicking User in the graph or the User hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
•
Top URLs
•
Top Hosts
•
Top Applications
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
