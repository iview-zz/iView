Reports for UTM Product Category : Source Host Based Usage Report : Reports by User and Destination
Reports by User and Destination
To view the following granular reports of a particular user and destination, go to Reports → Source Host Based Usage → Top users → User → Top Destinations widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Applications
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays amount of data transferred and number of connections per application by the selected user and destination.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Destination widget.
 
Bar graph displays amount of data transferred by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred through the application
 
To view the list of hosts from where the data is transferred through the selected application destination and user, drill down by clicking the application name in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays amount of data transferred and number of connections per host by the selected user and destination.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Destination widget.
 
Bar graph displays amount of data transferred by each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view the list of applications through which the data is transferred by the selected host destination and user, drill down by clicking the application name in the graph or hyperlink in the table.
