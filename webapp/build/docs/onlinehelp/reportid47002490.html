Reports for UTM Product Category : Web Usage : Top Web User Groups
Top Web User Groups
Report displays list of top web user groups along with the number of connections that generate the most traffic for various users, applications, hosts, contents, domains and categories.
 
View the report from Reports →Web Usage → Top Web User Groups.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred per user group while tabular report contains following information:
•
User Group: User Group name as defined in the monitored device. If the user group is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user group
•
Connections: Number of connections to the user group
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user drill down by clicking User Group hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
•
Top Users
•
Top URLs
•
Top Hosts
•
Top Applications
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
