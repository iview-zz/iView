Reports for EPS Product Category : eMail Scanning : Reports by Host
Reports by Host
To view the following granular reports of a particular host, go to Reports → eMail Scanning → Top Host and drill down by clicking host in the graph or the hyperlink in the table:
•
Top User
•
Top Action
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top User widget
Widget report displays number of connections per user by the selected host.
 
View the report from Reports → eMail Scanning → Top Host→ Host.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Windows login name of the user
•
Connections: Number of connections to the user.
 
To view list of actions for the selected host and user drill down by clicking user in the graph or the hyperlink in the table.
Top Action widget
Widget report displays number of connections per action by the selected host.
View the report from Reports → eMail Scanning → Top Host→ Host.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action: Action taken by the EPS product. Possible actions are
o
Allowed
o
Quarantined
o
Deleted
•
Connections: Number of connections to the action.
 
To view list of the users for the selected host and action drill down by clicking action in the graph or the hyperlink in the table.
