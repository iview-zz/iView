System : Configuration : Backup & Restore (Detail Data)
Backup & Restore (Detail Data)
•
Super Admin or Admin privilege required to access and manage Backup & Restore (Detail Data) page.
 
Cyberoam – iView allows the administrator to take scheduled backup of detailed report data on FTP server.
Use System → Configuration → Backup & Restore to configure scheduled backup of detailed report data which includes time stamp of any event, on FTP server.
 
Screen Components:
•
Backup Frequency: Frequency of taking backup. You can choose to take backup on daily basis or never.
•
FTP Server IP: IP address of the FTP server.
•
User Name: Username of FTP server.
•
Password: Password of FTP server.
•
Start Date: Start date to restore the backup.
•
End Date: End date to restore the backup.
 
Backup Detail Data:
 
1.
Go to System → Configuration → Backup & Restore (Detail Data) to configure backup frequency.
2.
Specify frequency of backup.
3.
Specify username and password of FTP server.
4.
Click Save to save changes.
 
Restore Detail Data:
 
1.
Go to System → Configuration → Backup & Restore (Detail Data)
2.
Use calendar control to select start and end date to restore the backup.
3.
Click Restore to restore the backup.
