Reports for EPS Product Category : Web Report : Top Web Host
Top Web Host
Report displays list of top hosts along with the number of connections. Report can be drilled down for host IP to view top web sites and top actions.
 
View the report from Reports → Web Report → Top Web Host.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the following granular reports for a web host, drill down by clicking host in the graph or the hyperlink in the table:
•
Web Sites Visited
•
Action Taken
