Reports for Access Gateway Product Category : Net Capture Report : Top Destination Ports by MAC Address, Clientip, User and Destination
Top Destination Ports by MAC Address, Clientip, User and Destination
Report displays a list of destination ports along with the number of hits to each port for selected client IP, MAC Address, user and destination.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users→ User → Top Destinations→ Destination.
