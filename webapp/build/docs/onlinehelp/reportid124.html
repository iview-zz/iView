Top Deny Rules – Host Wise
Top Deny Rules report enables to view traffic denied for various hosts.
 
View the report from Reports → Firewall Rules Based Usage → Top Deny Rules- Host Wise.
 
The bar graph displays number of connections per denied host while the tabular report contains following information:
•
Host: IP address of the host
•
Rule ID: Firewall rule ID as defined in the monitored device. If firewall rule ID is not defined in the monitored device then it will display firewall policy name.
•
Connections: Number of connections per host
 
To get in depth Deny Report- Host Wise visibility of a particular host, drill down by clicking the graph or the host hyperlink
