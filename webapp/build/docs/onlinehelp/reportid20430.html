Trend Reports : UTM Trend : Web Usage
Web Usage
 
Web Usage trend report provides statistics based on Internet usage. The report displays number of web events for each time interval.
 
View the report from Trend Reports → UTM Trend →Web Usage.
 
Bar graph displays number of events per time interval while tabular report contains following information:
•
Time: Time of the event in YYYY-MM-DD HH:MM::SS format
•
Event Type: Type of event
•
Event: Number of events
