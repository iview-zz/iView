Report by Application used for Spam and Recipient
Report displays amount of data transferred by the selected recipient and application.
 
View the report from Reports → Spam → Top Applications used for Spam →Application →Top Recipients widget → Recipient.
 
Bar graph displays number of connections per sender corresponding to selected recipient and application while tabular report contains following information:
•
Time: Time in YYYY-MM-DD HH:MM:SS format.
•
Sender: Email ID of the sender
•
Subject : Subject line of the email
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Host: IP address of the host
•
Destination: URL name or IP address of the destination
•
Count: Number of counts to the email
