Trend Reports : UTM Trend : Blocked Firewall
Blocked Firewall
 
Blocked Firewall trend report provides statistics based on blocked firewall events. The report displays number of blocked firewall events for each time interval.
 
View the report from Trend Reports → UTM Trend →Blocked Firewall.
 
Bar graph displays number of events per time interval while tabular report contains following information:
•
Time: Time of the event in YYYY-MM-DD HH:MM::SS format
•
Event Type: Type of event
•
Event: Number of events
