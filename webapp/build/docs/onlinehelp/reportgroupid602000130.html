Reports for EPS Product Category : USB Control : Reports by Host IP Address and USB Serial Number
Reports by Host IP Address and USB Serial Number
To view the following granular reports of a particular host IP address and USB serial number, go to Reports → USB Control → Top Host IP Addresses → Host IP Address → Top USB Serial Number widget and drill down by clicking serial number in the graph or the hyperlink in the table:
•
Top Users
•
Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user by the selected host and USB serial number.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Top USB Serial Number widget→ USB Serial Number.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the list of action taken for the selected host, user and USB serial number, click the user in the graph or hyperlink in the table.
Action Taken widget
Widget report displays number of connection per action by the selected host and USB serial number.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Top USB Serial Number widget→ USB Serial Number.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product.Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of users for the selected host, USB serial number and action, click the action in the graph or hyperlink in the table.
