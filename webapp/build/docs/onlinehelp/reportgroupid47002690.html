Reports for UTM Product Category : Web Usage : Reports by User Group, User and Host
Reports by User Group, User and Host
To view the following granular reports of a particular user group, user and host, go to Reports → Web Usage → Top Web User Groups → User Group → Top Users widget → User → Top Hosts widget and drill down by clicking the host hyperlink in the table.
•
Top Categories
•
Top Contents
•
Top Domains
•
Top URLs
•
Top Applications
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the selected user group, host and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users widget → User → Top Hosts widget → Host.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the list of domains from where the data is transferred through the selected category, host, user group and user, drill down by clicking the category name hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the selected host, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users widget → User → Top Hosts widget → Host.
 
Bar graph displays amount of data transferred by each content type while tabular report contains following information:
•
Content: Type of the content e.g. text, video, audio etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of domains from where the data is transferred through the selected content type, host, user group and user, drill down by clicking the content type hyperlink in the table.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the selected host, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users widget → User → Top Hosts widget → Host.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: Domain name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs from where the data is transferred through the selected domain, host, user group and user, drill down by clicking the domain name hyperlink in the table.
Top URLs widget
Widget report displays amount of data transferred and number of connections per URL by the selected host, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users widget → User → Top Hosts widget → Host.
 
Bar graph displays amount of data transferred by each URL while tabular report contains following information:
•
URL: URL name or IP address or the URL
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transferred through the URL
Top Applications widget
Widget report displays amount of data transferred and number of connections per application by the selected host, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users widget → User → Top Hosts widget → Host.
 
Bar graph displays amount of data transferred by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam-iView. If application is not defined in Cyberoam-iView then this field will display application identifier as combination of protocol and port number. To defined an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred through the application
 
To view the following granular report of each application drill down by clicking the application hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected application, user group, host and user
•
Top Contents i.e. number of connections and amount of data transferred per content type by the selected application, user group, host and user
