Reports for UTM Product Category : Source Host Based Usage Report : Top Users (Download)
Top Users (Download)
Report displays list of top users along with the number of connections that generate the most traffic for various applications groups, destinations and hosts.
 
View the report from Reports → Source Host Based Usage → Top Users (Download).
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the following granular reports of a particular user, drill down by clicking user name in the graph or the user name hyperlink in the table:
•
Top Application Groups (Download)
•
Top Destinations (Download)
•
Top Hosts (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
