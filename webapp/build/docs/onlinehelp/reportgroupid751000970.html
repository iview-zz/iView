Reports by Destination Port, User and Clientip
To view the following granular reports of a particular destination port, user and client IP go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget → User → Top Clientip widget and drill down by clicking the client IP in the graph or hyperlink in the table.
•
Top MAC Addresses
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User→ Top Clientips widget→ Client IP.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC address of the machine
•
Connections: Number of connections per MAC address
 
To view the list of destinations for selected destination port, user, client IP and MAC address drill down by clicking the MAC address in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected client IP, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User→ Top Clientips widget→ Client IP.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
