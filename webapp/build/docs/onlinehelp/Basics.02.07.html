Access Gateway Dashboard
Main Dashboard for Access Gateway product category provides the pattern of network events in graphical form. The main dashboard displays a graph between number of events and time of event. It helps Administrator to identify most traffic generating events of the network along with time of the event taken place. The peak in the graph represents highest traffic.
