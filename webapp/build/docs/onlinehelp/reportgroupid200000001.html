Top Tunnel VPN Users widget
Report displays list of top tunnel users along with amount of data transfer.
View report from Reports →SSL VPN →Top SSL VPN Users.
Report is displayed as graph as well as in tabular format.
 
Bar graph displays amount of data transfer per user while tabular report contains following information:
•
User name: Username of SSL VPN tunnel user
•
Data Transfer: Amount of data transfer in bytes
