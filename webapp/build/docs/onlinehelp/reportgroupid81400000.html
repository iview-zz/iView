Blocked Web Attempts Reports by Host and User
To view the following granular reports of a particular user and host, go to Reports → Blocked Web Attempts → Top Denied Web Users → Username → Hosts widget and drill down by clicking the Host name in the graph or hyperlink in the table.
•
Top Categories
•
Top Domains
•
Top URLs
•
Top Applications
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories Widget
Widget displays list of denied categories which a user has tried to access through the selected host, along with the number of connections.
View reportView the report from Reports → Blocked Web Attempts → Top Denied Web Users → User name → Top Hosts.
 
Bar graph displays various denied categories which a user has tried to access while tabular report contains following information:
•
Category: Displays the various denied web categories
•
Connections: Number of Connections
 
To view the list of domains which comes under a denied category, drill down by clicking the category in the graph or category hyperlink in the table.
 
Top Domains Widget
Widget displays list of denied domains which a user has tried to access through the selected host, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User name → Top Hosts.
 
Bar graph displays various denied domains which a user has tried to access while tabular report contains following information:
•
Domain: Displays the various denied web domains
•
Connections: Number of Connections
 
To view the list of URLs which comes under a denied domain, drill down by clicking the domain in the graph or domain hyperlink in the table.
Top URLs Widget
Widget displays list of URLs which the selected host for a user has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User name → Top Hosts.
 
Bar graph displays various denied URLs which the selected user has tried to access while tabular report contains following information:
•
URL: Name of URL
•
Connections: Number of Connections
Top Applications Widget
Widget displays list of denied applications which a user has tried to access through the selected host, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User name → Top Hosts.
 
Bar graph displays various denied applications which a user has tried to access while tabular report contains following information:
•
Applications: Displays the various denied applications
•
Connections: Number of Connections
 
To view the list of categories which comes under a denied application type, drill down by clicking the application in the graph or application hyperlink in the table.
