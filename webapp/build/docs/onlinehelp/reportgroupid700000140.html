Reports by Server and URL
 
To view the following granular reports of a particular server and URL go to Reports → Apache → Top Servers→ Server → Top URLs widget and drill down by clicking URL name in the graph or the hyperlink in the table:
•
Top User Agents
•
Top Content Types
•
Top Response Codes
•
Top Referrers
The granular reports page displays multiple reports in the widgets form.
 
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top User Agents widget
Widget report displays number of connections and amount of data transfer per user agent by the selected server and URL.
 
View the report from Reports → Apache → Top Server → Server → Top URLs widget → URL.
 
Bar graph displays amount of data transfer per user agent while tabular report contains following information:
•
User Agent: URL of the user agent
•
Connections: Number of connections to the user agent
•
Bytes: Amount of data transfer
