Reports by Application (Upload)
To view the following granular reports of a particular application, go to Reports → Application Based Usage → Top Applications (Upload) and drill down by clicking application name in the graph or the application name hyperlink in the table:
•
Top Users (Upload)
•
Top Destinations (Upload)
•
Top Hosts (Upload)
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users (Upload) widget
Widget report displays amount of data uploaded and number of connections per user for the selected application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application.
 
Bar graph displays amount of data uploaded by each application group while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data uploaded
 
To view the list of destinations and list of hosts through which the data is uploaded by the selected user and application, drill down by clicking the User in the graph or hyperlink in the table.
Top Destinations (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application.
 
Bar graph displays amount of data uploaded through each destination while tabular report contains following information:
•
Destination: Site URL or Destination IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
 
To view the list of users and list of hosts through which the data is uploaded by the selected destination and application, drill down by clicking the destination in the graph or hyperlink in the table.
Top Hosts (Upload) widget
Widget report displays amount of data uploaded and number of connections per host for the selected application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application.
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
 
To view the list of users and list of destinations through which the data is uploaded by the selected destination and application, drill down by clicking the host in the graph or hyperlink in the table.
