Reports for UTM Product Category : Internet Usage : Exclusive Report by Group
Exclusive Report by Group
Report displays list of user groups along with the amount of data transferred.
View the report from Reports → Internet Usage.
 
Bar graph displays user group wise data transfer while tabular report contains following information:
•
User Group: Name of the user group as defined in the monitored device. If the User Group is not defined in monitored device then it will be considered as traffic generated by ‘Unknown’ user group
•
Data Transfer: Total amount of data transferred (Upload + Download) by the user
•
Uploaded: Amount of uploaded data
•
Downloaded: Amount of downloaded data
 
To view detail report for the selected user group, drill down by clicking the User Group hyperlink in the table.
