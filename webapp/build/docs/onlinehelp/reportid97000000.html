Top Applications used by Attacks
Top Applications used by Attacks report enables to view the details of the top applications used by attacks that has hit the system and gives the detailed disintegration of attackers, victims and applications through individual reports. It also gives information about detected and dropped attacks under each application.
 
 
View the reports from Reports → Attacks → Top Applications used by Attacks.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
The bar graph displays the number of connections under for each application, while tabular report contains following information
•
Application: Name of the application under attack
•
Connections: Number of connections for each application
 
To view the following granular reports for each attack, drill down by clicking attack in the graph or attack name hyperlink in the table:
•
Top Attacks
•
Top Attackers
•
Top Victims
•
Top Detected Attacks
•
Top Dropped Attacks
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
