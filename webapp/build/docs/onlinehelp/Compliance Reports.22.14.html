Compliance Reports : GLBA : Top Attacks
Top Attacks
Widget report displays number of connections for every attack.
 
View the reports from Compliance Reports → GLBA→ Top Attacks.
Bar graph displays the type of attack while tabular report contains following information:
•
Attack: Name of the attack
•
Connections: Number of connections under the attack
 
For in depth reports refer to Reports by Attacks
