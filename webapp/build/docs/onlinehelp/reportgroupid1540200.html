Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group, Application and Host
Reports by Application Group, Application and Host
To view the following reports of a particular host, application group and application, go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications widget →Application →Top Hosts widget →Host.
•
Top Destinations
•
Top Users
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destinations widget
Widget report displays amount of data transferred and number of connections per destination by the selected host, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications widget →Application →Top Hosts widget →Host.
 
Bar graph displays amount of data transferred to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data transferred
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the selected host, application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications widget →Application →Top Hosts widget →Host.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
