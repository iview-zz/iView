Reports for UTM Product Category : FTP Usage : Reports by FTP Server
Reports by FTP Server
To view the following granular reports of a selected FTP Server, go to Reports → FTP Usage → Top FTP Servers and drill down by clicking the bar against in the graph or the server hyperlink in the table:
•
Top Files Uploaded
•
Top Files Downloaded
•
Top FTP Users
•
Top FTP Hosts
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files Uploaded widget
Widget report displays the list files uploaded through the selected server.
 
View the report from Reports → FTP Usage →Top FTP Servers → Server.
 
Bar graph displays amount of data uploaded through each file while tabular report contains following information:
•
File: Name of file which is uploaded through the selected server
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of hosts and users, which uploaded the data in form of files to a selected server, drill down by clicking the file name in the graph or hyperlink in the table.
Top Files Downloaded widget
Widget report displays the list of files downloaded through the selected server.
 
View the report from Reports → FTP Usage →Top FTP Servers → Server.
 
Bar graph displays amount of data downloaded through each file while tabular report contains following information:
•
File: Displays file name which is downloaded from that selected server
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of hosts and users which downloaded the data in form of files from a selected server, drill down by clicking the file name in the graph or hyperlink in the table.
Top FTP Users widget
Widget report displays list of users who transferred data through the selected server.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: User name who transferred data
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the granular upload and download for the selected user, drill down by clicking the bar in the graph or user hyperlink in the table.
Top FTP Hosts widget
Widget report displays list of hosts through which the data was transferred through the selected server.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server.
 
Bar graph displays amount of data transferred by each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the granular upload and download for the selected user, drill down by clicking the bar in the graph or user hyperlink in the table.
