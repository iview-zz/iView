Reports for UTM Product Category : Virus : Reports by Domain
Reports by Domain
To view the following granular reports of a particular domain, go to Reports → Virus →Web Viruses → Virus →Top Domains widget and drill down by clicking domain name in the graph or the domain name hyperlink in the table:
•
Top Viruses
•
Top Users
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Viruses widget
Widget report displays number of connections per virus for the selected domain.
 
View report from Reports → Virus →Web Viruses → Top Domains widget → Domain Name.
 
Bar graph displays number of connections per virus whille tabular report contains following information:
•
Virus: Name of the virus identified by monitored device
•
Connections: Number of connections to the virus
 
To view virus details for the selected virus and domain drill down by clicking the virus name in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user for the selected domain.
 
View report from Reports → Virus → Web Viruses → Top Domains widget → Domain Name.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view user details for the selected user and domain drill down by clicking the user name in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays number of connections per host for the selected domain.
 
View report from Reports → Virus → Web Viruses → Top Domains widget → Domain Name.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view host details for the selected host and domain drill down by clicking the host in the graph or hyperlink in the table.
