Reports for Access Gateway Product Category : Net Capture Report : Reports by MAC Address
Reports by MAC Address
To view the following granular reports of a MAC address, go to Reports → Net Capture → Top MAC Addresses and drill down by clicking MAC address in the graph or the MAC address hyperlink in the table:
•
Top Users
•
Top Clientips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user for the selected MAC address.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC Address.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular MAC address and user, go to Reports → Net Capture → Top MAC Addresses → MAC Address → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Destinations
•
Top Clientips
•
Top Liveips
Top Clientips widget
Widget report displays number of connections per client IP by the MAC address.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC Address.
 
Bar graph displays connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of users for particular client IP and MAC address, go to Reports → Net Capture → Top MAC Addresses → MAC Address → Top Clientip widget and drill down by clicking the client in the graph or hyperlink in the table.
