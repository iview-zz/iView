Reports by MAC Address, Clientip and User
To view the following granular reports of a particular MAC address, client IP and user, go to Reports → Web Surfing Logger → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Sites
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Sites widget
Widget report displays number of connections per web site by the selected client IP, MAC address and user.
.
View the report from Reports → Web Surfing Logger → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users → User.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
To view the list of URLs for selected MAC address, client IP, user and site, drill down by clicking the site name or IP address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected MAC address, client IP and user.
 
View the report from Reports → Web Surfing Logger → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users → User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: IP address of the client machine
•
Connections: Number of connections per live IP address
 
To view the list of sites for selected MAC address, client IP, user and Live IP, drill down by clicking the live IP in the graph or hyperlink in the table.
