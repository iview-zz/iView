Reports for UTM Product Category : Blocked Attempts : Reports by User and Application Group
Reports by User and Application Group
To view the following granular reports of a particular user and application group, go to Reports → Blocked Attempts → Top Denied Users→ User → Top Application Groups widget and drill down by clicking the application group name in the graph or hyperlink in the table.
•
Top Application
•
Top Destinations
•
Top Host
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected user and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User → User → Top Application Groups widget.
 
Bar graph displays number of connections per application in the group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of destinations for the selected application and user, drill down by clicking the application name in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected user and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User → User → Top Application Groups widget.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: Site name or IP address
•
Connections: Number of connections to the destination
 
To view the list of applications for by the selected destination, application group and user, drill down by clicking the destination in the graph or hyperlink in the table.
Top Hosts Widget
Widget report displays number of connections per host by the selected user and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User → Top Application Groups widget.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the destination
 
To view the list of applications for the selected host, application group and user, drill down by clicking the host in the graph or hyperlink in the table.
