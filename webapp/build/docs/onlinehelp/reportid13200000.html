Reports for UTM Product Category : IM Usage : Top User
Top User
Report displays list of top users along with total number of sent and received messages.
 
View the report from Reports →IM Usage → Top User.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date . Report date can be changed from the top most row of the page.
 
Bar graph displays total number of IM messages sent and received by each user while tabular report contains following information:
•
User: monitored device username of IM user
•
#Messages: Total number of IM messages sent and received by the user
 
To view the following granular reports of a particular user, drill down by clicking username hyperlink in the table:
•
Top Conversation
•
Top File Sent
•
Top File Received
•
Top Webcam Request Sent
•
Top Webcam Request Received
•
Top Protected Contact
•
Top Host
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
