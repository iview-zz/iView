Reports by Liveip and User
To view the following granular reports of a particular live IP and user, go to Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Sites
•
Top Clientips
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Sites widget
Widget report displays number of connections per web site by the selected live IP and user.
 
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget→ User.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
 
To view the list of URLs for the selected site, live IP and user, drill down by clicking the site name in the graph or hyperlink in the table.
Top Clientip widget
Widget report displays number of connections per client IP by the selected live IP and user.
 
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget→ User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of sites for selected client IP, live IP and user, drill down by clicking the client IP in the graph or hyperlink in the table.
Top MAC Addresses widget
Widget report displays number of connections MAC address by the selected live IP and user.
.
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of sites for selected MAC address, live IP and user, drill down by clicking the MAC address in the graph or hyperlink in the table.
