Reports by FTP Users (Upload)
To view the following granular reports of a particular user, go to Reports → FTP Usage → Top FTP Users (Upload) and drill down by clicking the bar against user name in the graph or the user hyperlink in the table:
•
Top Files
•
Top Hosts
•
Top Servers
•
 
•
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files widget
Widget report displays amount of data uploaded through various files by the selected user.
 
View the report from Reports → FTP Usage →Top FTP Users (Upload) → User.
 
Bar graph displays amount of data uploaded by each user through FTP protocl while tabular report contains following information:
•
File: Name of the file which is uploaded by the selected user
•
Connections: Number of connections established to upload the file
•
Bytes: Amount of data uploaded
 
To view the list of hosts and servers, which uploaded the data through that file for a selected user, drill down by clicking the file name in the graph or hyperlink in the table.
Top Servers widget
Widget report displays list of servers and amount of data uploaded through each server by the selected user.
 
View the report from Reports → FTP Usage → FTP Users (Upload) → User.
 
Bar graph displays amount of data uploaded by the users while tabular report contains following information:
•
Server: Server IP address used by a user to upload
•
Connections: Number of connections established to upload
•
Bytes: Amount of data uploaded
 
To view the list of hosts and files, which uploaded the data through that server for a selected user, drill down by clicking the bar in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays list of hosts and amount of data uploaded through each host by the selected user.
 
View the report from Reports → FTP Usage → FTP Users (Upload) → User.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: Host IP address through which user uploaded
•
Connections: Number of connections to upload
•
Bytes: Amount of data uploaded
 
To view the list of servers and files, which uploaded the data through that host for a selected user, drill down by clicking the bar in the graph or hyperlink in the table.
