System provides a number of configuration options to customize and use Cyberoam iView in your network after its successful installation.
 
The section describes how to add and configure devices that communicate with Cyberoam iView; how to add administrators and define their roles for device management; set data storage sizes for archiving logs; configure mail server and email schedule for mailing reports.
