Reports for UTM Product Category : Blocked Attempts : Reports by Application Group and Host
Reports by Application Group and Host
To view the following granular reports of a particular application group and host, go to Reports → Blocked Attempts → Top Denied Application Groups→ Application Group → Top Hosts widget and drill down by clicking the host in the graph or hyperlink in the table.
•
Top Applications
•
Top Destinations
•
Top Users
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected host and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Hosts widget→Host.
 
Bar graph displays number of connections to each application in the group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of users and list of destinations for the selected application, host and application group, drill down by clicking the application name in the graph or hyperlink in the table.
Top Destinations Widget
Widget report displays number of connections per destination by the selected host and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Hosts widget→ Host.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of applications and list of users for the selected destination, host and application group, drill down by clicking the destination in the graph or hyperlink in the table.
Top Users Widget
Widget report displays number of connections per user by the selected host and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Hosts widget→ Host.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view the list of applications and list of destinations for the selected user, host and application group drill down by clicking the host in the graph or hyperlink in the table.
