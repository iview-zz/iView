Reports for UTM Product Category : Blocked Attempts : Reports by Destination, Application Group and Host
Reports by Destination, Application Group and Host
To view the following reports of a destination, application group and application, go to Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Hosts widget →Host.
•
Top Applications
•
Top Users
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected destination, host and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Hosts widget → Host.
 
Bar graph displays number of connections to each application while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
Top Users widget
Widget report displays number of connections per user by the selected destination, host and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Hosts widget → Host.
 
Bar graph displays number of connections to each host while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the User
