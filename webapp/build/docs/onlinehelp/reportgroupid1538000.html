Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group and Destination (Upload)
Reports by Application Group and Destination (Upload)
To view the following granular reports of a particular application group and destination, go to Reports → Source Host Based Usage → Top Application Group → Application Group → Top Destinations (Upload) widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Application (Upload)
•
Top User (Upload)
•
Top Host (Upload)
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Upload) widget
Widget report displays amount of data uploaded and number of connections per application by the selected destination and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group → Top Destinations (Upload) widget → Destination.
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded
 
To view the list of users and list of hosts through which the data is uploaded by the selected destination, application and application group, drill down by clicking the application in the graph or hyperlink in the table.
Top User (Upload) widget
Widget report displays amount of data uploaded and number of connections per user by the selected destination and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group → Top Destinations (Upload) widget → Destination.
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
 
To view the list of applications and list of hosts through which the data is uploaded by the selected destination, user and application group, drill down by clicking the user name in the graph or hyperlink in the table.
Top Hosts (Upload) Widget
Widget report displays amount of data uploaded and number of connections per host by the selected destination and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group → Top Destinations (Upload) widget → Destination.
 
Bar graph displays amount of data uploaded through each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
 
To view the list of destinations and list of applications through which the data is uploaded by the selected host, destination and application group, drill down by clicking the host in the graph or hyperlink in the table.
