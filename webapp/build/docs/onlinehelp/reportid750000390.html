Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Clientips
Top Clientips
Report displays list of top client IPs along with the number of connections that generate the most traffic for various users, MAC addresses and live IPs.
 
View the report from Reports → Web Surfing Logger → Top Clientips.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per site while tabular report contains following information:
•
Clientip: IP address of the client machine
•
Connections: Number of connections to the client IP
 
To view the following granular reports of a particular client IP, drill down by clicking client IP or IP address in the graph or the site hyperlink in the table:
•
Top Users
•
Top MAC Addresses
•
Top Liveips
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
