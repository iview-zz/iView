Reports for UTM Product Category : Virus : Mail Viruses
Mail Viruses
Report displays list of top mail virus along with the number of connections for various users, host and applications.
 
View report from Reports → Virus → Mail Viruses → Virus.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per virus while tabular report contains following information:
•
Virus: Name of the virus identified by monitored device
•
Connections: Number of connections to the virus
 
To view the following granular reports of a particular virus drill down by clicking virus name in the graph or the virus name hyperlink in the table:
•
Top Senders (Email IDs)
•
Top Recipients (Email IDs)
•
Top Sender Hosts
•
Top Receiver Hosts
•
Top Applications
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
