Reports for UTM Product Category : Attacks : Reports by Attackers
Reports by Attackers
To view the following granular reports under particular attack; go to Reports → Attacks → Top Attackers and drill down by clicking attack name in the graph or the attack name hyperlink in the table:
•
Top Attacks
•
Top Victims
•
Top Applications
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Attacks widget
Widget report displays number of connections under every attack for the selected attacker.
 
View the reports from Reports → Attacks → Top Attackers → Top Attacks.
 
Bar graph displays the type of the attack, while tabular report contains following information:
•
Attack:Name of the attack
•
Connections: Number of connections under the attack
 
To view the list of victims by the attacker under particular attack, drill down by clicking the Attacker IP in the graph or hyperlink in the table.
Top Victims widget
Widget report displays number of connections for every victim under the selected attacker.
 
View the reports from Reports → Attacks → Top Attackers → Top Victims.
 
Bar graph displays the IP address of the victim, while tabular report contains following information:
•
Victim:IP address of the victim
•
Connections: Number of connections by the victim
 
To view the list of attacks, the victim has launched for particular attacker, drill down by clicking the victim IP in the graph or hyperlink in the table.
Top Applications widget
Widget report displays number of connections for every application under attack for the selected attacker.
 
View the reports from Reports → Attacks → Top Attackers → Top Applications.
 
Bar graph displays the name of the application used by the attacker, while tabular report contains following information:
•
Application: Name of the application
•
Connections: Number of connections for the application
 
To view the list of attacks, the application has launched under particular attacker, drill down by clicking the application name in the graph or hyperlink in the table.
