Reports for UTM Product Category : Virus : Reports by Virus (Upload)
Reports by Virus (Upload)
To view the following granular reports of a particular virus (upload), go to Reports → Virus →FTP Viruses → Virus →Top Viruses (Upload) widget and drill down by clicking virus name in the graph or the virus name hyperlink in the table:
•
Top Servers
•
Top Hosts
•
Top Users
•
Top Files
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections per server for the selected virus.
 
View report from Reports → Virus →FTP Viruses → Top Viruses (Upload) widget → Virus.
 
Bar graph displays number of connections per server whille tabular report contains following information:
•
Server: IP address of the server
•
Connections: Number of connections to the server
 
To view the following granular reports of each server, drill down by clicking the server in the graph or hyperlink in the table:
•
Top Hosts i.e. number of connections per host by the selected server and virus
•
Top Files i.e. number of connections per file by the selected server and virus
•
Top Users i.e. number of connections per user by the selected server and virus
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.

Top Users widget
 
Widget report displays number of connections per user for the selected virus.
 
View report from Reports → Virus →FTP Viruses → Top Viruses (Upload) widget → Virus.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
