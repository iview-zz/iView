Reports for UTM Product Category : Web Usage : Top Categories
Top Categories
Report displays list of top categories along with the number of connections that generate the most traffic for various domains, users and contents.
 
View the report from Reports → Web Usage → Top Categories.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transferred per category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user drill down by clicking category name in the graph or the category hyperlink in the table:
•
Top Domains
•
Top Users
•
Top Contents
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
