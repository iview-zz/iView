Report by Virus and Domain
Report displays number of connections per URL for the selected domain and virus.
 
View report from Reports → Virus →Web Viruses → Top Viruses widget → Virus → Top Domains widget → Domain Name.
 
Bar graph displays number of connections per URL whille tabular report contains following information:
•
URL: URL name or IP address of URL
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Host: IP address of the host
•
Connections: Number of connections to the URL
