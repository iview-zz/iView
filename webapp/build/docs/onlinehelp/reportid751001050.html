Reports for Access Gateway Product Category : Net Capture Report : Top Clientips by Destination Port, User, Live IP and Destination
Top Clientips by Destination Port, User, Live IP and Destination
Report displays a list of client IPs along with the number of connections to each client IP for the selected destination port, user, live IP and destination.
 
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Users widget→ User→Top Liveips widget→ Live IP →Top Destinations widget→ Destination.
 
To view the list of MAC addresses for selected destination port, user, live IP, destination and client IP drill down by clicking the client IP in the graph or hyperlink in the table.
