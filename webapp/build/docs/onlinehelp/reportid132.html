Reports for UTM Product Category : Firewall Rules Report : Deny Report- Host Wise
Deny Report- Host Wise
Report displays number of connections per destination for selected host.
 
Go to Reports → Firewall Rules Based Usage → Top Deny Rules- Host Wise→ Host.
 
The bar graph displays number of connection for denied destinations of the selected host while the tabular report contains following information:
•
Destination: IP address of destination
•
Application: Name of application. If application is not identified by Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
User: Username of the user. If User is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Connections: Number of connections per destination
