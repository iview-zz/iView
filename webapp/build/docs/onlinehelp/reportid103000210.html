Reports for UTM Product Category : IM Usage : Report by Host and Conversation
Report by Host and Conversation
Report displays full conversation with peer contact through the selected host.
 
View the report from Reports → IM Usage → Top Host → Host → Top Conversation Widget → Peer Contact.
 
Tabular report contains following information:
•
Peer Contact : IM username of the peer contact
•
Conversation: Complete conversation between protected contact and peer contact
•
To view full conversation you need to select Logging Level as ‘Full Data’ at the time of conversation rule creation in monitored device.
