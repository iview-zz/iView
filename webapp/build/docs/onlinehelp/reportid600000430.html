Reports for EPS Product Category : File Antivirus : Top Host
Top Host
Report displays list of hosts along with the number of connections. Report can be drilled down for hosts to view top application, top actions and top path of application.
 
View the report from Reports → File Antivirus → Top Host.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host : Displays IP address of the host
•
Connections: Number of connections to the host
 
To view the following granular reports of a host, drill down by clicking host IP in the graph or the hyperlink in the table:
•
Top Action Taken
•
Top Application
•
Top Path of Application
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
