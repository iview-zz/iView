Reports for UTM Product Category : Web Usage : Reports by User Group, User and Application
Reports by User Group, User and Application
To view the following granular reports of a particular user group, user and application, go to Reports → Web Usage → Top Web User Groups → User Group → Top Users widget → User → Top Hosts widget → Host→ Top Application widget and drill down by clicking the application hyperlink in the table.
•
Top Categories
•
Top Contents
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the selected application, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users → User → Top Application widget→ Application
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Name of the category
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the list of domains for the selected category, application, user group and user, drill down by clicking the category hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections per content type by the selected application, user group and user.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group→ Top Users → User → Top Application widget→ Application
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Content: Type of content
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of domains for the selected content, application, host, user group and user, drill down by clicking the content hyperlink in the table.
