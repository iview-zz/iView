Reports for UTM Product Category : Web Usage : Reports by Application and Category
Reports by Application and Category
To view the following granular reports of a particular application and category, go to Reports → Web Usage→ Top Applications → Application → Top Categories widget and drill down by clicking the category name in the graph or hyperlink in the table.
•
Top Domains
•
Top Users
•
Top Contents
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget→ Category Name.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the following granular reports of each domain drill down by clicking the domain name in the graph or hyperlink in the table:
•
Top Users i.e. number of connections and amount of data transferred per user for the selected domain, category and application
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected domain, category and application
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget→ Category Name.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred through the user
 
To view the following granular reports of each user drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Domain i.e. number of connections and amount of data transferred per domain for the selected user, category and application
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected user, category and application
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget→ Category Name.
 
Bar graph displays amount of data transferred by each content type while tabular report contains following information:
•
Content: Type of the content e.g. audio, video, text etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the following granular reports of each content type drill down by clicking the content type in the graph or hyperlink in the table:
•
Top Users i.e. number of connections and amount of data transferred per user for the selected content, category and application
•
Top Domains i.e. number of connections and amount of data transferred per domain for the selected content, category and application
