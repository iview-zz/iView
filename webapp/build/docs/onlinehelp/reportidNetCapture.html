Reports for Access Gateway Product Category : Net Capture Report
Net Capture Report
 
Net Capture reports provide snapshot of traffic generated by various applications and services. It helps identify highest traffic generator applications and services, which are affecting overall network traffic. It provides statistics based on traffic generated by network usage.
 
These reports can help determining network behavior and provide a basis for fine-tuning the configuration to better control traffic flow.
 
View Net Capture reports from Reports → Net Capture.
 
It enables to view traffic generated by:
•
Top Users
•
Top Liveips
•
Top Destinations
•
Top Clientips
•
Top MAC Addresses
•
Top Destination Ports
