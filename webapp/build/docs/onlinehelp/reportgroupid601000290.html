Reports for EPS Product Category : Application Control : Reports by File Name and Program
Reports by File Name and Program
To view the following granular reports of a particular file name and program, go to Reports → Application Control → Top File Name → File Name → Top Program widget and drill down by clicking program in the graph or the hyperlink in the table:
•
Top Host IP Address
•
Top Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Address widget
Widget report displays number of connections per host by the selected file name and program.
 
View the report from Reports → Application Control → Top File Names→ File Name→ Top Program widget→ Program.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the list of actions for the selected host, file name and program; click the host IP address in the graph or hyperlink in the table.
Top Action Taken widget
Widget report displays number of connections per action taken by the selected program and file.
View the report from Reports → Application Control → Top File Names→ File Name→ Top Program widget→ Program.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product. Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of host IP Address for the selected program, file name and action; click the action in the graph or hyperlink in the table.
