Reports for UTM Product Category : Web Usage : Top Contents
Top Contents
Report displays list of top content types along with the number of connections that generate the most traffic for various domains, users and categories.
 
View the report from Reports →Web Usage → Top Contents.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transferred content wise while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular content type drill down by clicking content type in the graph or the content type hyperlink in the table:
•
Top Domains
•
Top Users
•
Top Categories
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
