System : Configuration : Manual Purge
Manual Purge
Retention of data and log archives use enormous amount of disk space. You can configure the log retention period from Data Management. Based on the configured retention period data will be automatically deleted on day-by-day basis.
Use System → Configuration → Manual Purge page to delete logs manually.
Screen Components:
•
Logs Category: Log type
•
Detail: Select date to purge detailed reports
•
Summary: Select date to purge summary reports
•
Purge Button: Click to purge the logs manually
Purge Logs
1.
Go to System → Configuration → Manual Purge.
2.
Select log category to purge the logs.
3.
Specify date for which logs have to be purged.
4.
Click Purge.
