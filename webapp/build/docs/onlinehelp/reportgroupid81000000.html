Blocked Web Attempts Reports by User
To view the granular reports of selected user, go to Reports → Blocked Web Attempts Reports → Top Denied Web Users and drill down by clicking the bar against user in the graph or user hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top URLs
•
Top Hosts
•
Applications
 
The granular reports page displays multiple reports in the widgets form which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories Widget
Widget displays list of denied categories which the selected user has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User.
 
Bar graph displays various denied categories which a user has tried to access while tabular report contains following information:
•
Category: Web category name
•
Connections: Number of Connections
 
To view the list of domains which are falling under the denied category, drill down by clicking the category in the graph or category hyperlink in the table.
Top Domains Widget
Widget displays list of denied domains which the selected user has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User.
 
Bar graph displays various denied domains which the selected user has tried to access while tabular report contains following information:
•
Domain: Name of denied domain
•
Connections: Number of Connections
 
To view the list of URLs which are falling under the denied domain, drill down by clicking the domain in the graph or domain hyperlink in the table.
Top URLs Widget
Widget displays list of URLs which the selected user has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User.
 
Bar graph displays various denied URLs which the selected user has tried to access while tabular report contains following information:
•
URL: Name of URL
•
Connections: Number of Connections
Top Hosts Widget
Widget displays list of hosts through which a user has tried to access denied sites, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User.
 
Bar graph displays various hosts through which a user has tried to access denied sites while tabular report contains following information:
•
Host: Displays the host name
•
Connections: Number of Connections
 
To view the following granular reports of each host, drill down by clicking the Host name link in the graph or host name hyperlink in the table:
 
•
Top Categories i.e. list of denied categories a user tried to access through a host.
•
Top Domains i.e. list of denied domains a user tried to access through a host.
•
Top URLs i.e. the list of URLs a user tried to access through a host.
•
Top Applications i.e. the list of URLs a user tried to access through a host.
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
Top Applications Widget
Widget displays list of denied applications which a user has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Users → User.
 
Bar graph displays various denied applications which a user has tried to access while tabular report contains following information:
•
Applications: Displays the various denied applications
•
Connections: Number of Connections
 
To view the list of categories, which comes under a denied application type, drill down by clicking the application in the graph or application hyperlink in the table.
