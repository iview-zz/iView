Reports for UTM Product Category : Source Host Based Usage Report : Reports by User and Hosts (Upload)
Reports by User and Hosts (Upload)
To view the following granular reports of a particular user and host uploads, go to Reports → Source Host Based Usage → Top users → User → Top Hosts (Upload) widget and drill down by clicking the host name in the graph or hyperlink in the table.
•
Top Application (Upload)
•
Top Destination (Upload)
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications (Upload) widget
Widget report displays amount of data uploaded and number of connections per application by the selected user and host.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Hosts (Upload) widget→Host.
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded through the application
 
To view the list of destinations from where the data is uploaded through the selected application, host and user, drill down by clicking the application name in the graph or hyperlink in the table.
Top Destinations (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected user and host.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Host (Upload) widget→Host.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Destination: Destination IP address
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded through the application
To view the list of applications through which the data is uploaded by the selected destination, host and user, drill down by clicking the application name in the graph or hyperlink in the table.
