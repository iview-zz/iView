Reports for UTM Product Category : Virus : Report by Virus and User
Report by Virus and User
Report displays number of connections per URL for the selected user and virus.
 
View report from Reports → Virus → Web Viruses → Top Viruses widget → Virus → Top Users widget → User.
 
Bar graph displays number of connections per URL whille tabular report contains following information:
•
URL: URL name or IP address of URL
•
Host: IP address of the host
•
Connections: Number of connections to the URL
