Reports for UTM Product Category : FTP Usage : Report by FTP Server and User
Report by FTP Server and User
To view the following granular reports of a selected user, go to Reports → FTP Usage → Top FTP Servers → Server → Top FTP Users widget and drill down by clicking the bar against in the graph or the user hyperlink in the table:
•
Top Files Uploaded
•
Top Files Downloaded
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files Uploaded by User and Server
Report displays a list of files uploaded by the selected user through the selected server. It also displays number of connections established to upload the file and the amount of data uploaded.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server → Top FTP Users widget → User → Top Files Uploaded widget.
 
Top Files Downloaded by User and Server
Report displays a list of files downloaded by the selected user through the selected server. It also displays number of connections established to upload the file and the amount of data downloaded.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server →Top FTP Users widget → User →Top Files Downloaded widget.
