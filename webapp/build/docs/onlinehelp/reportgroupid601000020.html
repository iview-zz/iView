Reports for EPS Product Category : Application Control : Reports by Host IP Address
Reports by Host IP Address
To view the following granular reports of a particular host, go to Reports → Application Control → Top Host IP Address and drill down by clicking host IP in the graph or the hyperlink in the table:
•
Top File Names
•
Top Action Taken
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top File Names widget
Widget report displays number of connections per file by the selected host.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address.
 
Bar graph displays number of connections per file while tabular report contains following information:
•
File Names : Name of the file
•
Connections: Number of connections to the file
 
To view the following granular reports for selected host and file, click the file name in the graph or hyperlink in the table.
•
Top Action Taken
•
Top Program
Top Action Taken widget
Widget report displays number of connections per action taken by the selected host.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS. Possible actions are :
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and host, click the action taken in the graph or hyperlink in the table.
•
Top File Names
•
Top Program
Top Program widget
Widget report displays number of connections per program by the selected host.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address.
 
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the following granular reports for selected action and application, click the action taken in the graph or hyperlink in the table.
•
Top File Names
•
Top Action Taken
