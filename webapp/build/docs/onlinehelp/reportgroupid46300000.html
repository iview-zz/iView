Reports by Application and Web User
To view the following granular reports of a particular application and domain, go to Reports → Web Usage→ Top Applications → Application → Top Web Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Categories
•
Top Domains
•
Top Contents
•
Top URLs
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the user and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Web Users widget → User.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view list of domains for each category drill down by clicking the category name in the graph or hyperlink in the table:
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the user and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Web Users widget→ User.
 
Bar graph displays amount of data transferred per domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs for each domain drill down by clicking the domain name in the graph or hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the user and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Web Users widget→ User.
 
Bar graph displays amount of data transferred content wise while tabular report contains following information:
•
Content: Type of content e.g. audio, video, text etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of domains for each content type drill down by clicking the content type in the graph or hyperlink in the table.
Top URLs widget
Widget report displays amount of data transferred and number of connections per URL by the user and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Web Users widget→ User.
 
Bar graph displays amount of data transferred per URL while tabular report contains following information:
•
URL: Name or IP address of the URL
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transferred through the URL
