Reports for Access Gateway Product Category : Net Capture Report : Top Destinations
Top Destinations
Report displays list of top destinations along with the number of connections that generate the most traffic for various users and live IPs.
 
View the report from Reports → Net Capture → Top Destinations.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per site while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the following granular reports of a particular destination, drill down by clicking destination IP address in the graph or the destination hyperlink in the table:
•
Top Users
•
Top Liveips
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
