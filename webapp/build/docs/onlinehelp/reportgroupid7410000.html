Reports for UTM Product Category : Blocked Attempts : Reports by Destination and Application Group
Reports by Destination and Application Group
To view the following granular reports of a particular destination and application group, go to Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Application Groups widget and drill down by clicking the application group in the graph or hyperlink in the table.
•
Top Applications
•
Top Users
•
Top Hosts
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected destination and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Application Groups widget → Application Group.
 
Bar graph displays number of connections to each application in the group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of users and list of hosts for the selected application, application group and destination, drill down by clicking the application in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user by the selected destination and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Application Groups widget → Application Group.
 
Bar graph displays number of connections to each application in the group while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view the list of applications and list of hosts for the selected user, application group and destination, drill down by clicking the User in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays number of connections per host by the selected destination and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Application Groups widget → Application Group.
 
Bar graph displays number of connections to each host in the group while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view the list of applications and list of users for the selected host, application group and destination, drill down by clicking the host in the graph or hyperlink in the table.
