Reports by Clientip, MAC Address and User
To view the following granular reports of a particular client IP, MAC address and user, go to Reports → Net Capture → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Destinations
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destinations widget
Widget report displays number of connections per destination by the selected client IP, MAC address and user.
.
View the report from Reports → Net Capture → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users→ User.
 
Bar graph displays number of connections per Destination while tabular report contains following information:
•
Destinations: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of destination ports for selected client IP, MAC address, user and destination, drill down by clicking the destination IP address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected client IP, MAC address and user.
 
View the report from Reports → Net Capture → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users→ User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: IP address of the client machine
•
Connections: Number of connections per live IP address
 
To view the list of destinations for selected client IP, MAC address, user and Live IP, drill down by clicking the live IP in the graph or hyperlink in the table.
