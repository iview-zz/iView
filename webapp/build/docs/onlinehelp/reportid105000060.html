Date wise Break-up by User
Report displays date wise break-up of total amount of transferred data along with used time, uploaded data and downloaded data.
 
View the report from Reports → Internet Usage→ Top Users widget → User Name.
 
Bar graph displays date wise data transfer while tabular report contains following information:
•
Date: Time stamp in YYYY-MM-DD HH:MM:SS format
•
Data Transfer: Total amount of data transferred (Upload + Download) by the user
•
Uploaded: Amount of uploaded data
•
Downloaded: Amount of downloaded data
•
Used Time: Time required in data transfer in HH:MM:SS format
 
To view detail report for the selected date, user and user group, drill down by clicking the Date Name hyperlink in the table.
