Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Sites by Clientip, User and MAC Address
Top Sites by Clientip, User and MAC Address
Report displays number of connections per web site by the selected client IP, user and MAC address.
 
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top Users widget → User → Top MAC Address widget → MAC address.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: Displays URL name of the web sites
•
Connections: Number of connections to the web site
 
To view the list of Top URLs for selected client IP, user, MAC address and website, click the site name in the graph or hyperlink in the table.
