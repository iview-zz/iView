Reports by Virus (Upload) and File
To view the following granular reports of a particular virus (upload) and user, go to Reports → Virus →FTP Viruses → Virus →Top Viruses (Upload) widget →Viruses →Top Files and drill down by clicking file in the graph or the user hyperlink in the table:
•
Top Servers
•
Top Hosts
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections per server for the selected virus and file.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Files widget → File.
 
Bar graph displays number of connections per server whille tabular report contains following information:
•
Server: IP address of the server
•
Connections: Number of connections to the server
 
To view server details for the selected server, file and virus (upload), drill down by clicking the server in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays number of connections per host for the selected virus and file.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Files widget → File.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view host details for the selected host, file and virus (upload), drill down by clicking the host in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user for the selected virus and file.
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Files widget → File.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view user details for the selected user, file and virus (upload), drill down by clicking the user in the graph or hyperlink in the table.
