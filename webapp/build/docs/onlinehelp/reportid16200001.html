Reports for UTM Product Category : VPN : L2TP User Details
L2TP User Details
 
Report displays amount of data transfer along with local and remote interface information..
 
View the report from Reports → VPN → Top L2TP Users.
 
Bar graph displays amount of data transfer per local interface while tabular report contains following information:
•
Local Interface IP: IP address of the local interface
•
Leased IP: Leased IP address the user
•
Remote Interface IP: IP address of the remote interface
•
Up Time: Up time for the L2TP VPN tunnel
•
Down Time: Down time for the L2TP VPN tunnel
•
Bytes Sent: Number of bytes sent by L2TP user
•
Bytes Received: Number of bytes received by L2TP user
