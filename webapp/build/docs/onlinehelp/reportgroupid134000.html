Reports for UTM Product Category : Source Host Based Usage Report : Reports by Host and Application Group (Upload)
Reports by Host and Application Group (Upload)
To view the following granular reports of a particular application group and application, go to Reports → Source Host Based Usage → Top Hosts → Host → Top Application Groups (Upload) widget and drill down by clicking the application group in the graph or hyperlink in the table.
•
Top Application (Upload)
•
Top Destination (Upload)
•
Top User (Upload)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Upload) widget
Widget report displays amount of data uploaded and number of connections per application by the selected application group and host.
 
View the report from Reports → Source Host Based Usage → Top Hosts → Host → Top Application Groups (Upload) widget → Application Group.
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded
 
To view the list of destinations through which the data is uploaded by the selected host, application and application group, drill down by clicking the application in the graph or hyperlink in the table.
Top Destinations (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected host and application group.
 
View the report from Reports → Source Host Based Usage → Top Hosts → Host → Top Application Groups (Upload) widget→ Application Group.
 
Bar graph displays amount of data uploaded to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
 
To view the list of applications through which the data is uploaded by the selected host, destination and application group, drill down by clicking the destination in the graph or hyperlink in the table.
Top Users (Upload) widget
Widget report displays amount of data uploaded and number of connections per user by the selected host and application group.
 
View the report from Reports → Source Host Based Usage → Top Hosts → Host → Top Application Groups (Upload) widget → Application Group.
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data uploaded
 
To view the list of applications through which the data is uploaded by the selected user, host and application group, drill down by clicking the User in the graph or hyperlink in the table.
