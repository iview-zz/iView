Reports for UTM Product Category : Event : System Events
System Events
Report displays system events detail including time, event type, severity and message.
 
View report from Reports → Event → System Events
 
Tabular report contains following information:
•
Time: Date and time of the event
•
Event Type: Type of the event. Possible event types are :

Appliance

Interface

Gateway

DDNS

Quarantine

DHCP Server

HA

IPSec

L2TP

PPTP

Webcat

IPS

AV

Dial-in
•
Severity: Severity level of the events. Predefined level are:

EMERGENCY - System is not usable

ALERT - Action must be taken immediately

CRITICAL - Critical condition

ERROR - Error condition

WARNING - Warning condition

NOTICE - Normal but significant condition

INFORMATION – Informational

DEBUG- Debug - level messages
•
Message: Message associated with event. Complete message can be viewed by placing cursor on the message.
