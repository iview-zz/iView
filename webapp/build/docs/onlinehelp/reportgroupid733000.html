Reports for UTM Product Category : Blocked Attempts : Reports by Host and User
Reports by Host and User
To view the following granular reports of a particular host and user, go to Reports → Blocked Attempts → Top Denied Hosts→ Host → Top Users widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Applications
•
Top Destinations
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application widget
Widget report displays number of connections per application by the selected user and host.
 
View the report from Reports → Blocked Attempts → Top Denied Hosts → Host→ Top Users widget→ User.
 
Bar graph displays number of connections to each application while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of destinations for the selected application, user and host, drill down by clicking the application in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected host and user.
 
View the report from Reports → Blocked Attempts → Top Denied Hosts → Host→ Top Users widget→ User.
 
Bar graph displays number of connections to each destination in the group while tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of applications for the selected user, host and destination, drill down by clicking the destination in the graph or hyperlink in the table.
