Reports for UTM Product Category : FTP Usage : Top Users and Files by Server and Host
Top Users and Files by Server and Host
Report displays a list users and files downloaded through the selected server and host. It also displays the number of connections established to download the file and the time and amount of data downloaded.
 
View the report from Reports → FTP Usage → Top FTP Users (Download) → User → Top Servers widget → Server.
