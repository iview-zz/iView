Reports for UTM Product Category : Source Host Based Usage Report : Reports by User and Application Group (Download)
Reports by User and Application Group (Download)
To view the following granular reports of a particular user and application group, go to Reports → Source Host Based Usage → Top Users → User → Top Application Groups (Download) widget and drill down by clicking the application group name in the graph or hyperlink in the table.
•
Top Application (Download)
•
Top Destinations (Download)
•
Top Host (Download)
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Download) widget
Widget report displays amount of data downloaded and number of connections per application by the selected user and application group.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Application Groups (Download) widget→Application Group.
 
Bar graph displays amount of data downloaded by each application in the group while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data downloaded through the application
 
To view the list of destinations from where the data is downloaded through the selected application and user, drill down by clicking the application name in the graph or hyperlink in the table.
Top Destination (Download) widget
Widget report displays amount of data downloaded and number of connections per destination by the selected user and application group.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Application Groups (Download) widget→Application Group.
 
Bar graph displays amount of data downloaded through each destination while tabular report contains following information:
•
Destination: Site name or IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data downloaded
 
To view the list of applications through which the data is downloaded by the selected destination, application group and user, drill down by clicking the destination in the graph or hyperlink in the table.
Top Host (Download) Widget
Widget report displays amount of data downloaded and number of connections per host by the selected user and application group.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Application Groups (Upload) widget→Application Group.
 
Bar graph displays amount of data downloaded through each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
 
To view the list of applications through which the data is downloaded by the selected host, application group and user, drill down by clicking the host in the graph or hyperlink in the table.
