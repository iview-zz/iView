Reports by Clientip, MAC Address and User
To view the following granular reports of a particular client IP, MAC address and user, go to Reports → Web Surfing Logger → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Sites
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Sites widget
Widget report displays number of connections per web site by the selected client IP, MAC address and user.
.View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users→ User.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
 
To view the list of URLs for selected client IP, MAC address, user and site, drill down by clicking the site name or IP address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected client, MAC address and user.
 
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users→ User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: IP address of the client machine
•
Connections: Number of connections per live IP address
 
To view the list of sites for selected client IP, MAC address, user and Live IP, drill down by clicking the live IP in the graph or hyperlink in the table.
