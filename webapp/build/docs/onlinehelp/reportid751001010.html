Reports by Destination Port, User and Liveip
To view the following granular reports of a particular destination port, user and live IP go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget → User → Top Liveip widget and drill down by clicking the live IP in the graph or hyperlink in the table.
•
Top Clientips
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected live IP, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User→ Top Liveips widget→ Live IP.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of MAC Addresses for selected destination port, user, live IP and client IP drill down by clicking the client IP in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected live IP, user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User→ Top Liveips widget→ Live IP.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of client IPs for selected destination port, user, live IP and destination drill down by clicking the destination in the graph or hyperlink in the table.
