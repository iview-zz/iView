Reports for UTM Product Category : Application Based Usage Report : Top Users per Application Group (Download)
Top Users per Application Group (Download)
Report displays list of top users along with the application group and number of connections that generate the most traffic for various applications, destinations and hosts.
 
View the report from Reports → Application Based Usage → Top Users per Application Group (Download).
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Application Group: Displays name of the application group as defined in Cyberoam iView. If application group is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified applications and group unassigned applications please refer to Add Custom Application
•
Connections: Number of connections to the user
•
Bytes: Amount of data downloaded
 
To view the following granular reports of a particular user, drill down by clicking User in the graph or the User hyperlink in the table:
•
Top Applications (Download)
•
Top Destinations (Download)
Top Hosts (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
