Reports for EPS Product Category : Application Control : Reports by File Name
Reports by File Name
To view the following granular reports of a particular file name, go to Reports → Application Control → Top File Names and drill down by clicking file name in the graph or the hyperlink in the table:
•
Top Host IP Address
•
Top Action Taken
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Address widget
Widget report displays number of connections per host by the selected file.
 
View the report from Reports → Application Control → Top File Names→ File Name.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the following granular reports for selected host and file, click the file name in the graph or hyperlink in the table.
•
Top Action Taken
•
Top Program
Top Action Taken widget
Widget report displays number of connections per action taken by the selected file.
 
View the report from Reports → Application Control → Top File Names→ File Name.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product.Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and file, click the action taken in the graph or hyperlink in the table.
•
Top Host IP Address
•
Top Program
Top Program widget
Widget report displays number of connections per program by the selected file name.
 
View the report from Reports → Application Control → Top File Names→ File Name.
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the following granular reports for selected file and program, click the program, in the graph or hyperlink in the table.
•
Top Host IP Address
•
Top Action Taken
