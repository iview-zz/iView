Reports for EPS Product Category : File Antivirus : Reports by Host
Reports by Host
To view the following granular reports of a particular host, go to Reports → File Antivirus → Top Host and drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Action Taken
•
Top Application
•
Top Path of Application
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Action Taken widget
Widget report displays number of connections per action by the selected host.
 
View the report from Reports → File Antivirus → Top Host→ Host.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the application. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and application, click the application in the graph or hyperlink in the table.
•
Top Application
•
Top Path of Application
Top Application widget
Widget report displays number of connections per application by the selected host.
 
View the report from Reports → File Antivirus → Top Host→ Host.
 
Bar graph displays number of connections per application while tabular report contains following information:
•
Name of Application: Displays name of the application as defined in the EPS product.
•
Connections: Number of connections to the application
 
To view the following granular reports for the selected application and host, click the application in the graph or hyperlink in the table.
•
Top Action Taken
•
Top Path of Application
Top Path of Application widget
Widget report displays number of connections per path of virus-infected file by the selected host.
 
View the report from Reports → File Antivirus → Top Host→ Host.
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays the path of the application
•
Path of Application: Displays location of the virus infected file
 
To view the following granular reports for selected path and action, click the application path in the graph or hyperlink in the table.
•
Top Application
•
Top Action Taken
