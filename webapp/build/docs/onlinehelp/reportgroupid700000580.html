Reports for Web Server Product Category : Apache : Reports by Client and Server
Reports by Client and Server
To view the following granular reports of a particular client and server go to Reports → Apache → Top Clients→ Client IP→ Top Servers widget and drill down by clicking server in the graph or the server hyperlink in the table:
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected client and server.
 
View the report from Reports → Apache → Top Clients → Client IP → Top Servers widget → Server.
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Referrers widget
Widget report displays number of connections and amount of data transfer per referrer by the selected client and server.
 
View the report from Reports → Apache → Top Clients → Client IP → Top Servers widget → Server.
 
Bar graph displays amount of data transfer per referrer while tabular report contains following information:
•
Referrer: URL of the referrrer
•
Connections: Number of connections to the referrer
•
Bytes: Amount of data transfer
 
To view the list of URLs for the selected referrer, client and server click the referrer URL in the graph or hyperlink in the table.
