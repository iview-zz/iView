Reports for Web Server Product Category : Apache : Reports by URL
Reports by URL
 
To view the following granular reports of a particular URL, go to Reports → Apache → Top URLs and drill down by clicking URL name in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected URL.
 
View the report from Reports → Apache → Top URLs → URL.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
To view the list of user agents and list of response codes for selected URL and server, click the server in the graph or hyperlink in the table.
Top Clients widget
