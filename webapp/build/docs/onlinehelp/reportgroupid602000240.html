Reports for EPS Product Category : USB Control : Reports by User and Action Taken
Reports by User and Action Taken
To view the following granular reports of a particular action and user, go to Reports → USB Control → Top Users → User → Action Taken widget and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected user and action taken.
 
View the report from Reports → USB Control → Top Users→ User→ Action Taken widget→ Action.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IPs: IP address of the host
•
Connections: Number of connections to the host
 
To view the list of USB serial numbers for the selected host, user and action, click the host in the graph or hyperlink in the table.
Top USB Serial Number widget
Widget report displays number of connections per USB by the selected user and action.
 
View the report from Reports → USB Control → Top Users→ User→ Action Taken widget→ Action.
 
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the list of host IP address for the selected action, user and USB serial number, click the USB serial number in the graph or hyperlink in the table.
