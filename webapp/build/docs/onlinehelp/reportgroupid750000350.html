Reports by Site, Live IP and User
To view the following granular reports of a particular site, live IP and user, go to Reports → Web Surfing Logger → Top Sites → Site → Top Liveips widget → Liveips → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top MAC Address
•
Top Clientip
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top MAC Address widget
Widget report displays number of connections MAC address by the selected site and user.
.
View the report from Reports → Web Surfing Logger → Top Sites → Site → Top Users widget→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of URLs for selected MAC address, Live IP, site and user, drill down by clicking the MAC address in the graph or hyperlink in the table.
Top Clientips widget
Widget report displays number of connections per client IP by the selected site, live IP and user.
 
View the report from Reports → Web Surfing Logger → Top Sites → Site → Top Liveips widget → Liveips → Top Users → User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of URLs for selected client IP, Live IP, site and user, drill down by clicking the client IP in the graph or hyperlink in the table.
