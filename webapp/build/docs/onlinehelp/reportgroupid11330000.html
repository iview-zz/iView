Reports for UTM Product Category : Virus : Reports by User
Reports by User
To view the following granular reports of a particular user, go to Reports → Virus →Web Viruses → Virus →Top Users widget and drill down by clicking username in the graph or the username hyperlink in the table:
•
Top Viruses
•
Top Domains
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Viruses widget
Widget report displays number of connections per virus for the selected user.
 
View report from Reports → Virus → Web Viruses → Top Users widget → User.
 
Bar graph displays number of connections per virus whille tabular report contains following information:
•
Virus: Name of the virus identified by monitored device
•
Connections: Number of connections to the virus
 
To view virus details for the selected virus and user drill down by clicking the virus name in the graph or hyperlink in the table.
Top Domains widget
Widget report displays number of connections per domain for the selected user.
 
View report from Reports → Virus →Web Viruses → Top Users widget → User.
 
Bar graph displays number of connections per domain whille tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
 
To view domain details for the selected domain and user drill down by clicking the domain name in the graph or hyperlink in the table.
 
Top Hosts widget
Widget report displays number of connections per host for the selected user.
 
View report from Reports → Virus → Web Viruses → Top Users widget → User.
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view host details for the selected host and user drill down by clicking the host in the graph or hyperlink in the table.
