Reports by Host IP Address and Action Taken
To view the following granular reports of a particular host and action taken, go to Reports → Application Control → Top Host IP Address → Host IP Address → Top Action Taken widget and drill down by clicking action in the graph or the hyperlink in the table:
•
Top File Names
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top File Names widget
Widget report displays number of connections per file by the selected host and action.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top Action Taken widget→ Action Taken.
 
Bar graph displays number of connections per file while tabular report contains following information:
•
File Names : Name of the file
•
Connections: Number of connections to the file
 
To view the list of programs for the selected host, file name and action, click the file name in the graph or hyperlink in the table.
Top Program widget
Widget report displays number of connections per program by the selected host and action taken.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top Action Taken widget → Action Taken.
 
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the list of file names for the selected host, action and program, click the program in the graph or hyperlink in the table.
