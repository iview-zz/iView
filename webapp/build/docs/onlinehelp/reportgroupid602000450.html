Reports for EPS Product Category : USB Control : Reports by Action Taken and USB Serial Number
Reports by Action Taken and USB Serial Number
To view the following granular reports of a particular action and USB serial number, go to Reports → USB Control → Action Taken → Action → Top USB Serial Number widget and drill down by clicking USB serial number in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected USB serial number and action taken.
 
View the report from Reports → USB Control → Action Taken → Action → Top USB Serial Number widget→ USB Serial Number.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IPs: IP address of the host
•
Connections: Number of connections to the host
 
To view the list of users for the selected host, USB serial number and action, click the host in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user by the selected action and USB serial number.
 
View the report from Reports → USB Control → Action Taken → Action → Top USB Serial Number widget→ USB Serial Number.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the list of host IP address for the selected action, user and USB serial number, click the user in the graph or hyperlink in the table.
