Reports for UTM Product Category : Web Usage : Reports by User Group and User
Reports by User Group and User
To view the following granular reports of a particular user group and user, go to Reports → Web Usage → Top Web User Groups → User Group → Top Users widget and drill down by clicking the user hyperlink in the table.
•
Top Categories
•
Top Contents
•
Top Domains
•
Top URLs
•
Top Hosts
•
Top Applications
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the selected user and user group.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Users widget →User.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
Category: Displays name of the category
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the domain
 
To view the list of domains for the selected user, category and user group, drill down by clicking the category name hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections per content type by the selected user and user group.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Users widget →User.
 
Bar graph displays amount of data transferred by each content type while tabular report contains following information:
•
Content: Displays type of the content
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
To view the list of domains for the selected user, content and user group, drill down by clicking the content hyperlink in the table.

Top Hosts widget
Widget report displays amount of data transferred and number of connections per host by the selected user and user group.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Users widget →User.
 
Bar graph displays amount of data transferred by each host while tabular report contains following information:
•
Host: Displays IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred through the host
 
To view the following granular reports of each host drill down by clicking the host hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected host, user and user group
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected host and user group
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected host, user and user group
•
Top URLs i.e. number of connections and amount of data transferred per URL for the selected host, user and user group
•
Top Applications i.e. number of connections and amount of data transferred per application for the selected host, user and user group

Top Applications widget
Widget report displays amount of data transferred and number of connections per application by the selected user and user group.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Users widget →User.
 
Bar graph displays amount of data transferred by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam-iView. If application is not defined in Cyberoam-iView then this field will display application identifier as combination of protocol and port number. To defined an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the following granular reports of each application drill down by clicking the application hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred category wise for the selected application, user and user group
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected application, user and user group
