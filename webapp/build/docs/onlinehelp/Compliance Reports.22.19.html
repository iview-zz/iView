Top Spam Recipients
Widget report displays number of connections per spam recipient
 
View the report from Compliance Reports → SOX→ Top Spam Recipients
 
Bar graph displays number of connections per recipient while tabular report contains following information:
•
Recipient: Colon (: ) separated email IDs of the recipient(s)
•
Connections: Number of connections to the recipient
 
For in depth reports refer to Reports by Spam Recipients
