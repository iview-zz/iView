Top Accept Rules –Destination Wise
Top Accept Rules report enables to view amount of traffic allowed for various destinations.
View the report from Reports → Firewall Rules Based Usage → Top Accept Rules- Destination Wise.
 
The bar graph displays amount of data transferred to the destinations through firewall rule
The table contains following information:
 
•
Destination: IP address of the destination
•
Rule ID: Firewall rule ID as defined in the monitored device. If firewall rule ID is not defined in the monitored device then it will display firewall policy name.
•
Connections: Number of connections per destination
•
Bytes: Amount of data transferred to the destination
 
To get in depthAccept Report – Destination Wise for a particular destination, drill down by clicking the graph or the destination hyperlink.
