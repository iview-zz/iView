Reports for UTM Product Category : FTP Usage
FTP Usage
FTP Usage reports gives insight about the FTP activity - uploads and downloads i.e. FTP traffic volume over the reporting period.
 
It provides statistics based on traffic generated by host, users and servers.
View FTP usage reports from Reports  FTP Usage.
 
It enables to view FTP traffic generated by:
•
Top Files Uploaded via FTP
•
Top Files Downloaded via FTP
•
Top FTP Users (Upload)
•
Top FTP Users (Download)
•
Top FTP Hosts (Upload)
•
Top FTP Hosts (Download)
•
Top FTP Servers
