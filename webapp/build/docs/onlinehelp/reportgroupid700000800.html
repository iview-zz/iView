Reports for Web Server Product Category : Apache : Reports by Referrer
Reports by Referrer
To view the following granular reports of a particular referrer, go to Reports → Apache → Top Referrers and drill down by clicking referrer URL in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top URLs
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected referrer.
 
View the report from Reports → Apache → Top Referrers → Referrer.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
To view the list of URLs for selected server and referrer, click the server IP in the graph or hyperlink in the table.
