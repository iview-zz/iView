Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group, User (Upload) and Host (Upload)
Reports by Application Group, User (Upload) and Host (Upload)
To view the following reports of a particular host, user and application group go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Host (Upload) widget →Host.
•
Top Application (Upload)
•
Top Destination (Upload)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Upload) widget
Widget report displays amount of data uploaded and number of connections per application by the selected host, user and application group.
 
View the report from Reports → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Hosts (Upload) widget →Host.
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded
Top Destination (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected host, user and application group.
View the report from Reports → Top Application Groups → Application Group → Top Users (Upload) widget →User →Top Hosts (Upload) widget →Host.
 
Bar graph displays amount of data uploaded to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
