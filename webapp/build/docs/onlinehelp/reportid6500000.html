Reports for UTM Product Category : FTP Usage : Top FTP Hosts (Upload)
Top FTP Hosts (Upload)
Report displays the list of top hosts through which the maximum FTP uploaded traffic is generated with number of connections established to upload the data.
 
View the report from Reports → FTP Usage Reports→Top FTP Hosts (Upload).
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: Host IP address through which the file is uploaded
•
Connections: Number of Connections to upload the file
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular host; drill down by clicking the bar against host in the graph or the hyperlink in the table:
•
Top Files
•
Top Users
•
Top Servers
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
