Reports for EPS Product Category : USB Control : Reports by Host IP Address and Action Taken
Reports by Host IP Address and Action Taken
To view the following granular reports of a particular host IP address and action taken, go to Reports → USB Control → Top Host IP Addresses → Host IP Address → Action Taken widget and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Users
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user by the selected host and action.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Action Taken widget→ Action.
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the list of USB serial numbers for the selected host, user and action, click the user in the graph or hyperlink in the table.
Top USB Serial Number widget
Widget report displays number of connections per USB by the selected host and action.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Action Taken widget→ Action.
 
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the list of users for the selected host, action and USB serial number, click the USB serial number in the graph or hyperlink in the table.
