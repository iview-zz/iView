Reports for Web Server Product Category : Apache : Top Servers
Top Servers
 
Report displays list of top servers along with the number of connections that generate the most traffic for various sites, clients, response codes, content types and user agents.
 
View the report from Reports → Apache → Top Servers.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transfer per server while tabular report contains following information:
•
Server: Name or IP address of the server.
•
Connections: Number of connections
•
Bytes: Amount of data transfer
 
To view the following granular reports of a particular server, drill down by clicking server domain name or IP address in the graph or the hyperlink in the table:
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
