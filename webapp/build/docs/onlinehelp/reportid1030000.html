Reports for UTM Product Category : Spam : Top Applications used for Spam
Top Applications used for Spam
Report displays list of top applications used to generate spam along with the number of connections for various spam recipients, senders, users, destinations and source hosts.
 
View the report from Reports →Spam→ Top Applications used for Spam.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per spam sender while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the following granular reports of a particular application, drill down by clicking application in the graph or the application hyperlink in the table:
•
Top Senders
•
Top Recipients
•
Top Users
•
Top Hosts
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
