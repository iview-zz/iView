Reports for UTM Product Category : Blocked Web Attempts : Reports by Web Hosts
Reports by Web Hosts
To view the granular reports of particular web host, go to Reports → Blocked Web Attempts → Top Denied Web Hosts and drill down by clicking the bar against host name in the graph or host name hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top URLs
•
Top Users
•
Top Applications
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories Widget
Widget displays list of denied categories that a host has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Hosts → Host.
 
Bar graph displays various denied categories which a host tried to access while tabular report contains following information:
•
Category: Displays the various denied web categories
•
Connections: Number of Connections
 
To view the list of domains, which comes under a denied category, drill down by clicking the category in the graph or category hyperlink in the table.
Top Domains Widget
Widget displays list of denied domains which a host has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Hosts → Host.
 
Bar graph displays various denied domains which a host has tried to access while tabular report contains following information:
•
Domain: Displays the various denied web domains
•
Connections: Number of Connections
 
To view the list of URLs, which comes under a denied domain, drill down by clicking the domain in the graph or domain hyperlink in the table.
Top URLs Widget
Widget displays list of URLs which the selected host has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Hosts → Host.
 
Bar graph displays various denied URLs which the selected host has tried to access while tabular report contains following information:
•
URL: Name of URL
•
Connections: Number of Connections
Top Users Widget
Widget displays list of users along with the number of connections who tried to access denied sites through a host.
View the report from Reports → Blocked Web Attempts → Top Denied Web Hosts → Host.
 
Bar graph displays various users who tried to access denied sites through a host while tabular report contains following information:
•
Users: Displays the User name
•
Connections: Number of Connections
 
To view the following granular reports of each user, drill down by clicking the user name link in the graph or user name hyperlink in the table:
 
•
Top Categories i.e. list of denied categories a user tried to access through a host.
•
Top Domains i.e. list of denied domains a user tried to access through a host.
•
Top URLs i.e. the list of URLs a user tried to access through a host.
•
Top Applications i.e. the list of URLs a user tried to access through a host.
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
Top Applications Widget
Widget displays list of denied applications, which a host has tried to access, along with the number of connections.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Hosts → Host.
 
Bar graph displays various denied applications which a host has tried to access while tabular report contains following information:
•
Applications: Displays the various denied applications
•
Connections: Number of Connections
 
To view the list of categories, which comes under a denied application type, drill down by clicking the application in the graph or application hyperlink in the table.
