Reports for UTM Product Category : Application Based Usage Report : Reports by Application and Host (Upload)
Reports by Application and Host (Upload)
To view the following granular reports of a particular application and host, go to Reports → Application Based Usage → Top Applications (Upload) → Application → Top Hosts (Upload) widget and drill down by clicking the host in the graph or hyperlink in the table.
•
Top Destination
•
Top User
The granular reports page displays multiple reports in the widgets form
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destination (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected host and application.
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Hosts (Upload) → Host.
 
Bar graph displays amount of data uploaded to each destination while tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
Top User (Upload) widget
Widget report displays amount of data uploaded and number of connections per user by the selected host and application
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Hosts (Upload) → Host.
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data uploaded
