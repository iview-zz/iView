Reports for EPS Product Category : Application Control : Top Host IP Address
Top Host IP Address
Report displays list of top host IP addresses along with the number of connections. Report can be drilled down for host IP to view top file names, top action taken and top programs.
 
View the report from Reports → Application Control → Top Host IP Address.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the following granular reports of a host, drill down by clicking host in the graph or the hyperlink in the table:
•
Top File Names
•
Top Action Taken
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
