Reports for EPS Product Category : USB Control : Reports by Host IP Address and User
Reports by Host IP Address and User
To view the following granular reports of a particular host IP address and user, go to Reports → USB Control → Top Host IP Addresses → Host IP Address → Top Users widget and drill down by clicking user in the graph or the hyperlink in the table:
•
Action Taken
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Action Taken widget
Widget report displays number of connection per action by the selected host and user.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Top Users widget→ User.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product. Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of USB serial numbers for the selected host, user and action, click the action in the graph or hyperlink in the table.
Top USB Serial Number widget
Widget report displays number of connections per USB by the selected host and user.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP→ Top Users widget→ User.
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the list of actions for the selected host, user and USB serial number, click the USB serial number in the graph or hyperlink in the table.
