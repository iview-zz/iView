System : Configuration : Report Notification
Report Notification
 
Cyberoam iView can mail reports in PDF format to specified email addresses as per the configured frequency.
Use the System → Configuration → Report Notification to create and manage report notifications.
Screen Components:
•
Name: Name of the report.
•
Report Group: Category of the reports.
•
Device Name: Name of reported device(s).
•
Email Frequency: Report notification frequency- daily or weekly.
•
To Email Address: Email ID of recipient(s).
•
Last Sent Time: Last time when the report notification was sent.
 
Use the System → Configuration → Report Notification to
•
Add Report Notification
•
Update Report Notification
•
Delete Report Notification
Add Report Notification
 
•
Super Admin or Admin privilege required to create report notification
 
1.
Go to System → Configuration → Report Notification
2.
Click Add to create a new report notification
3.
Specify Report Name. Report name can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
4.
Specify description of the report notification, if required.
5.
Specify Email address of the recipient in ‘To Email Address’ field. Use comma to separate multiple E-mail IDs.
9.
Select product category from Category drop down list. Possible product categories are UTM, Access Gateway, EPS and Web Server
6.
Select type of the report. Possible types of reports are Reports and Bookmarks.
7.
Select report category from the Report Group drop down list or Bookmarks drop down list. Reports from selected category will be sent to the recipients.
8.
Click the device(s) whose reports are to be mailed from the ‘Available Devices’ list and click to move the selected devices to the 'Selected Devices' list To select multiple devices press Ctrl key and select devices using mouse.
9.
Set E-mail frequency and time. Reports can be mailed daily, weekly, monthly and only once at the configured interval. In case of daily notification select time of the day, report day and time duration of the report. In case of weekly notification, select day of the week, while in case of weekly notification select date and time of the month and in case of one time report select time and date along with time duration of report. Possible time durations to send one time report are previous day, same day, last week and last month.
10.
Click Add to save the changes.
Update Report Notification
 
•
Super Admin or Admin privilege required to update the report notification schedule
 
1.
Go to System → Configuration → Report Notification and click report notification, to be updated.
2.
Refer to Add Report Notification for information on each parameter.
•
All the fields except Report Name and Product Category are editable.
 
Delete Report Notification
 
•
Super Admin or Admin privilege required to delete the report notification schedule
 
1.
Go to System → Configuration → Report Notification
2.
Click checkbox against the report notification(s) to be deleted OR click Select All to delete all profiles.
3.
Click Delete.
