Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Users by Site and Liveip
Top Users by Site and Liveip
Report displays a list of users along with the number of connection per user for selected site and live IP.
 
View the report from Reports → Web Surfing Logger → Top Sites → Site → Top Liveips widget→ Liveip.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular site, live IP and user, go to Reports → Web Surfing Logger → Top Sites → Site → Top Liveips widget → Liveips → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top MAC Address
•
Top Clientip
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
