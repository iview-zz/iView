Reports for UTM Product Category : Blocked Attempts : Top Denied Hosts
Top Denied Hosts
Report displays list of top denied hosts along with the number of connections that generate the most traffic for various users, destinations and applications.
 
View the report from Reports →Blocked Attempts → Top Denied Hosts.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per denied host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view the following granular reports of a particular application group drill down by clicking application group in the graph or the application group hyperlink in the table:
•
Top Application Groups
•
Top Destinations
•
Top Users
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
