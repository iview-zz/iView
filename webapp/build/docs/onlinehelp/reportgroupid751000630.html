Reports for Access Gateway Product Category : Net Capture Report : Reports by MAC Address and User
Reports by MAC Address and User
To view the following granular reports of a particular MAC address and user, go to Reports → Net Capture → Top MAC Addresses → MAC Address → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Destinations
•
Top Clientips
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destinations widget
Widget report displays number of connections per destination by the selected MAC address and user.
.
View the report from Reports → Net Capture → Top MAC Addresses → MAC Address → Top Users widget→ User.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of destination ports for selected MAC address, user and destination drill down by clicking the destination IP address in the graph or hyperlink in the table.
Top Clientips widget
Widget report displays number of connections per client IP by the selected MAC address and user.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC Address → Top Users widget→ User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of destinations for selected client IP, user and MAC address, drill down by clicking the client IP address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected client IP and user.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC Address → Top Users widget→ User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the list of destinations for selected live IP, MAC address and user, drill down by clicking the live IP address in the graph or hyperlink in the table.
