Reports for UTM Product Category : Web Usage : Top Web Hosts
Top Web Hosts
Report displays list of top hosts along with the number of connections that generate the most traffic for various domains, users, contents, URLs, applications and categories.
 
View the report from Reports →Web Usage → Top Web Hosts.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transferred per host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular host drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
•
Top URLs
•
Top Web Users
•
Top Applications
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
