Reports for UTM Product Category : Mail Usage : Top Mail Senders
Top Mail Senders
Report displays list of top email senders along with the number of connections that generate the most traffic for various users, destinations, hosts and applications.
 
View the report from Reports →Mail Usage → Top Mail Senders.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred by each sender while tabular report contains following information:
•
Sender: Email ID of the sender
•
Connections: Number of connections to the sender
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular sender, drill down by clicking sender email in the graph or the sender email hyperlink in the table:
•
Top Recipients
•
Top Source Host
•
Top Destinations
•
Top Users
•
Top Applications
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
