Reports for UTM Product Category : Blocked Attempts : Reports by Application Group, Destination and User
Reports by Application Group, Destination and User
To view the following reports of a particular user, application group and destination go to Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Destinations widget→ Top Users widget → User.
•
Top Applications
•
Top Hosts
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected user, destination, and application group.
 
View the reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Destinations widget→ Top Users widget → User.
Bar graph displays number of connections to each application while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
Top Hosts widget
Widget report displays number of connections per host by the selected user, destination and application group.
 
View the reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Destinations widget→ Top Users widget → User.
 
Bar graph displays number of connections to each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
