Reports by Attacks
To view the following granular reports under particular attack; go to Reports → Attacks → Top Attacks and drill down by clicking attack name in the graph or the attack name hyperlink in the table:
•
Top Attackers
•
Top Victims
•
Top Applications
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Attackers widget
Widget report displays number of connections for every attacker under the selected attack.
 
View the reports from Reports → Attacks → Top Attacks → Top Attackers.
 
Bar graph displays the IP address of the attacker, while tabular report contains following information:
•
Attacker:IP address of the attacker
•
Connections: Number of connections by the attacker
 
To view the list of victims by the attacker under particular attack, drill down by clicking the Attacker IP in the graph or hyperlink in the table.
Top Victims widget
Widget report displays number of connections for every victim under the selected attack.
 
View the reports from Reports → Attacks → Top Attacks → Top Victims.
 
Bar graph displays the IP address of the victim, while tabular report contains following information:
•
Victim:IP address of the victim
•
Connections: Number of connections by the victim
 
To view the list of attackers, the victim has launched under particular attack, drill down by clicking the victim IP in the graph or hyperlink in the table.
Top Applications widget
Widget report displays number of connections for every application under the selected attack.
 
View the reports from Reports → Attacks → Top Attacks → Top Applications.
 
Bar graph displays the name of the application under attack, while tabular report contains following information:
•
Application: Name of the application under attack
•
Connections: Number of connections for the application
 
To view the list of attackers, the application has launched under particular attack, drill down by clicking the application name in the graph or hyperlink in the table.
