Reports for EPS Product Category : File Antivirus : Reports by Application and Path of Application
Reports by Application and Path of Application
To view the following granular reports of a particular application and path of application, go to Reports → File Antivirus → Top Application → Name of Application → Top Path of Application widget and drill down by clicking path in the graph or the hyperlink in the table:
•
Top Action Taken
•
Top Host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Action Taken widget
Widget report displays number of connections per action taken by the selected application and path.
 
View the report from Reports → File Antivirus → Top Application→ Top Path of Application widget → Path of Application.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the application. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
 
To view the list of hosts for the selected application, path and action click the action in the graph or hyperlink in the table.
Top Host widget
Widget report displays number of connection per host by the selected application and path.
 
View the report from Reports → File Antivirus → Top Application→ Top Path of Application widget → Path of Application.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host : Displays IP address of the host
•
Connections: Number of connections to the host
 
To view the list of actions taken for the selected application, path and host click the host IP in the graph or hyperlink in the table.
