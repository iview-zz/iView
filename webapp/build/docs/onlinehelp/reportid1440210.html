Reports for UTM Product Category : Source Host Based Usage Report : Top Applications by User, Application Group and Host
Top Applications by User, Application Group and Host
Report displays a list of applications, the number of hits to each application, and the amount of data transferred by the selected destination, application group and the user.
 
View the report from Reports → Source Host Based Usage → Top Users → User → Top Application Group widget →Application Group→Top Hosts widget →Host.
 
To view the list of destination from where the data is transferred by the selected application, host, application group and user, drill down by clicking the Application in the graph or hyperlink in the table.
