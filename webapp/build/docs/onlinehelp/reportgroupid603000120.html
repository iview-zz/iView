Reports by Action
To view the following granular reports of a particular action, go to Reports → Web Report → Action Taken and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Web Host
•
Web Sites Visited
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Web Host widget
Widget report displays number of connections per web host by the selected action.
 
View the report from Reports →Web Report → Action Taken→ Action.
 
Bar graph displays number of connections per web host while tabular report contains following information:
•
Web Host: IP address of the web host
•
Connections: Number of connections to the web host.
 
To view list of web sites for the selected web host and action taken drill down by clicking host in the graph or the hyperlink in the table.
Web Sites Visited widget
Widget report displays number of connections per web site by the selected action.
 
View the report from Reports →Web Report → Action Taken → Action.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Web Sites: URL name of the web site
•
Connections: Number of connections to the web site.
 
To view list of web hosts for the selected action and web site drill down by clicking URL in the graph or the hyperlink in the table.
