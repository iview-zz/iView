Reports for UTM Product Category : Web Usage : Top Domains by User Group, User and Content
Top Domains by User Group, User and Content
Report displays a list of domains, the number of connections to each domain and the amount of data transferred through the selected content, user group and by the user.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Users widget →User → Top Contents widget → Content.
 
To view the list of URLs for the selected domain, category, user group and user, drill down by clicking the domain name hyperlink in the table.
