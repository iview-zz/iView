Reports for UTM Product Category : Attacks : Reports by Victims
Reports by Victims
To view the following granular reports for particular victim ; go to Reports → Attacks → Top Victims and drill down by clicking victim in the graph or the victim IP hyperlink in the table:
•
Top Attacks
•
Top Attackers
•
Top Applications
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Attacks widget
Widget report displays number of connections under every attack for the selected victim.
 
View the reports from Reports → Attacks → Top Victims → Top Attacks.
 
Bar graph displays the type of the attack, while tabular report contains following information:
•
Attack:Name of the attack launched
•
Connections: Number of connections under the attack
 
To view the list of attackers for the victim under particular attack, drill down by clicking the attack in the graph or hyperlink in the table.
Top Attackers widget
Widget report displays number of connections for every attacker for the selected victim.
 
View the reports from Reports → Attacks → Top Victims → Top Attackers.
 
Bar graph displays the IP address of the victim, while tabular report contains following information:
•
Attacker: IP address of the attacker
•
Connections: Number of connections by the attacker
 
To view the list of attacks, the attacker has launched with particular victim, drill down by clicking the attacker IP in the graph or hyperlink in the table.
Top Applications widget
Widget report displays number of connections for every application under attack for the selected victim.
 
View the reports from Reports → Attacks → Top Victims → Top Applications.
 
Bar graph displays the name of the application used by the victim, while tabular report contains following information:
•
Application: Name of the application
•
Connections: Number of connections for the application
 
To view the list of attacks, the application has launched under particular victim, drill down by clicking the application name in the graph or hyperlink in the table.
