Reports for EPS Product Category : File Antivirus : Reports by Host and Application
Reports by Host and Application
To view the following granular reports of a particular application and host, go to Reports → File Antivirus → Top Host → Host → Top Application widget and drill down by clicking application in the graph or the hyperlink in the table:
•
Top Action Taken
•
Top Path of Application
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Action Taken widget
Widget report displays number of connections per action by the selected host.
 
View the report from Reports → File Antivirus → Top Host→ Host→ Top Application widget → Name of Application.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the application. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
 
To view the path of application for the selected action, application and host, click the action in the graph or hyperlink in the table.
Top Path of Application widget
Widget report displays number of connection per path of virus-infected file by the selected application and host.
 
View the report from Reports → File Antivirus → Top Host→ Host→ Top Application widget→ Name of Application.
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays location of the virus infected file
•
Connections: Number of connections to the application path
 
To view the list of actions for the selected application, host and path click the application path in the graph or hyperlink in the table.
