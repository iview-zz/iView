Reports for EPS Product Category : File Antivirus : Reports by Host and Path of Application
Reports by Host and Path of Application
To view the following granular reports of a particular host and application path, go to Reports → File Antivirus → Top Host → Host → Top Path of Application widget and drill down by clicking application path in the graph or the hyperlink in the table:
•
Top Application
•
Top Action Taken
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application widget
Widget report displays number of connections per application by the selected host and path of application.
 
View the report from Reports → File Antivirus → Top Host→ Host→ Top Path of Application→ Path of Application.
 
Bar graph displays number of connections per application while tabular report contains following information:
•
Name of Application: Displays name of the application as defined in the EPS product.
•
Connections: Number of connections to the application
 
To view the list of actions for the selected path of application, application and host, click the action in the graph or hyperlink in the table.
Top Action Taken widget
Widget report displays number of connections per action by the selected host and path of application.
 
View the report from Reports → File Antivirus → Top Host→ Host→ Top Path of Application widget → Path of Application.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the application. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
To view the application name for the selected action, path of application and host, click the action in the graph or hyperlink in the table.
