Reports for EPS Product Category : Application Control : Reports by Host IP Address and Program
Reports by Host IP Address and Program
 
To view the following granular reports of a particular host and program, go to Reports → Application Control → Top Host IP Address → Host IP Address → Top Program widget and drill down by clicking program in the graph or the hyperlink in the table:
•
Top File Names
•
Top Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top File Names widget
Widget report displays number of connections per file by the selected host and program.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top Program widget→ Program.
 
Bar graph displays number of connections per file while tabular report contains following information:
•
File Names : Name of the file
•
Connections: Number of connections to the file
 
To view the list of actions for the selected host, file name and program, click the file name in the graph or hyperlink in the table.
Top Action Taken widget
Widget report displays number of connections per action taken by the selected host and program.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top Program widget→ Program.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product.Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of file names for the selected host, action and program, click the action in the graph or hyperlink in the table.
