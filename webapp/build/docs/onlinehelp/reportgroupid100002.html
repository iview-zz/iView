Reports for UTM Product Category : Source Host Based Usage Report : Reports by Host (Download)
Reports by Host (Download)
To view the following granular reports of a particular host, go to Reports → Source Host Based Usage → Top Hosts (Download) and drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Application Groups (Download)
•
Top Destinations (Download)
•
Top Users (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications Groups (Download) widget
Widget report displays amount of data downloaded and number of connections per application group of the selected host
 
View the report from Reports → Source Host Based Usage → Top Hosts (Download) → Host.
 
Bar graph displays amount of data downloaded by each application group while tabular report contains following information:
•
Application Group: Displays name of the application group as defined in Cyberoam iView. If application group is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application group
•
Bytes: Amount of data downloaded
 
To view the following granular reports of each application group, drill down by clicking the application group name in the graph or hyperlink in the table:
•
Top Application (Download) i.e. amount of data downloaded and number of connections per application for selected application group and host
•
Top Destination (Download) i.e. amount of data downloaded and number of connections per destination by the selected application group and host
•
Top User (Download) i.e. amount of data downloaded and number of connections per user for selected application group and host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destinations (Download) widget
Widget report displays amount of data downloaded and number of connections per destination by the selected host.
 
View the report from Reports → Source Host Based Usage → Top Hosts (Download) → Host.
 
Bar graph displays amount of data downloaded through each destination while tabular report contains following information:
•
Destination: Site URL or Destination IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data downloaded
 
To view the following granular reports of each destination, drill down by clicking the destination link in the graph or hyperlink in the table:
•
Top Application (Download) i.e. amount of data downloaded and number of connections per application by the selected host and destination
•
Top User (Download) i.e. amount of data downloaded and number of connections per user by the selected host and destination
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Users (Download) widget
Widget report displays amount of data downloaded and number of connections per user for the selected host
 
View the report from Reports → Source Host Based Usage → Top Hosts (Download) → Host
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data downloaded
 
To view the following granular reports of each user, drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Application (Download) i.e. amount of data downloaded and number of connections per applications by the user for the selected host
•
Top Destination (Download) i.e. amount of data downloaded and number of connections per destination by the user for the selected host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
