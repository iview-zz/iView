Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Sites by Clientip, MAC Address, User and Liveip
Top Sites by Clientip, MAC Address, User and Liveip
Report displays a list of web sites along with the number of hits to each website for the selected client IP, MAC address, user and live IP.
 
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top MAC Addresses widget→ MAC address → Top Users→ User → Top Liveip Widget→ Live IP.
 
To view the list of URLs for selected client IP, MAC address, user, live IP and site drill down by clicking the client IP in the graph or hyperlink in the table.
