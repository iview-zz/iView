Reports for UTM Product Category : Internet Usage : Detail Report by User Group
Detail Report by User Group
Report displays user wise data transfer along with IP address, start time, stop time, surfing time, uploaded data and downloaded data..
 
View the report from Reports → Internet Usage → Exclusive Report by User Group → User Group.
 
Tabular report contains following information:
•
User Name: Name of the user as defined in the monitored device. If the User is not defined in monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Start Time: Data transfer starting time in HH:MM:SS format
•
Stop Time: Data transfer stop time in HH:MM:SS format
•
Used Time: Time required in data transfer in HH:MM:SS format
•
IP Address: IP address used to transfer data
•
Data Transfer: Total amount of data transferred (Upload + Download) by the IP address
•
Uploaded: Amount of uploaded data
•
Downloaded: Amount of downloaded data
