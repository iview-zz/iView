Reports for UTM Product Category : FTP Usage : Reports by File Upload
Reports by File Upload
To view the following granular reports of a particular file, go to Reports → FTP Usage → Files uploaded via FTP and drill down by clicking the bar against file name in the graph or the file hyperlink in the table:
•
Top Servers
•
Top FTP Hosts
•
Top FTP Users
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays amount of data uploaded on each server in the form of the selected file.
 
View the report from Reports → FTP Usage →Top Files uploaded via FTP → File.
 
Bar graph displays amount of data uploaded to each server while tabular report contains following information:
•
Server: Server IP on which the file is uploaded
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of hosts and users who uploaded the data to that particular server for the file, drill down by clicking the server in the graph or hyperlink in the table.
Top FTP Hosts widget
Widget report displays amount of data uploaded through host in the form the selected file.
 
View the report from Reports → FTP Usage →Top Files uploaded via FTP → File.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: Hosts through which the file is uploaded
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of servers and users, drill down by clicking the server in the graph or hyperlink in the table.
Top FTP Users widget
Widget report displays amount of data uploaded by users in form of the selected file.
 
View the report from Reports → FTP Usage →Top Files uploaded via FTP → File.
 
Bar graph displays amount of data uploaded by the users in the group while tabular report contains following information:
•
Users: Displays top users through whom file is uploaded
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of hosts and servers, drill down by clicking the server in the graph or hyperlink in the table.
