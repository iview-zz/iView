Reports for EPS Product Category : Application Control : Reports by Action Taken
Reports by Action Taken
To view the following granular reports of a particular action, go to Reports → Application Control → Top Action Taken and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Host IP Address
•
Top File Names
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Address widget
Widget report displays number of connections per host by the selected action.
 
View the report from Reports → Application Control → Top Action Taken→ Action Taken.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the following granular reports for selected action and host, click the host IP address in the graph or hyperlink in the table.
•
Top File Names
•
Top Program
Top File Names widget
Widget report displays number of connections per file by the selected action.
 
View the report from Reports → Application Control → Top Action Taken→ Action Taken.
 
Bar graph displays number of connections per file while tabular report contains following information:
•
File Names : Name of the file
•
Connections: Number of connections to the file
 
To view the following granular reports for selected action and file, click the file name in the graph or hyperlink in the table.
•
Top Host IP Address
•
Top Program
Top Program widget
Widget report displays number of connections per program by the selected action.
 
View the report from Reports → Application Control → Top Action Taken→ Action Taken.
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the following granular reports for selected action and program, click the program, in the graph or hyperlink in the table.
•
Top Host IP Address
•
Top File Names
