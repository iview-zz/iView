Reports for Web Server Product Category : Apache : Reports by Server
Reports by Server
 
To view the following granular reports of a particular server, go to Reports → Apache → Top Servers and drill down by clicking server name in the graph or the server name hyperlink in the table:
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Clients widget
Widget report displays number of connections and amount of data transfer per client by the selected server.
 
View the report from Reports → Apache → Top Servers → Server.
 
Bar graph displays amount of data transfer per client while tabular report contains following information:
•
Client IP: IP address of the client
•
Connections: Number of connections to the client
•
Bytes: Amount of data transfer
To view the following granular reports for selected client and server click the client IP in the graph or hyperlink in the table.
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
