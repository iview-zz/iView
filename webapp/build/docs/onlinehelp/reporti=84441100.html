Reports for UTM Product Category : Blocked Web Attempts : Top Domains by Category, Application, User and Host
Top Domains by Category, Application, User and Host
Report displays a list of domains and the number of connections to that category under an application and through a host for a user.
 
View the report from Reports → Blocked Web Attempts → Top Denied Web Hosts → Host → Top Users → Top Applications → Top Categories → Top Domains.
 
To view the list of URLs under a domain and category for an application and user through a host, drill down by clicking the domain name in the graph or hyperlink in the table.
