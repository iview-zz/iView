Reports for EPS Product Category : eMail Scanning : Top Action by Host and User
Top Action by Host and User
Report displays list of actions along with the number of hits to each action for selected host and user.
 
View the report from Reports → eMail Scanning → Top Host → Host → Top User widget→ User.
To view the list of files for selected host, user and action, click action in the graph or hyperlink in the table.
