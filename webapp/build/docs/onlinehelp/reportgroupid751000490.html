Reports for Access Gateway Product Category : Net Capture Report : Reports by Clientip, Liveip and User
Reports by Clientip, Liveip and User
To view the following granular reports of a particular client IP, MAC address and user, go to Reports → Net Capture → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Destinations
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destinations widget
Widget report displays number of connections per destination by the selected client IP, live IP and user.
.
View the report from Reports → Net Capture → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users→ User.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of destination ports for selected client IP, Live IP, user and destination, drill down by clicking the destination IP address in the graph or hyperlink in the table.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP, live IP and user.
 
View the report from Reports → Net Capture → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of destinations for selected client IP, Live IP, user and MAC address drill down by clicking the MAC address in the graph or hyperlink in the table.
