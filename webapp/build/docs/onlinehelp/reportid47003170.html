Reports for UTM Product Category : Web Usage : Top Category Types
Top Category Types
Report displays list of top category types along with the number of connections that generate the most traffic for various users, user groups and categories.
 
View the report from Reports → Web Usage → Top Category Types.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred per category type while tabular report contains following information:
•
Category Type: Displays of category as defined in monitored device. If the category type is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ category type
•
Connections: Number of connections to the category type
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user drill down by clicking category type in the category hyperlink in the table:
•
Top Categories
•
Top Users
•
Top Web User Groups
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
