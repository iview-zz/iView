Reports by File Name and Action Taken
To view the following granular reports of a particular file name and action, go to Reports → Application Control → Top File Name → File Name → Top Action Taken widget and drill down by clicking action in the graph or the hyperlink in the table:
•
Top Host IP Address
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Address widget
Widget report displays number of connections per host by the selected file name and action.
 
View the report from Reports → Application Control → Top File Names→ File Name→ Top Action Taken widget→ Action Taken.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the list of programs for the selected host, file name and action taken; click the host IP address in the graph or hyperlink in the table.
Top Program widget
Widget report displays number of connections per program by the selected action and file name.
 
View the report from Reports → Application Control → Top File Names→ File Name→ Top Action Taken widget→ Action Taken.
 
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the list of hosts for the selected action, file name and program, click the program in the graph or hyperlink in the table.
