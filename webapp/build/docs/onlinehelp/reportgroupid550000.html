Reports by Mail Application
To view the following granular reports of a particular email application, go to Reports → Mail Usage → Top Mail Applications and drill down by clicking application in the graph or the application hyperlink in the table:
•
Top Senders
•
Top Recipients
•
Top Users
•
Top Hosts
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Senders widget
Widget report displays amount of data transferred and number of connections per sender for the selected application.
 
View the report from Reports → Mail Usage → Top Mail Applications →Application.
 
Bar graph displays amount of data transferred by each sender whille tabular report contains following information:
•
Sender: Email ID of the sender
•
Connections: Number of connections to the sender
•
Bytes: Amount of data transferred
 
To view sender details for the selected application, drill down by clicking the sender in the graph or hyperlink in the table.
Top Recipients widget
Widget report displays amount of data transferred and number of connections per recipient for the selected application.
View the report from Reports → Mail Usage → Top Mail Applications →Application.
 
Bar graph displays amount of data transferred to each recipient whille tabular report contains following information:
•
Recipient: Colon (: ) separated email IDs of recipient(s)
•
Connections: Number of connections to the recipient
•
Bytes: Amount of data transferred
 
To view recipient details for the selected application, drill down by clicking the recipient in the graph or hyperlink in the table.
Top Users widget
Widget report displays amount of data transferred and number of connections per user for the selected application.
 
View the report from Reports → Mail Usage → Top Mail Applications →Application.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
•
To view user details for selected application, drill down by clicking the User in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays amount of data transferred and number of connections per host for the selected application.
 
View the report from Reports → Mail Usage → Top Mail Applications →Application.
 
Bar graph displays amount of data transferred by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view host details for selected application, drill down by clicking the host in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays amount of data transferred and number of connections per destination for the selected application.
 
View the report from Reports → Mail Usage → Top Mail Applications →Application.
 
Bar graph displays amount of data transferred to each destination while tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data transferred
 
To view destination details for the selected application, drill down by clicking the destination in the graph or hyperlink in the table.
