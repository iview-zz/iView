Reports for UTM Product Category : Virus : FTP Viruses
FTP Viruses
Report displays multiple reports in the widgets form, which can again be drilled down to view the detailed report:
•
Top Viruses (Upload)
•
Top Viruses (Download)
Top Viruses (Upload) widget
Report displays list of top viruses uploaded along with the number of connections.
 
View report from Reports →Virus →FTP Viruses.
 
Report is displayed as graph as well as in tabular format.
 
Bar graph displays number of connections per virus while tabular report contains following information:
•
Virus: Name of the virus identified by monitored device
•
Connections: Number of connections to the virus
To view the following granular reports of a particular virus drill down by clicking virus name in the graph or the virus name hyperlink in the table:
•
Top Servers
•
Top Hosts
•
Top Users
•
Top Files
 
The granular reports page displays multiple reports in the widgets form which can again be drilled down to view the detailed report
