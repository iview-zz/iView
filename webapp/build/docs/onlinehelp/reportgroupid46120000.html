Reports for UTM Product Category : Web Usage : Reports by Application, Category and User
Reports by Application, Category and User
To view the following granular reports of a particular application, category and domain, go to Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Users widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Domains
•
Top Contents
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the user, category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Users widget→ User.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs for selected user, domain, category and application drill down by clicking the domain name in the graph or hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the user, category and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Categories widget → Category name → Top Users widget→ User.
 
Bar graph displays amount of data transferred by each content type while tabular report contains following information:
•
Content: Type of the content e.g. audio, video, text etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of domains for selected content, user, category and application drill down by clicking the content type in the graph or hyperlink in the table.
