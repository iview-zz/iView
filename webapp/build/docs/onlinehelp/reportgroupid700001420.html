Reports for Web Server Product Category : Apache : Reports by Content Type and Response Code
Reports by Content Type and Response Code
To view the following granular reports of a particular content type and response code, go to Reports → Apache → Top Content Types→ Content Types→ Top Response Code widget and drill down by clicking response code in the graph or the hyperlink in the table:
•
Top URLs
•
Top User Agents
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected content type and response code.
 
View the report from Reports → Apache → Top Content Types → Content Type→ Top Response Codes widget→ Response Code.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top User Agents widget
Widget report displays number of connections and amount of data transfer per user agent by the selected content type and response code.
 
View the report from Reports → Apache → Top Content Types → Content Type→ Top Response Codes widget→ Response Code.
 
Bar graph displays amount of data transfer per user agent while tabular report contains following information:
•
User Agent: URL of the user agent
•
Connections: Number of connections to the user agent
•
Bytes: Amount of data transfer
