Report by Virus (Upload), Server and User
Report displays number of connections per host for the selected user, server and virus (upload).
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Upload) widget →Viruses→Top Servers widget → Server →Top Users widget → User.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
File: Name of the file
•
Connections: Number of connections to the host
