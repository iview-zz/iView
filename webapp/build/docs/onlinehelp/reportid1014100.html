Reports for UTM Product Category : Spam : Report by Spam Recipient and User
Report by Spam Recipient and User
Report displays amount of data transferred by the selected user and spam recipient.
 
View the report from Reports → Spam → Top Spam Recipients → Recipient→Top Users widget → User.
 
Bar graph displays number of connections per sender corresponding to selected user and spam recipient while tabular report contains following information:
•
Time: Time in YYYY-MM-DD HH:MM:SS format.
•
Sender: Email ID of the sender
•
Subject : Subject line of the email
•
Destination: URL name or IP address of the destination
•
Host: IP address of the host
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Count: Number of counts to the sender
