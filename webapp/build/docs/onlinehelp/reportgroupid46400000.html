Reports for UTM Product Category : Web Usage : Reports by Application and Content
Reports by Application and Content
To view the following granular reports of a particular application and domain, go to Reports → Web Usage→ Top Applications → Application → Top Contents widget and drill down by clicking the content type in the graph or hyperlink in the table.
•
Top Domains
•
Top Users
•
Top Categories
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the content type and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Contents widget→ Content Type.
 
Bar graph displays amount of data transferred per domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of users for each domain drill down by clicking the domain name in the graph or hyperlink in the table.
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the content type and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Contents widget → Content Type.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred through the user
 
To view the following granular reports of each user drill down by clicking the User in the graph or hyperlink in the table:
•
Top Domains i.e. number of connections and amount of data transferred per domain for the selected user, content and application
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user, content and application
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the content and application.
 
View the report from Reports → Web Usage→ Top Applications → Application → Top Contents widget → Content Type.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the following granular reports of each user drill down by clicking the category name in the graph or hyperlink in the table:
•
Top Users i.e. number of connections and amount of data transferred per user by the selected category, content and application
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected category, content and application
