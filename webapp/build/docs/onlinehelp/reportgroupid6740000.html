Reports for UTM Product Category : FTP Usage : Report by Host and Server
Report by Host and Server
To view the following granular reports of a selected host, go to Reports → FTP Usage → Top FTP Servers → Server → Top FTP Hosts and drill down by clicking the bar against in the graph or the host hyperlink in the table:
•
Top Files Uploaded
•
Top Files Downloaded
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files (Uploaded) by Host and Server
Report displays a list of files uploaded through the selected host and server. It also displays number of connections established to upload the file and the time and amount of data uploaded.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server → Top FTP Hosts widget → Top Files Uploaded widget.
Top Files (Downloaded) by Host and Server
Report displays a list of files downloaded through the selected host and server. It also displays number of connections established to download the file and the time and amount of data downloaded.
 
View the report from Reports → FTP Usage → Top FTP Servers → Server →Top FTP Host widget → Top Files Downloaded widget.
