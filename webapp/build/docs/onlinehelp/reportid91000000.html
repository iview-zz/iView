Severity wise break-down
Severity wise break-down report enables to view the severity of the attack that has hit the system and gives the detailed disintegration of attacks, attackers, victims and applications through individual reports under severity. It also gives information about detected and dropped attacks.
 
View the reports from Reports → Attacks → Severity wise break-down.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
The bar graph displays the number of connections under each severity, while tabular report contains following information:
•
Severity: Severity level of the attack attempt. Predefined level are:
EMERGENCY - System is not usable
ALERT - Action must be taken immediately
CRITICAL - Critical condition
ERROR - Error condition
WARNING - Warning condition
NOTICE - Normal but significant condition
INFORMATION – Informational
DEBUG - Debug - level messages
•
Connections: Number of connections under each severity
 
To view the following granular reports under each severity level, drill down by clicking severity in the graph or severity number hyperlink in the table:
•
Top Attacks
•
Top Attackers
•
Top Victims
•
Top Applications
•
Top Detected Attacks
•
Top Dropped Attacks
 
The granular reports page displays multiple reports in the widgets which can be again drilled down to view the detailed report
