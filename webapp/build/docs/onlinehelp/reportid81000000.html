Reports for UTM Product Category : Blocked Web Attempts : Top Denied Web Users
Top Denied Web Users
Report displays a list of top users who made the most attempts to access the blocked sites.
 
View the report from Reports → Blocked Web Attempts →Top Denied Web Users.
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays list of users while tabular report contains following information:
•
User: Name of the User
•
Connections: Number of Connections
 
To view the following granular reports of a particular user, drill down by clicking the bar against user name in the graph or the user name hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top URLs
•
Top Hosts
•
Applications
 
The granular reports page displays multiple reports in the widgets form which can again be drilled down to view the detailed report.
