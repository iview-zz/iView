Reports for UTM Product Category : FTP Usage : Top FTP Servers
Top FTP Servers
Report displays the list of top servers through which the most FTP traffic is generated.
 
View the report from Reports → FTP Usage →Top FTP Servers.
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data transferred through each server while tabular report contains following information:
•
Server: Server IP address
•
Connections: Number of Connections
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular Server; drill down by clicking the bar against server in the graph or the hyperlink in the table:
•
Top Files Uploaded
•
Top Files Downloaded
•
Top FTP Users
•
Top FTP Hosts
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
