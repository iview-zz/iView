Reports for EPS Product Category : File Antivirus : Reports by Application
Reports by Application
To view the following granular reports of a particular application, go to Reports → File Antivirus → Top Application and drill down by clicking application name in the graph or the application name hyperlink in the table:
•
Top Action Taken
•
Top Path of Application
•
Top Host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Action Taken widget
Widget report displays number of connections per action taken by the selected application.
 
View the report from Reports → File Antivirus → Top Application→ Name of Application.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the application. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and application, click the action taken in the graph or hyperlink in the table.
•
Top Path of Application
•
Top Host
Top Path of Application widget
Widget report displays number of connection per virus-infected file by the selected application.
 
View the report from Reports → File Antivirus → Top Application→ Name of Application.
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays location of the virus infected file
•
Connections: Number of connections to the application path
To view the following granular reports for selected path and application, click the application path in the graph or hyperlink in the table.
•
Top Action Taken
•
Top Host
Top Host widget
Widget report displays number of connection per host by the selected application.
 
View the report from Reports → File Antivirus → Top Application→ Name of Application.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host : Displays IP address of the host
•
Connections: Number of connections to the host
To view the following granular reports for selected host and application, click the host IP address in the graph or hyperlink in the table.
•
Top Action Taken
•
Top Path of Application
