Reports for EPS Product Category : USB Control : Reports by Host IP Addresses
Reports by Host IP Addresses
To view the following granular reports of a particular host, go to Reports → USB Control → Top Host IP Addresses and drill down by clicking host in the graph or the host hyperlink in the table:
•
Top Users
•
Action Taken
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user by the selected host.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the following granular reports for selected user and host, click the user in the graph or hyperlink in the table.
•
Action Taken
•
Top USB Serial Number
Action Taken widget
Widget report displays number of connection per action by the selected host.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product. Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and host, click the action taken in the graph or hyperlink in the table.
•
Top Users
•
Top USB Serial Number
Top USB Serial Number widget
Widget report displays number of connection per USB serial number by the selected host.
 
View the report from Reports → USB Control → Top Host IP Addresses→ Host IP.
 
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the following granular reports for selected USB serial number and host IP address, click the USB serial number in the graph or hyperlink in the table.
•
Top Users
•
Action Taken
