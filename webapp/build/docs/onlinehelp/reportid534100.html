Reports for UTM Product Category : Mail Usage : Report by User and Destination
Report by User and Destination
Report displays amount of data transferred by selected destination and user.
 
View the report from Reports → Mail Usage → Top Mail Users → User →Top Destinations → Destination.
 
Bar graph displays amount of data transferred by each sender correspondeing to selected destination and user while tabular report contains following information:
•
Time: Time in YYYY: MM: DD HH:MM:SS format
•
Sender: Email ID of the sender
•
Recipient: Colon (: ) separated email IDs of recipient(s)
•
Subject : Subject line of the email
•
Host: IP address of the host
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application or group unassigned application please refer to Add Custom Application
•
Size: Size of the email
