Reports for EPS Product Category : File Antivirus : Reports by Action Taken and Application
Reports by Action Taken and Application
To view the following granular reports of a particular application and action taken, go to Reports → File Antivirus → Top Action Taken → Action → Top Application widget and drill down by clicking application in the graph or the hyperlink in the table:
•
Top Path of Application
•
Top Host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Path of Application widget
Widget report displays number of connection per virus-infected file by the selected application and action.
 
View the report from Reports → File Antivirus → Top Action Taken → Action → Top Application widget → Name of Application.
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays location of the virus infected file
•
Connections: Number of connections to the application path
 
To view the list of hosts for the selected application, action and path click the application path in the graph or hyperlink in the table.
Top Host widget
Widget report displays number of connection per host by the selected application and action.
 
View the report from Reports → File Antivirus → Top Action Taken → Action → Top Application widget → Name of Application.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host : Displays IP address of the host
•
Connections: Number of connections to the host
 
To view the path of application for the selected application, action and host click the host IP in the graph or hyperlink in the table.
