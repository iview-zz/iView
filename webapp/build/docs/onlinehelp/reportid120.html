Top Deny Rules
Top Deny Rules report enables to view traffic denied for various applications, users, hosts and destinations. The administrator can identify top denied firewall rules and fine-tune data transfer policy, Internet access policy and bandwidth policy
 
View the report from Reports → Firewall Rules Based Usage → Top Deny Rules.
 
The bar graph displays number of connections per denied firewall rule while the tabular report contains following information:
•
Rule ID: Firewall rule ID as defined in the monitored device. If firewall rule ID is not defined in the monitored device then it will display firewall policy name.
•
Connections: Number of connections per firewall rule
 
To get in depth Deny Report of a particular firewall rule, drill down by clicking the graph or the Rule ID hyperlink.
