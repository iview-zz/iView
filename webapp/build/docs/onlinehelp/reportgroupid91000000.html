Reports by Severity wise break-down
To view the following granular reports under particular severity, go to Reports → Attacks → Severity wise break-down and drill down by clicking severity level in the graph or the severity number hyperlink in the table:
•
Top Attacks
•
Top Attackers
•
Top Victims
•
Top Applications
•
Top Detected Attacks
•
Top Dropped Attacks
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Attacks widget
Widget report displays number of connections for every attack under the selected severity.
 
View the reports from Reports → Attacks → Severity wise break-down → Top Attacks.
 
Bar graph displays the type of attack while tabular report contains following information:
•
Attack: Name of the attack
•
Connections: Number of connections under the attack
 
To view the list of attackers who launched the attack under particular severity, drill down by clicking the Attack name in the graph or hyperlink in the table.
Top Attackers widget
Widget report displays number of connections for every attacker under the selected severity.
 
View the reports from Reports → Attacks → Severity wise break-down → Top Attackers.
 
Bar graph displays the IP address of the attacker, while tabular report contains following information:
•
Attacker:IP address of the attacker
•
Connections: Number of connections by the attacker
 
To view the list of attacks, the attacker has launched under particular severity, drill down by clicking the Attacker IP in the graph or hyperlink in the table.
Top Victims widget
Widget report displays number of connections for every victim under attack for the selected severity.
 
View the reports from Reports → Attacks → Severity wise break-down → Top Victims.
 
Bar graph displays the IP address of the victim, while tabular report contains following information:
•
Victim:IP address of the victim
•
Connections: Number of connections by the victim
 
To view the list of attacks, the victim has launched under particular severity, drill down by clicking the victim IP in the graph or hyperlink in the table.
Top Applications widget
Widget report displays number of connections for every application under attack for the selected severity.
 
View the reports from Reports → Attacks → Severity wise break-down → Top Applications.
 
Bar graph displays the name of the application under attack, while tabular report contains following information:
•
Application: Name of the application under attack
•
Connections: Number of connections for the application
 
To view the list of attacks, the application has launched under particular severity, drill down by clicking the application name in the graph or hyperlink in the table.
