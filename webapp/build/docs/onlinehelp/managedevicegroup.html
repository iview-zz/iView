Device Group
Device group is logical grouping of devices available in the product category. It is mainly based on device location, device model or device administrator. E.g., Group of all the UTM devices deployed at same geographical location to get network visibility of that area.
 
E.g. all the devices sending logs of Inventory department of the Organization can be grouped to generate consolidated report.
All the devices deployed at same geographical location can be grouped to get network visibility of that location.
Use System → Configuration → Device Group page to add and manage device groups in Cyberoam iView.
 
Screen Components
•
Device Group: Name of device group.
•
Description: Description of the grouping.
•
Device names: Name of the member devices.
•
Add button: Click to add a new device group.
•
Delete button: Click to delete a device group.

Add Device Group
 
•
Super Admin privilege required to add a device group
 
1.
Go to System → Configuration → Device Group
2.
Click Add to create a new device group.
3.
Specify name of device group. Device group name can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
4.
Specify device group description, if required.
5.
Click drop-down to select the product category. Possible product categories are UTM,
Access Gateway, EPS and Web Server.
6.
Click checkbox against device(s) to be added OR click Select All to add all devices
7.
Click OK to add devices in the device group.
8.
Click Add to create a new device group.
•
You need to select at least one device to create a device group.
•
A single device can be a member of multiple device groups
 
Update Device Group
 
•
Super Admin privilege required to update an existing device group
 
1.
Go to System → Configuration → Device Group and click device to be updated.
2.
Refer Add Device Group for information on each parameter.
 
•
All fields except category are editable
 
Delete Device Group
 
•
Super Admin privilege required to delete an existing device group
 
1.
Go to System → Configuration → Device Group
2.
Click checkbox against the device group(s) to be deleted OR click Select All to delete all device groups.
3.
Click Delete.
 
•
A group can be deleted without removing devices from the group. Removing a group will not remove the devices from Cyberoam iView.
