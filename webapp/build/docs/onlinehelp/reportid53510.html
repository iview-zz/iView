Reports for UTM Product Category : Mail Usage : Report by User and Application
Report by User and Application
Report displays amount of data transferred by selected application and user.
 
View the report from Reports → Mail Usage → Top Mail Users → User →Top Applications → Application.
 
Bar graph displays amount of data transferred by each sender correspondeing to selected application and user while tabular report contains following information:
•
Time: Time in YYYY: MM: DD HH:MM:SS format
•
Sender: Email ID of the sender
•
Recipient: Colon (: ) separated email IDs of recipient(s)
•
Subject : Subject line of the email
•
Host: IP address of the host
•
Destination: IP address of the destination
•
Size: Size of the email
