Reports for UTM Product Category : Blocked Web Attempts : Reports by Denied Categories
Reports by Denied Categories
To view the granular reports of particular category, go to Reports → Blocked Web Attempts Reports → Top Denied Categories and drill down by clicking the bar against category name in the graph or category name hyperlink in the table:
•
Top Domains
•
Top Users
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains Widget
Widget displays list of denied domains along with the number of connections, which comes under the selected category.
View the report from Reports → Blocked Web Attempts → Top Denied Categories → Category.
 
Bar graph displays various denied domains, which falls under a category while tabular report contains following information:
•
Domain: Displays the various denied web domains
•
Connections: Number of Connections
 
To view the list of users who tried to access sites under a domain, drill down by clicking the domain in the graph or domain hyperlink in the table.
Top Users Widget
Widget displays list of denied users along with the number of connections, which comes under the selected category.
 
View the report from Reports → Blocked Web Attempts → Top Denied Categories → Category.
 
Bar graph displays various users who tried to access sites under a denied category while tabular report contains following information:
•
User: Displays the various denied users
•
Connections: Number of Connections
 
To view the list of domains under a category which users tried to access, drill down by clicking the domain in the graph or domain hyperlink in the table.
