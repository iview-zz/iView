Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Sites by MAC Address, User and Liveip
Top Sites by MAC Address, User and Liveip
Report displays number of connections per web site by the selected MAC address, user and client IP.
 
View the report from Reports → Web Surfing Logger → Top MAC Addresses → MAC Address → Top Users widget → User → Top Liveips widget → Liveip.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: Displays URL name of the web sites
•
Connections: Number of connections to the web site
 
To view the list of URLs for selected MAC address, user, client IP and website, click the site name in the graph or hyperlink in the table.
