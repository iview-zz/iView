Reports for EPS Product Category : USB Control : Reports by USB Serial Number
Reports by USB Serial Number
To view the following granular reports of a particular USB serial number, go to Reports → USB Control → Top USB Serial Number and drill down by clicking serial number in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Top Users
•
Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected USB serial number.
 
View the report from Reports → USB Control → Top USB Serial Number→ USB Serial Number.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
To view the following granular reports for selected USB serial number and host, click the host IP address in the graph or hyperlink in the table.
•
Top Users
•
Action Taken
Top Users widget
Widget report displays number of connections per user by the selected USB serial number.
 
View the report from Reports → USB Control → Top USB Serial Number→ USB Serial Number.
Bar graph displays number of connections per user while tabular report contains following information:
•
Users: Name of the user
•
Connections: Number of connections to the user
 
To view the following granular reports for selected user and host, click the user in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Action Taken
Action Taken widget
Widget report displays number of connection per action by the selected USB serial number.
 
View the report from Reports → USB Control → Top USB Serial Number→ USB Serial Number.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and user, click the action taken in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Top Users
