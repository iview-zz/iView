Detail Report by User and Date wise Break-up
Report displays IP address wise surfing time along with amount of data transfer, MAC address, start time, stop time, uploaded data and downloaded data..
 
View the report from Reports → Internet Usage→ Top Surfing Time → Top Users widget → User Name→ Date wise Break-up → Date.
 
Bar graph displays IP address wise surfing time while tabular report contains following information:
•
IP Address: IP address used to transfer data
•
MAC Address: MAC address of the machine used to transfer data
•
Start Time: Data transfer starting time in HH:MM:SS format
•
Stop Time: Data transfer stop time in HH:MM:SS format
•
Used Time: Time required in data transfer in HH:MM:SS format
•
Data Transfer: Total amount of data transferred (Upload + Download) by the IP address
•
Uploaded: Amount of uploaded data
•
Downloaded: Amount of downloaded data
