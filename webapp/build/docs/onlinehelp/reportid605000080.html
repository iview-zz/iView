Reports for EPS Product Category : eMail Scanning : Top User
Top User
 
Report displays list of top users along with the number of connections to each user. Report can be drilled down for user to view top hosts and top actions.
 
View the report from Reports → eMail Scanning → Top User.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Windows login name of the user
•
Connections: Number of connections to the user
 
To view the following granular reports for a user, drill down by clicking user in the graph or the hyperlink in the table:
•
Top Host
•
Top Action
