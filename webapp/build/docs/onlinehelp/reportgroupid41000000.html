Reports for UTM Product Category : Web Usage : Reports by Web Users
Reports by Web Users
To view the following granular reports of a particular user, go to Reports → Web Usage → Top Web Users and drill down by clicking User in the graph or the User hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
•
Top URLs
•
Top Hosts
•
Top Applications
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Categories widget
 
Widget report displays number of connections and amount of data transferred per category for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User.
 
Bar graph displays number of connections and amount of data transferred per category whille tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view the list of domains for the selected category and user, drill down by clicking the category name in the graph or hyperlink in the table:
Top Domains widget
Widget report displays number of connections and amount of data transferred per domain for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User
 
Bar graph displays number of connections and amount of data transferred per domain whille tabular report contains following information:
•
Domain: Domain name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred
 
To view the list of URLs for each domain, drill down by clicking the domain name in the graph or hyperlink in the table.
Top Contents widget
Widget report displays number of connections and amount of data transferred content wise for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User.
 
Bar graph displays number of connections and amount of data transferred content wise whille tabular report contains following information:
•
Content: Type of the content e.g. text, video, audio etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of domains for selected content type and user, drill down by clicking the content type in the graph or hyperlink in the table.
Top URLs widget
Widget report displays number of connections and amount of data transferred per URL for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User.
 
Bar graph displays number of connections and amount of data transferred per URL whille tabular report contains following information:
•
URL: URL name or IP address or the URL
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transferred
Top Hosts widget
Widget report displays number of connections and amount of data transferred per host for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User.
 
Bar graph displays number of connections and amount of data transferred per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view the following granular reports of each host drill down by clicking the host in the graph or hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected host and user
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected host and user
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected host and user
•
Top URLs i.e. number of connections and amount of data transferred per URL for the selected host and user
•
Top Applications i.e. number of connections and amount of data transferred per application for the selected host and user
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
Top Applications widget
Widget report displays number of connections and amount of data transferred per application for the selected user.
 
View the report from Reports → Web Usage → Top Web Users→ User.
 
Bar graph displays number of connections and amount of data transferred per application whille tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To defined an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the following granular report of each application, drill down by clicking the application name in the graph or hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected application and user
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected application and user
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
