Reports by Clientip, Liveip and User
To view the following granular reports of a particular client IP, MAC address and user, go to Reports → Web Surfing Logger → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Sites
•
Top MAC Addresses
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Sites widget
Widget report displays number of connections per web site by the selected client IP, live IP and user.
.
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users→ User.
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
 
To view the list of URLs for selected client IP, Live IP user and site, drill down by clicking the site name or IP address in the graph or hyperlink in the table.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP, live IP and user.
 
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top Liveips widget→ Live IP → Top Users→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of sites for selected client IP, Live IP, user and MAC address drill down by clicking the MAC address in the graph or hyperlink in the table.
