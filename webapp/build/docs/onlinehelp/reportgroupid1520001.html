Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group (Upload)
Reports by Application Group (Upload)
To view the following granular reports of a particular application group, go to Reports → Source Host Based Usage → Top Application Groups (Upload) and drill down by clicking application group name in the graph or the application group hyperlink in the table:
•
Top Application (Upload)
•
Top User (Upload)
•
Top Destination (Upload)
•
Top Host (Upload)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Upload) widget
Widget report displays amount of data uploaded and number of connections per application of the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group (Upload) → Application Group (Upload).
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded
 
To view the following granular reports of each application, drill down by clicking the application name in the graph or hyperlink in the table:
•
Top User (Upload) i.e. amount of data uploaded and number of connections per user for selected application of the application group
•
Top Destination (Upload) i.e. amount of data uploaded and number of connections per destination by the selected application of the application group
•
Top Host (Upload) i.e. amount of data uploaded and number of connections per host by the application of the application group
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top User (Upload) widget
Widget report displays amount of data uploaded and number of connections per user for the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group (Upload) → Application Group (Upload).
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data uploaded
 
To view the following granular reports of each user, drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Application (Upload) i.e. amount of data uploaded and number of connections per applications by the user for the selected application group
•
Top Destination (Upload) i.e. amount of data uploaded and number of connections per destination by the user for the selected application group
•
Top Host (Upload) i.e. amount of data uploaded and number of connections per host by the user for the selected application group
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destination (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group (Upload) → Application Group (Upload).
 
Bar graph displays amount of data uploaded through each destination while tabular report contains following information:
•
Destination: Site URL or Destination IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
 
To view the following granular reports of each destination, drill down by clicking the destination link in the graph or hyperlink in the table:
•
Top Application (Upload) i.e. amount of data uploaded and number of connections per application by the selected application group and destination
•
Top User (Upload) i.e. amount of data uploaded and number of connections per user by the selected application group and destination
•
Top Host (Upload) i.e. amount of data uploaded and number of connections per host by the selected application group and destination
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Host (Upload) widget
Widget report displays amount of data uploaded and number of connections per host for the selected application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group (Upload) → Application Group (Upload).
Bar graph displays amount of data transferred through each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
 
To view the following granular reports of each host, drill down by clicking the host link in the graph or hyperlink in the table:
•
Top Applications (Upload) i.e. amount of data uploaded and number of connections per application by the selected application group and host
•
Top Destinations (Upload) i.e. amount of data uploaded and number of connections per destination by the selected application group and host
•
Top Users (Upload) i.e. amount of data uploaded and number of connections per user by the selected application group and host
