Reports for UTM Product Category : Web Usage : Reports by Content and Category
Reports by Content and Category
To view the following granular reports of a particular category and content, go to Reports → Web Usage →Top Contents → Content Type →Top Categories widget and drill down by clicking the category name in the graph or hyperlink in the table.
•
Top Users
•
Top Domains
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the selected category and content type.
 
View the report from Reports → Web Usage→ Top Contents → Content Type → Top Categories widget → Category Name.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred through the user
 
To view the list of domains from where the data is transferred through the selected user, category and content, drill down by clicking the User in the graph or hyperlink in the table.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the selected category and content type.
 
View the report from Reports → Web Usage→ Top Contents → Content Type → Top Categories widget → Category Name.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address or the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs from where the data is transferred through the selected domain, category and content drill down by clicking the domain name in the graph or hyperlink in the table.
