Reports for UTM Product Category : VPN : Top Users
Top Users
 
Report displays amount of data transfer along with username, internal network, local and remote interface information.
 
View the report from Reports → VPN → Top Connections.
 
Bar graph displays amount of data transfer per user while tabular report contains following information:
•
User: Username for the VPN connection
•
Internal Network: IP address of the internal network
•
Local Interface IP: IP address of the local interface
•
Local Gateway IP: IP Address of the local gateway
•
Remote Interface IP: IP address of the remote interface
•
Remote Network: IP address of the remote network
•
Up Time: Up time for the VPN tunnel
•
Down Time: Down time for the VPN tunnel
•
Bytes Sent: Number of bytes sent by the user
•
Bytes Received : Number of bytes received by the user
