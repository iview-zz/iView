Reports for Access Gateway Product Category : Web Surfing Logger Report : Reports by Clientip and User
Reports by Clientip and User
To view the following granular reports of a particular client IP and user, go to Reports → Web Surfing Logger → Top Clientips → Client IP → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top MAC Addresses
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP and user,
.
View the report from Reports → Web Surfing Logger → Top Clientip → Client IP → Top Users widget→ User.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
 
To view the list of sites for selected MAC address, client IP and user, drill down by clicking the MAC address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected client IP and user.
 
View the report from Reports → Web Surfing Logger → Top Clientips → Client IP → Top Users widget→ User.
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the list of sites for selected live IP, client IP and user, drill down by clicking the live IP address in the graph or hyperlink in the table.
