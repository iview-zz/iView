Reports for UTM Product Category : Blocked Attempts : Reports by Application Group and Application
Reports by Application Group and Application
To view the following granular reports of a particular user and host, go to Reports → Blocked Attempts → Top Denied Application Groups→ Application Group → Top Applications widget and drill down by clicking the application in the graph or hyperlink in the table.
•
Top Users
•
Top Destinations
•
Top Hosts
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user by the selected application and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Applications widget→ Application.
 
Bar graph displays number of connections to each user in the group while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view the list of destinations and list of hosts for the selected user, application and application group, drill down by clicking the user name in the graph or hyperlink in the table.
Top Destinations Widget
Widget report displays number of connections per destination by the selected application and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Applications widget→ Application.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of users and list of hosts for the selected destination, application, and application group, drill down by clicking the destination in the graph or hyperlink in the table.
Top Hosts Widget
Widget report displays number of connections per host by the selected application and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Application Groups → Application Group→ Top Applications widget→ Application.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view the list of destinations and list of users for the selected host, application and application group drill down by clicking the host in the graph or hyperlink in the table.
