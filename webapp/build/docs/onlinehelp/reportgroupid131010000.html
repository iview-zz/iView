Reports for UTM Product Category : IM Usage : Reports by Protected Contact
Reports by Protected Contact
To view the following granular reports of a particular protected contact, go to Reports → IM Usage → Top Protected Contact and drill down by clicking the protected contact or protocol hyperlink in the table:
•
Top Conversation
•
Top File Sent
•
Top File Received
•
Top Webcam Request Sent
•
Top Webcam Request Received
•
Top User
•
Top Host
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date . Report date can be changed from the top most row of the page.
Top Conversation widget
Widget report displays list of peer contacts and total number of sent and received message for the selected protected contact
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays IM username of peer contact and total number of IM messages for the peer contact while tabular report contains following information:
•
Peer Contact: IM Username of Peer contact
•
#Messages : Total number of IM messages
 
To view full conversation for the selected peer contact, drill down by clicking the peer contact hyperlink in the table.
 
•
To view full conversation you need to select Logging Level as ‘Full Data’ at the time of conversation rule creation in monitored device.
Top Files Sent widget
Widget report displays list of name of the files sent along with recipien and number of times the file has been sent for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of times the file has been sent while tabular report contains following information:
•
File Name: Name of the file
•
Sent to: IM username of the file recipient
•
#Count: Number of times the file has been sent
Top Files Received widget
Widget report displays list of name of files received along with sender and number of times the file has been received for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of times the file has been received while tabular report contains following information:
•
File Name: Name of the file
•
Received from: IM username of the file sender
•
#Count: Number of times the file has been received
Top Webcam Request Sent widget
Widget report displays list of recipients of webcam request along with number of webcam requests for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of webcam requests to a particular recipient while tabular report contains following information:
•
Request sent to: IM username of the webcam request recipient
•
#Messages: Number of webcam requests
Top Webcam Request Received widget
Widget report displays list of sender of webcam request along with number of webcam requests for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of webcam requests from a particular sender while tabular report contains following information:
•
Request sent from: IM username of the webcam request sender
•
#Messages: Number of webcam requests
Top User widget
Widget report displays list of users along with number of messages for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of messages for a particular user while tabular report contains following information:
•
User: Username of monitored device user
•
#Messages: Total number of messages
Top Host widget
Widget report displays list of hosts along with number of messages for the selected protected contact.
 
View the report from Reports → IM Usage → Top Protected Contact → Protected Contact.
 
Bar graph displays number of messages for a particular host while tabular report contains following information:
•
Host: IP address of the host
•
#Messages: Total number of messages
