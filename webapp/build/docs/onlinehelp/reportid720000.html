Reports for UTM Product Category : Blocked Attempts : Top Denied Application Groups
Top Denied Application Groups
Report displays list of top denied application groups along with the number of connections that generate the most traffic for various users, destinations and hosts.
 
View the report from Reports →Blocked Attempts → Top Denied Application Groups.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per denied application group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application group
 
To view the following granular reports of a particular application group drill down by clicking application group in the graph or the application group hyperlink in the table:
•
Top Applications
•
Top Users
•
Top Destinations
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
