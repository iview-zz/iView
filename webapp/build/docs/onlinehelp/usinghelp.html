Using Online Help
Cyberoam iView Online Help is a Web-based help which can be viewed from any of the pages of Web Admin console. It is installed automatically with the software.
To view context sensitive (page-specific) help topic
•
Click the Help icon that appears in the top right corner of the screen
•
Press F1
