Reports for UTM Product Category : Web Usage : Top Users by Category Type and User Group
Top Users by Category Type and User Group
Report displays a list of user, the number of connections to each user and the amount of data transferred by the selected user group and category type.
 
View the report from Reports → Web Usage → Top Category Types → Category Type → Top User Groups→ User Group.
 
To view following granular reports for the selected user, user group and category type, drill down by clicking the user hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user, user group and category type
•
Top Domains i.e. number of connections and amount of data transferred per domain for the selected user, user group and category type
•
Top Contents i.e. number of connections and amount of data transferred per content type for the selected user, user group and category type
