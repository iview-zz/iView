Reports for UTM Product Category : Blocked Attempts : Reports by Destination and User
Reports by Destination and User
To view the following granular reports of a particular destination and user, go to Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Users widget and drill down by clicking the user in the graph or hyperlink in the table.
•
Top Applications
•
Top Hosts
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected destination and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Users widget → User.
 
Bar graph displays number of connections to each application in the group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of hosts for the selected application, user and destination, drill down by clicking the application in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays number of connections per host by the selected destination and user.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations→ Destination → Top Users widget → User.
 
Bar graph displays number of connections to each host in the group while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the user
 
To view the list of applications for the selected host, user and destination, drill down by clicking the host in the graph or hyperlink in the table.
