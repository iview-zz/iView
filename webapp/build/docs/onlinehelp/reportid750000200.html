Reports for Access Gateway Product Category : Web Surfing Logger Report : Top Sites by Liveip, User and Clientip
Top Sites by Liveip, User and Clientip
Report displays number of connections per web site by the selected live IP, user and client IP.
 
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget → User → Top Clientip widget → Client IP.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: Displays URL name of the web sites
•
Connections: Number of connections to the web site
 
To view the list of Top URLs for selected live IP, user, client IP and website, click the site name in the graph or hyperlink in the table.
