Reports for Access Gateway Product Category : Net Capture Report : Reports by Liveip
Reports by Liveip
To view the following granular reports of a particular live IP, go to Reports → Net Capture → Top Liveips and drill down by clicking Live IP in the graph or the Live IP hyperlink in the table:
•
Top Destinations
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destinations widget
Widget report displays number of connections per destination by the selected live IP.
 
View the report from Reports → Net Capture → Top Liveips → Live IP.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of Top Destination Ports for selected live IP and destination, click the destination IP address in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user for the selected live IP.
 
View the report from Reports → Net Capture → Top Liveips → Live IP.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular live IP and user, go to Reports → Net Capture → Top Liveips → Live IP → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Destinations
•
Top Clientip
•
Top MAC Address
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
