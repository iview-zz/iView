Top Attacks by Applications and Victims
Report displays top attacks launched by the application under particular victim.
 
View the report from Reports → Attacks → Top Victims → Top Applications → Application.
 
The bar graph displays number of connections and type of the attacks while tabular report contains following information:
•
Attack:Name of the attack
•
Attacker: IP address of the attacker
•
User: Username of the victim. If username is not defined in monitored device then it will display traffic generated by ‘Unknown’ user.
•
Severity: Severity of the attack by application
•
Action: Number of times action is performed
•
Connections: Number of Connections by the victim
