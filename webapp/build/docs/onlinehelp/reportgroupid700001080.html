Reports for Web Server Product Category : Apache : Reports by User Agent and Server
Reports by User Agent and Server
To view the following granular reports of a particular user agent and server, go to Reports → Apache → Top User Agents → User Agent → Top Servers widget and drill down by clicking server in the graph or the hyperlink in the table:
•
Top URLs
•
Top Content Types
•
Top Response Codes
•
Top Clients
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected user agent and server.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget→ Server.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Content Types widget
Widget report displays number of connections and amount of data transfer per content type by the selected user agent and server.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget→ Server.
 
Bar graph displays amount of data transfer per content type while tabular report contains following information:
•
Content Type: Type of content
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transfer
Top Response Codes widget
Widget report displays number of connections and amount of data transfer per response code by the selected user agent and server.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget→ Server.
 
Bar graph displays amount of data transfer per response code while tabular report contains following information:
•
Response Code: Response code number
•
Connections: Number of connections to the response code
•
Bytes: Amount of data transfer
 
Top Clients widget
Widget report displays number of connections and amount of data transfer per client by the selected user agent and server.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget→ Server.
 
Bar graph displays amount of data transfer per client while tabular report contains following information:
•
Client IP: IP address of the client
•
Connections: Number of connections to the client
•
Bytes: Amount of data transfer
To view the list of URLs and list of response codes for selected user agent, server and client click the client IP in the graph or hyperlink in the table.
