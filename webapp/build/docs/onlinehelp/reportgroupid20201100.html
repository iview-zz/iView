Reports by Application and User (Upload)
To view the following granular reports of a particular application and user, go to Reports → Application Based Usage → Top Applications (Upload) → Application → Top Users (Upload) widget and drill down by clicking the User in the graph or hyperlink in the table.
•
Top Destination (Upload)
•
Top Host (Upload)
The granular reports page displays multiple reports in the widgets form.
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Destination (Upload) widget
Widget report displays amount of data uploaded and number of connections per destination by the selected user and application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Users (Upload)→ User.
 
Bar graph displays amount of data uploaded to each destination while tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data uploaded
Top Host (Upload) widget
Widget report displays amount of data uploaded and number of connections per host by the selected user and application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Users (Upload) → User.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
