Reports for UTM Product Category : Internet Usage : Top Surfing Time
Top Surfing Time
To view the following granular data transfer reports, go to Reports → Internet Usage → Top Surfing Time:
•
Top User Groups
•
Top Users
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date . Report date can be changed from the top most row of the page.
Top User Groups widget
Widget report displays list of user groups along with surfing time.
View the report from Reports → Internet Usage → Top Surfing Time.
 
Bar graph displays user group wise surfing time while tabular report contains following information:
•
User Group: Name of the user group as defined in the monitored device. If the User Group is not defined in monitored device then it will be considered as traffic generated by ‘Unknown’ user group
•
Used Time: Total Surfing time in HH:MM:SS format
 
To view list of users for the selected user group, drill down by clicking the User Group hyperlink in the table.
Top Users widget
Widget report displays list of users along with the amount of data transferred.
View the report from Reports → Internet Usage → Top Surfing Time.
 
Bar graph displays user wise data transfer while tabular report contains following information:
•
User Name: Name of the user as defined in the monitored device. If the User is not defined in monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Data Transfer: Total amount of data transferred (Upload + Download) by the user
•
Uploaded: Amount of uploaded data
•
Downloaded: Amount of downloaded data
 
To view datewise break-up for the selected user, drill down by clicking the User Name hyperlink in the table.
