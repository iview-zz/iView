Reports for UTM Product Category : Web Usage : Reports by User Group and Application
Reports by User Group and Application
To view the following granular reports of a particular user group and application, go to Reports → Web Usage → Top Web User Groups → User Group → Top Applications widget and drill down by clicking the application hyperlink in the table.
•
Top Categories
•
Top Contents
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the selected application and user group.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group → Top Applications widget → Application.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the list of domains from where the data is transferred through the selected category, application and user, drill down by clicking the category name hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the selected application and user group.
 
View the report from Reports → Web Usage→ Top Web User Groups → User Group → Top Applications widget → Application.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of domains from where the data is transferred through the selected content type, application and user, drill down by clicking the content type hyperlink in the table.
