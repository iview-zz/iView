Reports for UTM Product Category : Web Usage : Top Domains by Host, User and Content
Top Domains by Host, User and Content
Report displays a list of domains, the number of connections to each domain and the amount of data transferred through the selected content type, host and by the user.
 
View the report from Reports → Web Usage → Top Web Hosts widget → Top Web Users widget → User → Top Hosts widget →Host → Top Contents → Content Type.
 
To view the list of URLs for the selected domain, content type, host and user, drill down by clicking the domain name in the graph or hyperlink in the table.
