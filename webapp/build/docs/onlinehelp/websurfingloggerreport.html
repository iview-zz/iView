Reports for Access Gateway Product Category : Web Surfing Logger Report
Web Surfing Logger Report
 
Web Surfing Logger reports provide snapshot of Internet usage in your network. It helps identify highest traffic generators who are affecting overall network traffic. It provides statistics based on traffic generated by Internet usage.
 
These reports can help determining Internet behavior and provide a basis for fine-tuning the configuration to better control traffic flow.
 
View web surfing logger reports from Reports → Web Surfing Logger.
 
It enables to view traffic generated by:
•
Top Users
•
Top Liveips
•
Top Sites
•
Top Clientips
•
Top MAC Addresses
