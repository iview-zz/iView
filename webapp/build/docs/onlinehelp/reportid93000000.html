Reports for UTM Product Category : Attacks : Top Attackers
Top Attackers
Top Attackers report enables to view the details of the attacker who have hit the system and gives the detailed disintegration of attacks, victims and applications through individual reports.
 
View the reports from Reports → Attacks → Top Attackers.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
The bar graph displays the number of connections by each attacker, while tabular report contains following information
•
Attacker: IP address of the attacker
•
Connections: Number of connections for each attacker
 
To view the following granular reports for each attacker, drill down by clicking attacker in the graph or attacker IP hyperlink in the table:
•
Top Attacks
•
Top Victims
•
Top Applications
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
