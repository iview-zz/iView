Reports for UTM Product Category : FTP Usage : Reports by FTP Hosts (Download)
Reports by FTP Hosts (Download)
To view the following granular reports of a particular host, go to Reports → FTP Usage → Top FTP Hosts (Download) and drill down by clicking the bar against in the graph or the host hyperlink in the table:
•
Top Files
•
Top Users
•
Top Servers
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files widget
Widget report displays amount of data downloaded through various files by a selected host.
 
View the report from Reports → FTP Usage →Top FTP Hosts (Download) → Host.
 
Bar graph displays amount of data downloaded by each host while tabular report contains following information:
•
File: Name of file which is downloaded by that selected host
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of users and servers, which downloaded the data through that file for a selected host, drill down by clicking the file name in the graph or hyperlink in the table.
Top Servers widget
Widget report displays amount of data downloaded through a various servers by the selected host.
 
View the report from Reports → FTP Usage → FTP Users (Download) → Host.
 
Bar graph displays amount of data downloaded through each server by the selected host while tabular report contains following information:
•
Server: Server IP address through which the data is downloaded
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of users and files, which downloaded the data through that server for a selected host, drill down by clicking the bar in the graph or hyperlink in the table.
Top Users widget
Widget report displays amount of data downloaded by various users through the selected host.
 
View the report from Reports → FTP Usage → FTP Users (Download) → Host.
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Name of users who have uploaded the data through the selected host. N/A will be displayed instead of username if username is not defined in the monitored device. If more than one such user exists then traffic details of all such users will be grouped and displayed under N/A.
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of servers and files, which downloaded the data through that user for a selected host, drill down by clicking the bar in the graph or hyperlink in the table.
