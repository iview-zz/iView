Reports by Destination Port, Destination, Live IP and Client IP
To view the following granular reports of a particular destination port, destination and live IP go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Liveips Widget→ Liveips → Top Clientips Widget and drill down by clicking the client IP in the graph or hyperlink in the table.
•
Top MAC Addresses
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected client IP, live IP, destination and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Destinations widget→ Destination→Top Liveips widget→ Live IP →Top Clientips widget→ Client IP.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC Address of the user’s machine
•
Connections: Number of connections per MAC address
Top Users widget
Widget report displays number of connections per user by the selected destination port, destination, client IP and live IP.
.
View the report from Reports → Net Capture → Top Destination Ports→ Destination Port → Top Destinations widget→ Destination→Top Liveips widget→ Live IP →Top Clientips widget→ Client IP.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
