System : Configuration : Mail Server
Mail Server
 
To send the report notification through E-mail, you need to configure SMTP server.
Use System → Configuration → Mail Server Configuration to configure mail server to send report notifications.
 
•
Super Admin privilege required to configure a mail server
 
1.
Go to System → Configuration → Mail Server Configuration to configure Cyberoam iView mail server
2.
Specify IP address and port number of SMTP mail server, a port number must be a numeric value in between 1 to 65535.
3.
Specify name of the mail sender.
4.
Specify E-mail ID of the sender. Email ID can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
5.
Click checkbox to enable SMTP authentication, if required.
6.
If SMTP authentication is enabled, specify username and password. Username can be any combination of alphanumeric characters and special characters “_”, “@” and “.”.Password field can not be blank.
7.
Click Apply to save the configuration.
