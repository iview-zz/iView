Reports for EPS Product Category : File Antivirus : Reports by Action Taken
Reports by Action Taken
To view the following granular reports of a particular action, go to Reports → File Antivirus → Top Action Taken and drill down by clicking action in the graph or the action hyperlink in the table:
•
Top Application
•
Top Path of Application
•
Top Host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application widget
Widget report displays number of connections per application by the selected action.
 
View the report from Reports → File Antivirus → Top Action Taken→ Action.
 
Bar graph displays number of connections per application while tabular report contains following information:
•
Name of Application: Displays name of the application as defined in the EPS product.
•
Connections: Number of connections to the application
 
To view the following granular reports for selected action and application, click the application in the graph or hyperlink in the table.
•
Top Path of Application
•
Top Host
Top Path of Application widget
Widget report displays number of connections per virus-infected file by the selected action.
 
View the report from Reports → File Antivirus → Top Action Taken→ Action.
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays location of the virus infected file
•
Connections: Number of connections to the application path
 
To view the following granular reports for selected path and action, click the application path in the graph or hyperlink in the table.
•
Top Application
•
Top Host
Top Host widget
Widget report displays number of connection per host by the selected action.
 
View the report from Reports → File Antivirus → Top Action Taken→ Action.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host : Displays IP address of the host
•
Connections: Number of connections to the host
 
To view the following granular reports for selected host and action, click the host IP address in the graph or hyperlink in the table.
•
Top Application
•
Top Path of Application
