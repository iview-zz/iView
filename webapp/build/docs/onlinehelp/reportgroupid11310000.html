Reports for UTM Product Category : Virus : Reports by Virus
Reports by Virus
To view the following granular reports of a particular virus, go to Reports → Virus →Web Viruses → Virus →Top Viruses widget and drill down by clicking virus name in the graph or the virus name hyperlink in the table:
•
Top Domains
•
Top Users
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Domains widget
Widget report displays number of connections per domain for the selected virus.
 
View report from Reports → Virus →Web Viruses → Top Viruses widget → Virus.
 
Bar graph displays number of connections per domain whille tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
 
To view domain details for the selected domain and virus drill down by clicking the domain name in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user for the selected virus.
 
View report from Reports → Virus →Web Viruses → Top Viruses widget → Virus.
 
Bar graph displays number of connections per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
 
To view user details for the selected user and virus drill down by clicking the user name in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays number of connections per host for the selected virus.
 
View report from Reports → Virus →Web Viruses → Top Viruses widget → Virus.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view host details for the selected host and virus drill down by clicking the host in the graph or hyperlink in the table.
