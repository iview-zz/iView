Reports for UTM Product Category : FTP Usage : Reports by FTP Users (Download)
Reports by FTP Users (Download)
To view the following granular reports of a particular user, go to Reports → FTP Usage → Top FTP Users (Download) and drill down by clicking the bar against user in the graph or the user name hyperlink in the table:
•
Top Files
•
Top Hosts
•
Top Servers
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files widget
Widget report displays amount of data downloaded through various files by the selected user.
 
View the report from Reports → FTP Usage →Top FTP Users (Download) → User.
 
Bar graph displays amount of data downloaded by each user through FTP protocol while tabular report contains following information:
•
File: File name which is downloaded by the selected user
•
Connections: Number of connections established to download the file
•
Bytes: Amount of data downloaded
 
To view the list of hosts and servers, which downloaded the data through that file for a selected user, drill down by clicking the file name in the graph or hyperlink in the table.
Top Servers widget
Widget report displays list of servers and amount of data downloaded through each server by the selected user.
 
View the report from Reports → FTP Usage → FTP Users (Download) → User.
 
Bar graph displays amount of data downloaded by the users while tabular report contains following information:
•
Server: Displays top servers which are used by a user to download
•
Connections: Number of connections established to download
•
Bytes: Amount of data downloaded
 
To view the list of hosts and files, which downloaded the data through that server for a selected user, drill down by clicking the bar in the graph or hyperlink in the table.
Top Hosts widget
Widget report displays list of hosts and amount of data downloaded through each host by the selected user.
 
View the report from Reports → FTP Usage → FTP Users (Download) → User.
 
Bar graph displays amount of data downloaded by each host while tabular report contains following information:
•
Host: Host IP address through which user downloaded
•
Connections: Number of connections
•
Bytes: Amount of data downloaded
 
To view the list of servers and files, which downloaded the data through that host for a selected user, drill down by clicking the bar in the graph or hyperlink in the table.
