Source Host Based Usage Report
 
Source Host based reports provide snapshot of the traffic through your network. It helps identify high volume traffic generators who are affecting overall network traffic. It provides statistics based on traffic generated by host.
 
These reports can help determining traffic flow issues and provide a basis for fine-tuning the configuration to better control traffic flow.
 
View source host based reports from Reports → Source Host Based Usage.
