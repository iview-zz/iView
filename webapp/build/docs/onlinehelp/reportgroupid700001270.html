Reports for Web Server Product Category : Apache : Reports by Content Type
Reports by Content Type
To view the following granular reports of a particular content type, go to Reports → Apache → Top Content Types and drill down by clicking content type in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected content type.
 
View the report from Reports → Apache → Top Content Types → Content Type.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
To view the following granular reports for the selected content type and server click the server IP in the graph or hyperlink in the table
•
Top URLs
•
Top User Agents
•
Top Response Codes

Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected content type.
 
View the report from Reports → Apache → Top Content Types → Content Type.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Referrers widget
Widget report displays number of connections and amount of data transfer per referrer by the selected content type.
 
View the report from Reports → Apache → Top Content Types → Content Type.
 
Bar graph displays amount of data transfer per referrer while tabular report contains following information:
•
Referrer: URL of the referrer
•
Connections: Number of connections to the referrer
•
Bytes: Amount of data transfer
 
To view the list of URLs for the selected referrer and content type, click the referrer URL in the graph or hyperlink in the table.
Top User Agents widget
Widget report displays number of connections and amount of data transfer per user agent by the selected content type.
 
View the report from Reports → Apache → Top Content Types → Content Type.
 
Bar graph displays amount of data transfer per user agent while tabular report contains following information:
•
User Agent: URL of the user agent
•
Connections: Number of connections to the user agent
•
Bytes: Amount of data transfer
To view the list of URLs and list of response codes for selected user agent and content type, click the user agent URL in the graph or hyperlink in the table.
Top Response Codes widget
Widget report displays number of connections and amount of data transfer per response code by the selected content type.
 
View the report from Reports → Apache → Top Content Types → Content Type.
 
Bar graph displays amount of data transfer per response code while tabular report contains following information:
•
Response Code: Response code number
•
Connections: Number of connections to the response code
•
Bytes: Amount of data transfer
 
To view the list of URLs and list of user agents for the selected response code and user agent, click the response code in the graph or hyperlink in the table.
