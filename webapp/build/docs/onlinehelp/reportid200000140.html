Report by SSL VPN User and Web Access User
Report displays list of source IP addresses along with tunnel up and down time.
View report from Reports → SSL VPN →Top SSL VPN Users → Top Web Access Users widget → User Name.
 
Tabular report contains following information:
•
Source IP Address: SSL VPN tunnel source IP address
•
Up Time: SSL VPN tunnel up time
•
Down Time: SSL VPN tunnel down time
