Reports for UTM Product Category : Source Host Based Usage Report : Reports by Host and User (Download)
Reports by Host and User (Download)
To view the following granular reports of a particular host and user, go to Reports → Source Host Based Usage → Top Hosts → Host → Top Users (Download) widget and drill down by clicking the user in the graph or hyperlink in the table.
•
Top Application (Download)
•
Top Destination (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Download) widget
Widget report displays amount of data downloaded and number of connections per application by the selected host and user.
 
View the report from Reports → Source Host Based Usage → Top Hosts → Host → Top Users (Download) widget → User.
 
Bar graph displays amount of data downloaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data downloaded
To view the list of destinations through which the data is downloaded by the selected application, user and host, drill down by clicking the application in the graph or hyperlink in the table.
Top Destination (Download) widget
Widget report displays amount of data downloaded and number of connections per destination by the selected host and user.
 
View the report from Reports → Source Host Based Usage → Top Hosts → Host → Top Users (Download) widget → User.
 
Bar graph displays amount of data downloaded by each application while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the application
•
Bytes: Amount of data downloaded
 
To view the list of applications through which the data is downloaded by the selected destination, user and host, drill down by clicking the destination in the graph or hyperlink in the table.
