Reports for Access Gateway Product Category : Net Capture Report : Top Destinations by MAC Address, Clientip, User and Liveip
Top Destinations by MAC Address, Clientip, User and Liveip
Report displays a list of destinations along with the number of hits to each destinations for the selected client IP, MAC address, user and live IP.
 
View the report from Reports → Net Capture → Top MAC Addresses → MAC address → Top Clientips widget→ Client IP → Top Users→ User → Top Liveips → Live IP.
 
To view the list of destination ports for selected client IP, MAC address, user, live IP and destination, drill down by clicking the destination in the graph or hyperlink in the table.
