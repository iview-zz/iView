System : Configuration : User
User
 
Use the System → Configuration → Users page to configure and maintain administrators, set user's administrative access, password maintenance.
 
Screen components:
•
Username: Login name preferred by the user.
•
Name: Name of the user.
•
Role: Role defines administrative access privilege.
•
Email: Email address of the user.
•
Created By: Username of the Administrator who added this user.
•
Last Login Time: Last time when the user had logged in.
•
Add button: Click to add a new user.
•
Delete button: Click to delete a user.
 
 
Cyberoam iView supports three types of user roles:
•
Super admin – Default account. No additional account can be created.
•
Admin - Only administrator with the Super Admin role can add Admin roles
•
Viewer - Adminsitrator with Super Admin and Admin roles can add Viewer roles

Use the System → Configuration → Users page to:
•
Add User
•
Update User
•
Delete User
 
Add User
 
•
Super Admin or Admin privilege required to add a new user.
 
1.
Go to System → Configuration → Users and click Add
2.
Specify name of the user
3.
Specify username, which uniquely identifies the user and will be used for login. Username can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
4.
Specify password. Password is case sensitive.
5.
Specify the same password to confirm spelling.
6.
Specify a valid Email ID. The Email ID can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
7.
Select user role from the drop down. Roles define administrative access privilege. Refer to Privilege Matrix for details
8.
Select the device or device group, which the user can manage. Click checkbox against the device/device group(s) OR click Select All to select all device/device group(s).
9.
Click Add to add the user. Depending on the role, user will be able to configure and view the information of the selected devices only.
•
Multiple administrators can have rights to manage same device.
•
In case of simultaneous updation by multiple administrators, last updation will be saved
 
Update User
 
•
Super Admin or Admin privilege required to update a user.
 
1.
Go to System → Configuration → Users and click the username to be modified.
2.
Refer Add User for information on each parameter.
•
All the fields except Username are editable.
 
Delete User
 
•
Super Admin or Admin privilege required to delete a user.
 
1.
Go to System → Configuration → Users
2.
Click checkbox against the user(s) to be deleted OR click Select All to delete all the users.
3.
Click Delete
 
•
Default account- Super Admin can not be deleted
