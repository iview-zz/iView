Reports by Application and Destination (Upload)
To view the following granular reports of a particular application and destination, go to Reports → Application Based Usage → Top Applications (Upload) → Application → Top Destinations (Upload) widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top User(Upload)
•
Top Host (Upload)
The granular reports page displays multiple reports in the widgets form
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top User (Upload) widget
Widget report displays amount of data uploaded and number of connections per user by the selected destination and application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Destinations (Upload)→ Destination.
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data uploaded
Top Host (Upload) widget
Widget report displays amount of data uploaded and number of connections per host by the selected destination and application.
 
View the report from Reports → Application Based Usage → Top Applications (Upload) → Application → Top Destinations (Upload) → Destination.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
