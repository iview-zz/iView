Reports for Access Gateway Product Category : Web Surfing Logger Report : Reports by Liveip
Reports by Liveip
 
To view the following granular reports of a particular live IP, go to Reports → Web Surfing Logger → Top Liveips and drill down by clicking Live IP in the graph or the Live IP hyperlink in the table:
•
Top Sites
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Sites widget
Widget report displays number of connections per web site by the selected live IP.
 
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP.
 
Bar graph displays number of connections per web site while tabular report contains following information:
•
Sites: URL name or IP address of the web site
•
Connections: Number of connections to the web site
 
To view the list of Top URLs for selected live IP and web site, click the site name in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user for the selected live IP.
View the report from Reports → Web Surfing Logger → Top Liveips → Live IP.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports of a particular live IP and user, go to Reports → Web Surfing Logger → Top Liveips → Live IP → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Sites
•
Top Clientips
•
Top MAC Address
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
