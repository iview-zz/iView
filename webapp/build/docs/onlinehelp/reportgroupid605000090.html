Reports for EPS Product Category : eMail Scanning : Reports by User
Reports by User
To view the following granular reports of a particular user, go to Reports → eMail Scanning → Top User and drill down by clicking user in the graph or the hyperlink in the table:
•
Top Host
•
Top Action
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host widget
Widget report displays number of connections per host by the selected user.
 
View the report from Reports → eMail Scanning → Top User→ User.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view list of actions for the selected host and user drill down by clicking user in the graph or the hyperlink in the table.
Top Action widget
Widget report displays number of connections per action by the selected user.
View the report from Reports → eMail Scanning → Top User→ User.
Bar graph displays number of connections per action while tabular report contains following information:
•
Action: Action taken by the EPS product. Possible actions are
o
Allowed
o
Quarantined
o
Deleted
•
Connections: Number of connections to the action.
 
To view list of the hosts for the selected user and action drill down by clicking action in the graph or the hyperlink in the table.
