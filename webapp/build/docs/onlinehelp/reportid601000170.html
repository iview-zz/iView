Reports for EPS Product Category : Application Control : Top File Names
Top File Names
Report displays list of top file names along with the number of connections. Report can be drilled down for file name to view top host IP address, top action taken and top programs.
 
View the report from Reports → Application Control → Top File Names.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per file while tabular report contains following information:
•
File Names : Name of the file
•
Connections: Number of connections to the file
 
To view the following granular reports of a file, drill down by clicking file name in the graph or the hyperlink in the table:
•
Top Host IP Address
•
Top Action Taken
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
