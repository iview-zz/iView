Reports for EPS Product Category : File Antivirus : Reports by Action Taken and Host
Reports by Action Taken and Host
To view the following granular reports of a particular action and host, go to Reports → File Antivirus → Top Action Taken → Action → Top Host widget and drill down by clicking host IP in the graph or the hyperlink in the table:
•
Top Application
•
Top Path of Application
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application widget
Widget report displays number of connections per application by the selected action and host.
 
View the report from Reports → File Antivirus → Top Action Taken → Action → Top Host widget → Host
 
Bar graph displays number of connections per application while tabular report contains following information:
•
Name of Application: Displays name of the application as defined in the EPS product.
•
Connections: Number of connections to the application
 
To view the path of application for the selected action, path and application, click the name of application in the graph or hyperlink in the table.
Top Path of Application widget
Widget report displays number of connection per virus-infected file by the selected action and host.
 
View the report from Reports → File Antivirus → Top Action Taken → Action → Top Host widget → Host
 
Bar graph displays number of connections per path while tabular report contains following information:
•
Path of Application: Displays the path of the application
•
Connections: Number of connections to the application path
 
To view the list of applications for the selected action, host and path, click the application path in the graph or hyperlink in the table.
