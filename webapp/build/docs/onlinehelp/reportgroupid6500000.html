Reports for UTM Product Category : FTP Usage : Reports by FTP Hosts (Upload)
Reports by FTP Hosts (Upload)
To view the following granular reports of a selected host, go to Reports → FTP Usage Reports → Top FTP Hosts (Upload) and drill down by clicking the bar against in the graph or the user name hyperlink in the table:
•
Top Files
•
Top Users
•
Top Servers
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Files widget
Widget report displays amount of data uploaded through various files by the selected host.
 
View the report from Reports → FTP Usage →Top FTP Hosts (Upload) → Host.
 
Bar graph displays amount of data uploaded by each host through FTP in the group while tabular report contains following information:
•
File: Name of the file which is uploaded by the selected host
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of users and servers, which uploaded the data through that file for a selected host, drill down by clicking the file name in the graph or hyperlink in the table.
Top Servers widget
Widget report displays list of servers and amount of data uploaded through each server by the selected host.
 
View the report from Reports → FTP Usage → FTP Users (Upload) → Host.
 
Bar graph displays amount of data uploaded by the host through various servers while tabular report contains following information:
•
Server: Server IP address used by a user to upload
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of users and files, which uploaded the data through that server for a selected host, drill down by clicking the bar in the graph or hyperlink in the table.
Top Users widget
Widget report displays amount of data uploaded by various users through the selected host.
 
View the report from Reports → FTP Usage → FTP Users (Upload) → Host.
 
Bar graph displays amount of data uploaded by each user while tabular report contains following information:
•
User: User name who have uploaded the data through the selected host. N/A will be displayed instead of username if username is not defined in the monitored device. If more than one such user exists then traffic details of all such users will be grouped and displayed under N/A.
•
Connections: Number of connections
•
Bytes: Amount of data uploaded
 
To view the list of servers and files, which uploaded the data through that user for a selected host, drill down by clicking the bar in the graph or hyperlink in the table.
