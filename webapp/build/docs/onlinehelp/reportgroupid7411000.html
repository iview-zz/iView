Reports for UTM Product Category : Blocked Attempts : Reports by Destination, Application Group and Application
Reports by Destination, Application Group and Application
To view the following reports of a destination, application group and application, go to Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Applications widget → Application.
•
Top Users
•
Top Hosts
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays number of connections per user by the selected destination, application and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Applications widget → Application.
 
Bar graph displaysnumber of connections to each destination while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
Top Hosts widget
Widget report displays number of connections per host by the selected destination, application and application group.
 
View the report from Reports → Blocked Attempts → Top Denied Destinations →Destination→ Top Application Groups widget→ Top Applications widget → Application.
 
Bar graph displays number of connections to each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
