Reports for UTM Product Category : Web Usage : Reports by Domain
Reports by Domain
To view the following granular reports of a particular domain, go to Reports → Web Usage → Top Domains and drill down by clicking domain name in the graph or the domain name hyperlink in the table:
•
Top Users
•
Top Contents
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
 
Widget report displays number of connections and amount of data transferred per user for the selected domain.
 
View the report from Reports → Web Usage → Top Domains→ Domain name.
 
Bar graph displays number of connections and amount of data transferred per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the list of URLs for the selected user and domain, drill down by clicking the User in the graph or hyperlink in the table.
Top Contents widget
Widget report displays number of connections and amount of data transferred content wise for the selected domain.
 
View the report from Reports → Web Usage → Top Domains→ Domain name.
 
Bar graph displays number of connections and amount of data transferred content wise whille tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of users for the selected content and domain, drill down by clicking the content type in the graph or hyperlink in the table.
