Reports for UTM Product Category : Blocked Attempts : Reports by Denied Users
Reports by Denied Users
To view the following granular reports of a particular user, go to Reports → Blocked Attempts → Top Denied Users and drill down by clicking User in the graph or the User hyperlink in the table:
•
Top Application Groups
•
Top Destinations
•
Top Hosts
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application Groups widget
Widget report displays number of connections per application group for the selected user.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User.
 
Bar graph displays number of connections per application group whille tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application group
 
To view the following granular reports of each application group, drill down by clicking the application group name in the graph or hyperlink in the table:
•
Top Applications i.e. number of connections per application by the selected user and application group.
•
Top Destinations i.e. number of connections per destination by the selected user and application group.
•
Top Hosts i.e. number of connections per host by the selected user and application group.
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
Top Destinations widget
Widget report displays number of connections per destination for the selected user.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User.
 
Bar graph displays number of connections per destination whille tabular report contains following information:
•
Destination: URL name or IP address of the destination
•
Connections: Number of connections to the destination
To view the following granular reports of each destination, drill down by clicking the destination in the graph or hyperlink in the table:
•
Top Applications i.e. number of connections per application by the selected user and destination
•
Top Hosts i.e. number of connections per host by the selected user and destination
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
Top Hosts widget
Widget report displays number of connections per host for the selected user.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User.
 
Bar graph displays number of connections per host whille tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
 
To view the following granular reports of each host, drill down by clicking the host in the graph or hyperlink in the table:
•
Top Applications i.e. number of connections per application by the selected user and host
•
Top Destinations i.e. number of connections per destination by the selected user and host
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report
