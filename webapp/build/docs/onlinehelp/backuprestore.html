System : Archives : Backup Management
Backup Management
 
Cyberoam iView allows the Administrator to take backup of historical archived logs to improve usage of available storage space. The Administrator can download and restore the backup files as and when required.
Use System → Archives → Backup Management to
•
Backup Archive Files
•
Download Backup File
•
Restore Backup Files
Backup Archive Files
•
Super Admin or Admin privilege required to take backup of archived file
•
Unloading of the archived file is required to take backup
 
1.
Go to System → Archives → Backup Management
2.
Select one or multiple network devices from the device drop down given at top left.
3.
Select date range from the given calendar
4.
Click OK to save the selection.
5.
Click Go to view archived files for selected date
6.
Select check box against file(s) and click Backup Now to take backup of the files in Cyberoam iView machine.
7.
Select Full Day Backup checkbox and click Backup Now to take full day back up at once.
•
If the archived file is partially loaded, then backup of only unloaded data will be taken.
•
Once the backup file is created, Administrator can download the backup file on any machine including Cyberoam iView machine itself.
 
Download Backup Files
 
•
Super Admin or Admin privilege required to download backup of archive file
 
1.
Go to System → Archives → Backup Management.
2.
Click Download Backup Files.
3.
Click Download against the filename. The file will be downloaded on local machine from where Cyberoam iView Web Admin Console is accessed.
•
To help identity the backup of each device, Backup file is named as <Device ID_ YYYYMMDDStartHourEndHour> Where:
o
Device ID - As configured in Cyberoam iView
o
YYYYMMDD - Date as displayed on Archive Files page under Date column
o
Start Hour End Hour – Time as displayed on Archive Files page under File Details column
Restore Backup Files
 
•
Super Admin or Admin privilege required to restore backup files
 
1.
Go to System → Archives → Backup Management.
2.
Browse file to be restored. Click Add to restore multiple files.
3.
Click Restore.
