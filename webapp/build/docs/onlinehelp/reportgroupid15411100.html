Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group, Host (Download) and Destination (Download)
Reports by Application Group, Host (Download) and Destination (Download)
To view the following reports of a particular host, user and application group go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Hosts (Download) widget →Host →Top Destination (Download) widget →Destination.
•
Top Application (Download)
•
Top User (Download)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Download) widget
Widget report displays amount of data downloaded and number of connections per application by the selected host, destination and application group.
 
View the report from Reports → Top Application Groups → Application Group → Top Hosts (Download) widget →Host →Top Destination (Download) widget →Destination
Bar graph displays amount of data downloaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data downloaded
Top User (Download) widget
Widget report displays amount of data downloaded and number of connections per user by the selected host, destination and application group.
 
View the report from Reports → Top Application Groups → Application Group → Top Hosts (Download) widget →Host →Top Destination (Download) widget →Destination.
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the destination
•
Bytes: Amount of data downloaded
