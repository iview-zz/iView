Reports for UTM Product Category : Blocked Attempts : Reports by User and Destination
Reports by User and Destination
To view the following granular reports of a particular user and destination, go to Reports → Blocked Attempts → Top Denied Users→ User → Top Destination widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Application
•
Top Host
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Applications widget
Widget report displays number of connections per application by the selected user and destination.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User → User → Top Destination widget.
 
Bar graph displays number of connections to each application in the group while tabular report contains following information:
•
Application Group: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
 
To view the list of hosts for the selected application, destination and user, drill down by clicking the application name in the graph or hyperlink in the table.
Top Hosts Widget
Widget report displays number of connections per host by the selected user and destination.
 
View the report from Reports → Blocked Attempts → Top Denied Users→ User → Top Destinations widget.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the destination
 
To view the list of applications for the selected host, destination and user, drill down by clicking the host in the graph or hyperlink in the table.
