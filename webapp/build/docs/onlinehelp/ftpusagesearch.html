Search Reports
1.
Go to Search → FTP
2.
Specify file transfer type. Available options:
o
Download
o
Upload
o
Any
3.
Specify search criteria: Available options:
o
User Name
o
File Name
4.
Specify username or file name to be searched. If the user name or file name is not specified then search result will be displayed for all the files and users.
5.
Click Search
Refer to FTP Search Report to view report.
