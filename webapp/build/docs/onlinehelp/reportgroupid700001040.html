Reports for Web Server Product Category : Apache : Reports by URL and Response Code
Reports by URL and Response Code
To view the following granular reports of a particular URL and client go to Reports → Apache → Top URLs → URL → Top Response Codes widget and drill down by clicking the response code in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected URL and response code.
 
View the report from Reports → Apache → Top URLs → URL → Top Response Codes widget →Response Code.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
Top Clients widget
Widget report displays number of connections and amount of data transfer per client by the selected URL and response code.
 
View the report from Reports → Apache → Top URLs → URL → Top Response Codes widget →Response Code.
 
Bar graph displays amount of data transfer per client while tabular report contains following information:
•
Client IP: IP address of the client
•
Connections: Number of connections to the client
•
Bytes: Amount of data transfer
