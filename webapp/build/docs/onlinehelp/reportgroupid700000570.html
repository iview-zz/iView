Reports for Web Server Product Category : Apache : Reports by Client
Reports by Client
To view the following granular reports of a particular client, go to Reports → Apache → Top Clients and drill down by clicking client IP in the graph or the hyperlink in the table:
•
Top Servers
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Servers widget
Widget report displays number of connections and amount of data transfer per server by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per server while tabular report contains following information:
•
Server: IP address or domain name of the server
•
Connections: Number of connections to the server
•
Bytes: Amount of data transfer
To view the following granular reports for selected client and server click the server IP in the graph or hyperlink in the table.
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Referrers widget
Widget report displays number of connections and amount of data transfer per referrer by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per referrer while tabular report contains following information:
•
Referrer: URL of the referrrer
•
Connections: Number of connections to the referrer
•
Bytes: Amount of data transfer
 
 
To view the list of URLs for the selected referrer and client, click the referrer URL in the graph or hyperlink in the table.
Top User Agents widget
Widget report displays number of connections and amount of data transfer per user agent by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per user agent while tabular report contains following information:
•
User Agent: URL of the user agent
•
Connections: Number of connections to the user agent
•
Bytes: Amount of data transfer
To view the list of content types, list of response codes and list of URLs for selected user agent and client, click the user agent URL in the graph or hyperlink in the table.
Top Content Types widget
Widget report displays number of connections and amount of data transfer per content type by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per content type while tabular report contains following information:
•
Content Type: Type of content
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transfer
To view the list of response codes and list of URLs for selected content type and client, click the content type in the graph or hyperlink in the table.
Top Response Codes widget
Widget report displays number of connections and amount of data transfer per response code by the selected client.
 
View the report from Reports → Apache → Top Clients → Client IP.
 
Bar graph displays amount of data transfer per response code while tabular report contains following information:
•
Response Code: Response code number
•
Connections: Number of connections to the response code
•
Bytes: Amount of data transfer
 
To view the list of URLs, list of user agents and list of content types for the selected response code and client, click the response code in the graph or hyperlink in the table.
