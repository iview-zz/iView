System : Configuration : Bookmark Management
Add Bookmark Group
 
1.
Go to System → Configuration → Bookmark Management.
2.
Click Add to create new bookmark group.
3.
Specify Bookmark Group Name, name can be any combination of alphanumeric characters and special characters “_”, “@” and “.”
4.
Click Add to add new bookmark group.
