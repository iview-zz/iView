Reports for Web Server Product Category : Apache : Reports by User Agent, Server and Client
Reports by User Agent, Server and Client
To view the following granular reports of a particular user agent, server and client go to Reports → Apache → Top User Agents → User Agent → Top Servers widget →Server → Top Clients widget. and drill down by clicking client IP in the graph or the hyperlink in the table:
•
Top URLs
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top URLs widget
Widget report displays number of connections and amount of data transfer per web page by the selected user agent, server and client.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget →Server → Top Clients widget →Client IP.
 
Bar graph displays amount of data transfer per web page while tabular report contains following information:
•
URL: URL name of the web page
•
Connections: Number of connections to the URL
•
Bytes: Amount of data transfer
Top Response Codes widget
Widget report displays number of connections and amount of data transfer per response code by the selected user agent, server and client.
 
View reports from Reports → Apache → Top User Agents → User Agent → Top Servers widget →Server → Top Clients widget →Client IP.
 
Bar graph displays amount of data transfer per response code while tabular report contains following information:
•
Response Code: Response code number
•
Connections: Number of connections to the response code
•
Bytes: Amount of data transfer
