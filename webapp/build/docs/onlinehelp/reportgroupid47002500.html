Reports for UTM Product Category : Web Usage : Reports by User Group
Reports by User Group
To view the following granular reports of a particular user group, go to Reports → Web Usage → Top Web User Groups and drill down by clicking User Group hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
•
Top Users
•
Top URLs
•
Top Hosts
•
Top Applications
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
 
Widget report displays number of connections and amount of data transferred per category for the selected user group.
 
View the report from Reports → Web Usage → Top Web User Groups→ User Group.
 
Bar graph displays amount of data transferred per category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view the list of domains for the selected category and user group, drill down by clicking the category name hyperlink in the table:
Top Domains widget
Widget report displays number of connections and amount of data transferred per domain for the selected user group.
 
View the report from Reports → Web Usage → Top Web User Groups→ User Group
 
Bar graph displays amount of data transferred per domain while tabular report contains following information:
•
Domain: Domain name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred
 
To view the list of URLs for each domain, drill down by clicking the domain name hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred content wise for the selected user group.
 
View the report from Reports → Web Usage → Top Web User Groups→ User Group.
 
Bar graph displays amount of data transferred content wise while tabular report contains following information:
•
Content: Type of the content e.g. text, video, audio etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of domains for selected content type and user group, drill down by clicking the content type hyperlink in the table.
Top Users widget
Widget report displays amount of data transferred per user for the selected user group.
 
View the report from Reports → Web Usage → Top Web User Groups→ User Group.
 
Bar graph displays amount of data transferred per user while tabular report contains following information:
•
User: User name as defined in the monitored device. If the user is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the following granular reports of each user drill down by clicking the User hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user group and user
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected host and user
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected user group and user
•
Top URLs i.e. number of connections and amount of data transferred per URL for the selected user group and user
•
Top Hosts i.e. number of connections and amount of data transferred per host for the selected user group and host
•
Top Applications i.e. number of connections and amount of data transferred per application for the selected user group and user
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.

Top Applications widget
Widget report displays number of connections and amount of data transferred per application for the selected user group.
 
View the report from Reports → Web Usage → Top Web User Groups→ User Group.
 
Bar graph displays amount of data transferred per application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam-iView. If application is not defined in Cyberoam-iView then this field will display application identifier as combination of protocol and port number. To defined an unidentified application and group an unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data transferred
 
To view the following granular report of each application, drill down by clicking the application name hyperlink in the table:
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected application and user group
•
Top Contents i.e. number of connections and amount of data transferred content wise for the selected application and user group
