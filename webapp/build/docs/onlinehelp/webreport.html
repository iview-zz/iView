Reports for EPS Product Category : Web Report
Web Report
Web Report provides snapshot of Internet traffic monitored by EPS product. It provides reports based on web hosts, websites and actions.
 
These reports can help determining Internet utilization along with user’s Internet behavior. It also provides a basis for fine-tuning the configuration to better control traffic flow.
 
View web reports from Reports → Web Report.
 
It enables to view traffic generated by:
•
Top Web Host
•
Web Sites Visited
•
Action Taken
