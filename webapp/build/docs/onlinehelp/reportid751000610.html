Reports for Access Gateway Product Category : Net Capture Report : Top MAC Addresses
Top MAC Addresses
Report displays list of top MAC addresses along with the number of connections that generate the most traffic for various users, and client IPs.
 
View the report from Reports → Net Capture → Top MAC Addresses.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per MAC address while tabular report contains following information:
•
MAC Address: MAC address of the user’s machine
•
Connections: Number of connections to the MAC address
 
To view the following granular reports of a particular MAC address, drill down by clicking MAC address in the graph or the MAC Address hyperlink in the table:
•
Top Users
•
Top Clientips
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
