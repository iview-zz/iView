Syslog Server Port
•
Super Admin or Admin privilege required to access and manage Syslog Server Port page.
 
To provide flexibility in the port configuration for receiving syslogs Cyberoam iView allows configuration of Garner Port.
Use System → Configuration → Port to configure port number to receive syslog stream
 
1.
Go to System → Configuration → Port to specify port number to receive syslog steam.
2.
Click Save to save changes.
