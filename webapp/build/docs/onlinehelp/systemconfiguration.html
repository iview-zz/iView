System : Configuration
Configuration
 
This section describes how to add network devices to Cyberoam iView, configure Cyberoam iView for generating reports for added devices and customize Cyberoam iView as per requirement.
This section covers the following topics:
•
User Management – Create and manage role based Cyberoam iView users;
•
Device Group Management – Create and manage groups of devices to generate reports;
•
Device Management – Add and manage network devices;
•
Application Groups – Create and manage protocol groups;
•
Mail Server Configuration – Configure email server to send report notifications;
•
Custom View -– Create and manage customized view of reports;
•
Report Notification – Send reports in PDF format to configured email address;
•
Data Management – Manage disk and data required to generate reports;
•
Bookmark Management – Create and manage report and report group bookmarks
•
Logs – Configure MD5 checksum
•
Syslog Server Port – Configure syslog port number to receive syslog data stream
•
Backup and Restore (Detail Data ) – Take backup of detailed report data
