Reports for UTM Product Category : FTP Usage : Top Servers by File (Upload) and User
Top Servers by File (Upload) and User
Report displays a list of servers and hosts through which the selected file is downloaded, time in YYYY:MM:DD HH:MM:SS format, the number of connections established to that server through the host to upload the file by the selected user.
 
View the report from Reports → FTP Usage Reports→ Top FTP Users (Upload) → User → Top Files widget.
