Reports for UTM Product Category : Web Usage : Reports by Category and Domain
Reports by Category and Domain
To view the following granular reports of a particular domain and category, go to Reports → Web Usage → Top Categories → Category Name → Top Domains widget and drill down by clicking the domain in the graph or hyperlink in the table.
•
Top Users
•
Top Contents
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the selected domain and category.
 
View the report from Reports → Web Usage→ Top Categories → Category Name → Top Domains widget → Domain Name.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred through the user
 
To view the list of URLs from where the data is transferred through the selected user, domain and category drill down by clicking the user name in the graph or hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections content wise by the selected domain and category.
 
View the report from Reports → Web Usage→ Top Categories → Category Name → Top Domains widget → Domain name.
 
Bar graph displays amount of data transferred by each contetnt type while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred through the content type
 
To view the list of users from where the data is transferred through the selected content, domain and category, drill down by clicking the content type in the graph or hyperlink in the table.
