Reports for UTM Product Category : Attacks : Reports by Applications
Reports by Applications
To view the following granular reports of particular application under attack; go to Reports → Attacks → Top Applications used by Attacks and drill down by clicking application name in the graph or the application name hyperlink in the table:
•
Top Attacks
•
Top Attackers
•
Top Victims
•
Top Detected Attacks
•
Top Dropped Attacks
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Attacks widget
Widget report displays number of connections for every attack under the selected application.
 
View the reports from Reports → Attacks → Top Appications used by Attacks → Top Attacks.
 
Bar graph displays the type of attack while tabular report contains following information:
•
Attack: Name of the attack
•
Connections: Number of connections under the attack
 
To view the list of attackers who launched the attack under particular application, drill down by clicking the Attack name in the graph or hyperlink in the table.
Top Attackers widget
Widget report displays number of connections for every attacker under the selected application.
 
View the reports from Reports → Attacks → Top Appications used by Attacks → Top Attackers.
 
Bar graph displays the IP address of the attacker, while tabular report contains following information:
•
Attacker:IP address of the attacker
•
Connections: Number of connections by the attacker
 
To view the list of attacks, the attacker has launched under particular application, drill down by clicking the Attacker IP in the graph or hyperlink in the table.
Top Victims widget
Widget report displays number of connections for every victim under attack for the selected application.
 
View the reports from Reports → Attacks → Top Appications used by Attacks → Top Victims.
 
Bar graph displays the IP address of the victim, while tabular report contains following information:
•
Victim:IP address of the victim
•
Connections: Number of connections by the victim
 
To view the list of attacks, the victim has launched under particular application, drill down by clicking the victim IP in the graph or hyperlink in the table.
Top Detected Attacks widget
Widget report displays number of connections for every detected attack under the selected application.
 
View the reports from Reports → Attacks → Top Appications used by Attacks → Top Detected Attacks.
 
Bar graph displays the type of detected attack while tabular report contains following information:
•
Attack: Name of the attack
•
Connections: Number of connections under the attack
 
To view the list of attackers who launched the detected attack under particular application, drill down by clicking the Attack name in the graph or hyperlink in the table.
Top Dropped Attacks widget
Widget report displays number of connections for every dropped attack under the selected application.
 
View the reports from Reports → Attacks → Top Appications used by Attacks → Top Dropped Attacks.
 
Bar graph displays the type of dropped attack while tabular report contains following information:
•
Attack: Name of the attack
•
Connections: Number of connections under the attack
 
To view the list of attackers who launched the dropped attack under particular application, drill down by clicking the Attack name in the graph or hyperlink in the table.
