Reports for Access Gateway Product Category : Net Capture Report : Reports by Destination Port, Destination and User
Reports by Destination Port, Destination and User
To view the following granular reports of a particular destination port, destination and user go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Users Widget and drill down by clicking the username in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected destination, destination port and user.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination→ Top Users Widget→ User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of MAC addresses for selected client IP, user, destination and destination port, drill down by clicking the client IP address in the graph or hyperlink in the table.
Top Liveips widget
Widget report displays number of connections per live IP by the selected destination, destination port and user.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget → Destination → Top Users Widget→ User.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the list of client IPs for selected live IP, user, destination port and destination drill down by clicking the live IP in the graph or hyperlink in the table.
