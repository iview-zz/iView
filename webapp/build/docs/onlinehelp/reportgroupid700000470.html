Reports by Server and Response Code
To view the following granular reports of a particular server and content type go to Reports → Apache → Top Servers→ Server → Top Content Types widget and drill down by clicking the content type in the graph or the hyperlink in the table:
•
Top Clients
•
Top URLs
•
Top Referrers
•
Top User Agents
•
Top Content Types
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
 
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Clients widget
Widget report displays number of connections and amount of data transfer per client by the selected server and response code.
 
View the report from Reports → Apache → Top Server → Server → Top Response Codes widget → Response Code.
 
Bar graph displays amount of data transfer per client while tabular report contains following information:
•
Client IP: IP address of the client
•
Connections: Number of connections to the client
•
Bytes: Amount of data transfer
 
To view the list of URLs for the selected client, response code and server, click the client IP in the graph or hyperlink in the table.
