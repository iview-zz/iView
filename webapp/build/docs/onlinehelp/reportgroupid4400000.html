Reports for UTM Product Category : Web Usage : Reports by Content
Reports by Content
To view the following granular reports of a particular content type, go to Reports → Web Usage → Top Contents and drill down by clicking content type in the graph or the content hyperlink in the table:
•
Top Domains
•
Top Users
•
Top Categories
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Top Domains widget
Widget report displays number of connections and amount of data transferred per domain for the selected content.
 
View the report from Reports → Web Usage → Top Contents→Content Type.
 
Bar graph displays number of connections and amount of data transferred per domain whille tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred
 
To view the list of users for the selected domain and content type, drill down by clicking the domain name in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections and amount of data transferred per user for the selected content.
 
View the report from Reports → Web Usage → Top Contents→Content Type.
 
Bar graph displays number of connections and amount of data transferred per user whille tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view following granular reports for the selected user and content type, drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected user and content
•
Top Categories i.e. number of connections and amount of data transferred per category for the selected user and content
Top Categories widget
Widget report displays number of connections and amount of data transferred per category for the selected content.
 
View the report from Reports → Web Usage → Top Contents→Content Type.
 
Bar graph displays number of connections and amount of data transferred per category whille tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
To view following granular reports for the selected category and content type, drill down by clicking the user name in the graph or hyperlink in the table:
•
Top Users i.e. number of connections and amount of data transferred per user for the selected category and content
•
Top Domains i.e. number of connections and amount of data transferred per domain by the selected category and content
