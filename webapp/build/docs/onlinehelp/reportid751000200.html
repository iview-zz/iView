Reports for Access Gateway Product Category : Net Capture Report : Top Destinations by Liveip, User and Clientip
Top Destinations by Liveip, User and Clientip
Report displays number of connections per destination by the selected live IP, user and client IP.
 
View the report from Reports → Net Capture → Top Liveips → Live IP → Top Users widget → User → Top Clientip widget → Client IP.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destinations: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the list of Top Destination Ports for selected live IP, user, client IP and destination, click the destination IP address in the graph or hyperlink in the table.
