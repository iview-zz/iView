Reports for EPS Product Category : USB Control : Reports by User
Reports by User
To view the following granular reports of a particular host, go to Reports → USB Control → Top Users and drill down by clicking user in the graph or the host hyperlink in the table:
•
Top Host IP Addresses
•
Action Taken
•
Top USB Serial Number
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected user.
 
View the report from Reports → USB Control → Top Users→ User.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IPs: IP address of the host
•
Connections: Number of connections to the host
 
To view the following granular reports for selected user and host, click the host in the graph or hyperlink in the table.
•
Action Taken
•
Top USB Serial Number
Action Taken widget
Widget report displays number of connection per action by the selected user.
 
View the report from Reports → USB Control → Top Users→ User.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product.Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the following granular reports for selected action and user, click the action taken in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Top USB Serial Number
Top USB Serial Number widget
Widget report displays number of connection per USB serial number by the selected user.
 
View the report from Reports → USB Control → Top Users→ User.
 
Bar graph displays number of connections per USB serial number while tabular report contains following information:
•
USB Serial Number : Displays serial number of USB
•
Connections: Number of connections to the USB serial number
 
To view the following granular reports for selected USB serial number and user, click the USB serial number in the graph or hyperlink in the table.
•
Top Host IP Addresses
•
Action Taken
