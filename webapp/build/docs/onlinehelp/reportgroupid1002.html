Device Dashboard
Cyberoam iView Device Dashboard provides in depth traffic visibility of the selected device.
 
To view the dashboard of the particular device, drill down by clicking Application in graph or Device hyperlink in the Main Dashboard. It provides following reports in the form of Widgets for the selected device:
 
•
Allowed Traffic Summary
•
Denied Traffic Summary
•
Web Traffic Summary
•
Mail Traffic Summary
•
FTP Traffic Summary
•
Virus Summary
•
Spam Summary
•
IDP Attacks Summary
•
Firewall Denied Summary
•
Content Filtering Denied Summary
 
Device Dashboard is available for devices of UTM product category only
 
Allowed Traffic Summary widget
 
Widget report displays percentage and amount of allowed traffic by each application by the selected device.
 
View report from Dashboard → Main Dashboard → Allowed Traffic Overview widget → Device.
Pie chart displays percentage of allowed traffic, while tabular report contains following information:
•
Allowed Traffic: Name of top applications
•
Bytes: Amount of data transferred through the application
•
Percent: Percentage wise distribution of allowed traffic
Denied Traffic Summary widget
 
Widget report displays number of connections and percentage of denied traffic for the selected device.
View report from Dashboard → Main Dashboard → Allowed Traffic Overview widget → Device.
 
Pie chart displays percentage of denied traffic for the selected device, while tabular report contains following information:
•
Traffic: Type of denied traffic. Possiple types of traffic:
•
Firewall Denied: Traffice denied by firewall
•
Spam: Traffic denied considering as spam
•
Content Filteing Denied: Web and Mail traffic denied
•
IDP Attack: Traffic denied considering as IDP attack
•
Virus: Traffic denied considering as Virus
•
Connection: Number of denied connections
•
Percent: Percentage wise distribution of denied traffic
Web Traffic Summary widget

View report from Dashboard → Main Dashboard → Allowed Traffic Overview widget → Device.
 
Pie chart displays percentage of web traffic for the selected device, while tabular report contains following information:
•
Traffic: Type of web traffic,Possible traffic types:
•
CF Allowed :Web traffic allowed by the monitored device.
•
CF Denied:Web traffic denied by the monitored device.
•
Virus: Virus traffic detected by the monitored device.
•
Bytes: Amount of data transferred by traffic type
•
Percent: Percentage wise distribution of web traffic by traffic type.

Virus Summary widget
 
Widget report displays number of connections and percentage of viruses for the selected device.
 
View report from Dashboard → Main Dashboard → Allowed Traffic Overview→ Device
 
Pie chart displays percentage of virus connections for the selected device, while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam- iView. If application is not defined in Cyberoam- iView then this field will display application identifier as combination of protocol and port number. To define the unidentified application or to group unassigned application please refer to Add Custom Application
•
Connection: Number of connections to the application
•
Percent: Percentage wise distribution of application

IDP Attacks Summary widget
 
Widget report displays list of attacks along with number of connections and percentage of IDP attacks for selected device.
 
View report from Dashboard → Main Dashboard → Allowed Traffic Overview widget → Device.
 
Pie chart displays percentage of IDP attacks for the selected device, while tabular report contains following information:
•
Attack Type: Severity of the attack.
•
0: Emergency : System is not usable
•
1: Alert: Action must be taken immediately
•
2: Critical: Critical condition
•
3: Error: Error condition
•
4: Warning: Warning condition
•
5: Notice: Normal but significant condition
•
6: Info: Informational
•
7: Debug: Debug-level messages
•
Connection: Number of connections to the attack
•
Percent: Percentage wise distribution of IDP attacks by attack type
