Reports for Web Server Product Category : Apache : Top URLs
Top URLs
 
Report displays list of top URLs along with the number of connections that generate the most traffic for various clients, servers, referrers, response codes, content types and user agents.
 
View the report from Reports → Apache → Top URLs.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections and amount of data transfer per referrer while tabular report contains following information:
•
URL: URL name of the web page.
•
Connections: Number of connections
•
Bytes: Amount of data transfer
 
To view the following granular reports of a particular URL, drill down by clicking URL name in the graph or the hyperlink in the table:
•
Top Servers
•
Top Clients
•
Top Referrers
•
Top User Agents
•
Top Content Types
•
Top Response Codes
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
