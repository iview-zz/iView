Top Deny Rules – Application Group Wise
Top Deny Rules report enables to view traffic denied for various application groups.
 
View the report from Reports → Firewall Rules Based Usage → Top Deny Rules- Application Group Wise.
 
The bar graph displays number of connections per denied application group while the tabular
report contains following information:
•
Application: Name of application. If application is not identified by Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define an unidentified application and group an unassigned application please refer to Add Custom Application
•
Rule ID: Firewall rule ID as defined in the monitored device. If firewall rule ID is not defined in the monitored device then it will display firewall policy name.
•
Connections: Number of connections per firewall rule
 
To get in depth Deny Report- Application Group Wise for a particular application group, drill down by clicking the graph or the application group hyperlink.
