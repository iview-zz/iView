Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group and Application
Reports by Application Group and Application
To view the following granular reports of a particular application group and application, go to Reports → Source Host Based Usage → Top Application Group → Application Group → Top Applications widget and drill down by clicking the application name in the graph or hyperlink in the table.
•
Top Users
•
Top Destinations
•
Top Hosts
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Users widget
Widget report displays amount of data transferred and number of connections per user by the selected application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Group → Application Group → Top Applications widget → Application.
 
Bar graph displays amount of data transferred by each user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If User is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections to the user
•
Bytes: Amount of data transferred
 
To view the list of destinations and list of hosts through which the data is transferred by the selected user, application and application group, drill down by clicking the User in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays amount of data transferred and number of connections per destination by the selected application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications widget→ Application.
 
Bar graph displays amount of data transferred to each destination while tabular report contains following information:
•
Destination: Site name or IP address of the destination
•
Connections: Number of connections to the destination
•
Bytes: Amount of data transferred
 
To view the list of users and list of hosts through which the data is transferred by the selected destination, application and application group, drill down by clicking the destination in the graph or hyperlink in the table.
Top Hosts Widget
Widget report displays amount of data transferred and number of connections per host by the selected application and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Applications widget→ Application.
 
Bar graph displays amount of data transferred through each host while tabular report contains following information:
•
Host: Host IP address
•
Connections: Number of connections to the host
•
Bytes: Amount of data transferred
 
To view the list of destinations and list of users through which the data is transferred by the selected host, application and application group, drill down by clicking the host in the graph or hyperlink in the table.
