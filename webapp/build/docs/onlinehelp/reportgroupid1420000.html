Reports for UTM Product Category : Source Host Based Usage Report : Reports by User
Reports by User
To view the following granular reports of a particular user, go to Reports → Source Host Based Usage → Top Users and drill down by clicking user name in the graph or the user name hyperlink in the table:
•
Top Application Groups
•
Top Destinations
•
Top Hosts
•
Top Rules
•
Top Application Groups (Upload)
•
Top Application Groups (Download)
•
Top Destinations (Upload)
•
Top Destinations (Download)
•
Top Hosts (Upload)
•
Top Hosts (Download)
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application Groups widget
Widget report displays amount of data transferred and number of connections per application group by the selected user.
 
View the report from Reports → Source Host Based Usage → Top Users → User.
 
Bar graph displays amount of data transferred by each application in the group while tabular report contains following information:
•
Application Group: Displays name of the application group as defined in Cyberoam iView. If application group is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define the unidentified application and to group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application group
•
Bytes: Amount of data transferred
 
To view the following granular reports of each application in a group, drill down by clicking the application group name in the graph or hyperlink in the table:
•
Top Applications i.e. amount of data transferred and number of connections per application by the selected user and application group
•
Top Destinations i.e. amount of data transferred and number of connections per destination by the selected user and application group
•
Top Hosts i.e. amount of data transferred and number of connections per host by the selected user and application group
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Destinations widget
Widget report displays amount of data transferred and number of connections per destination by the selected user.
 
View the report from Reports → Source Host Based Usage → Top Users → User.
 
Bar graph displays amount of data transferred through each destination while tabular report contains following information:
•
Destination: Site URL or Destination IP address
•
Connections: Number of connections to the destination
•
Bytes: Amount of data transferred
 
To view the following granular reports of each destination, drill down by clicking the destination link in the graph or hyperlink in the table:
•
Top Applications i.e. amount of data transferred and number of connections per application by the selected user and destination.
•
Top Hosts i.e. amount of data transferred and number of connections per host by the selected user and destination.
