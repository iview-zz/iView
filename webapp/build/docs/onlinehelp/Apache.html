Reports for Web Server Product Category : Apache
Apache
 
Apache is an open source Web server developed to provide a secure, efficient and extensible HTTP services in sync with the current HTTP standards. Apache reports provide snapshot of network resource utilization by various web server components. It provides statistics based on traffic generated by servers, clients, content types, response codes and user agents.
 
View Apache reports from Reports → Apache.
 
It enables to view traffic generated by:
•
Top Servers
•
Top Clients
•
Top Referrers
•
Top URLs
•
Top User Agents
•
Top Content Types
•
Top Response Codes
