Reports for UTM Product Category : Spam : Top Spam Recipients
Top Spam Recipients
Report displays list of top spam recipients along with the number of connections that generate the most traffic for various spam senders, users, destinations, source hosts and applications.
 
View the report from Reports →Spam→ Top Spam Recipients.
 
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per spam sender while tabular report contains following information:
•
Recipient: Colon (: ) separated email IDs of the recipients(s)
•
Connections: Number of connections to the recipient
 
To view the following granular reports of a particular spam recipient, drill down by clicking recipient email in the graph or the recipient email hyperlink in the table:
•
Top Spam Senders
•
Top Source Hosts
•
Top Destinations
•
Top Users
•
Top Applications
 
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
