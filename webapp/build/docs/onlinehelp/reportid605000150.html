Reports for EPS Product Category : eMail Scanning : Top Action
Top Action
Report displays list of actions along with the number of connections to each action. Report can be drilled down for action to view top hosts and top users.
 
View the report from Reports → eMail Scanning → Top Action.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Bar graph displays number of connections per action while tabular report contains following information:
•
Action: Action taken by the EPS product. Possible actions are
o
Allowed
o
Quarantined
o
Deleted
•
Connections: Number of connections to the action.
 
To view the following granular reports for an action, drill down by clicking action in the graph or the hyperlink in the table:
•
Top Host
•
Top User
