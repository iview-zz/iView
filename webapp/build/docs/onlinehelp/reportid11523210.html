Report by Virus (Download), User and File
Report displays number of connections per server for the selected file, user and virus (download).
 
View report from Reports → Virus →FTP Viruses → Virus→Top Viruses (Download) widget →Viruses→Top Users widget → User →Top Files widget → File.
 
Bar graph displays number of connections per server whille tabular report contains following information:
•
Server: IP address of the server
•
Host: IP address of the host
•
Connections: Number of connections to the server
