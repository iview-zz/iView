Reports for UTM Product Category : Web Usage : Reports by Category Type and User
Reports by Category Type and User
To view the following granular reports of a particular category type and user go to Reports → Web Usage → Top Category Types → Category Type → Top Users widget and drill down by clicking user hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top Contents
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date. Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays number of connections and amount of data transferred per category for the selected category type and user.
 
View the report from Reports → Web Usage → Top Category Types→ Category Type → Top Users widget → User.
 
Bar graph displays amount of data transferred per category while tabular report contains following information:
•
Category: Name of the category as defined in monitored device
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred
 
To view list of domains for the selected category, user and category type, drill down by clicking the category type hyperlink in the table.
Top Domains widget
Widget report displays amount of data transferred and number of connections per domain by the selected user and category type.
 
View the report from Reports → Web Usage → Top Category Types→ Category Type → Top Users widget → User.
 
Bar graph displays amount of data transferred by each domain while tabular report contains following information:
•
Domain: URL name or IP address of the domain
•
Connections: Number of connections to the domain
•
Bytes: Amount of data transferred through the domain
 
To view the list of URLs for the selected domain, user and category type drill down by clicking the domain name hyperlink in the table
Top Contents widget
Widget report displays number of connections and amount of data transferred content wise for the selected user and category type.
 
View the report from Reports → Web Usage → Top Category Types→ Category Type → Top Users widget → User.
 
Bar graph displays amount of data transferred content wise while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of domains for the selected content, user and category type drill down by clicking the content type hyperlink in the table.
