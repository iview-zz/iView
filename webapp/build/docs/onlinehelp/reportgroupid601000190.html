Reports by File Name and Host IP Address
To view the following granular reports of a particular host and file name, go to Reports → Application Control → Top File Name → File Name → Top Host IP Address widget and drill down by clicking host IP in the graph or the hyperlink in the table:
•
Top Action Taken
•
Top Program
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Action Taken widget
Widget report displays number of connections per action taken by the selected host and file.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top File Names widget→ File Name.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product.Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of programs for the selected host, file name and action, click the action in the graph or hyperlink in the table.
Top Program widget
Widget report displays number of connections per program by the selected host and file name.
 
View the report from Reports → Application Control → Top Host IP Address→ Host IP Address→ Top File Names widget→ File Name.
 
Bar graph displays number of connections per program while tabular report contains following information:
•
Programs: Name of the program
•
Connections: Number of connections to the program
 
To view the list of actions for the selected host, file name and program, click the program in the graph or hyperlink in the table.
