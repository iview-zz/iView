Trend Reports : UTM Trend : Clean FTP
Clean FTP
 
Clean FTP trend report provides statistics based on uninfected file transfer though and from the network. The report displays number of clean FTP events for each time interval.
 
View the report from Trend Reports → UTM Trend →Clean FTP.
 
Bar graph displays number of events per time interval while tabular report contains following information:
•
Time: Time of the event in YYYY-MM-DD HH:MM::SS format
•
Event Type: Type of event
•
Event: Number of events
