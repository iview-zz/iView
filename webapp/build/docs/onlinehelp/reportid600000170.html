Reports for EPS Product Category : File Antivirus : Top Action Taken
Top Action Taken
Report displays list of actions along with the number of connections. Report can be drilled down for actions to view top application, top hosts and top path of application.
 
View the report from Reports → File Antivirus → Top Action Taken.
Report is displayed as graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the antivirus. Possible actions are:
o
File Quarantine
o
Access Denied
•
Connections: Number of connections to the action
 
To view the following granular reports of an action, drill down by clicking action in the graph or the hyperlink in the table:
•
Top Application
•
Top Path of Application
•
Top Host
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
