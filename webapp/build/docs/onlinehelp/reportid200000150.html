Detailed Report by SSL VPN User and Web Access User
Report displays list of resource(s) access through by the user using web access mode of SSL VPN and type of the resource.
View report from Reports → SSL VPN →Top SSL VPN Users → Top Web Access Users widget → User Name→ Source IP Address.
 
Tabular report contains following information:
•
Resource URL/IP: URL name or IP address of the accessed resource(s)
•
Resource Type: Type of access resource .Available types:
o
HTTP
o
HTTPS
o
RDP
o
TELNET
o
SSH
o
FTP
