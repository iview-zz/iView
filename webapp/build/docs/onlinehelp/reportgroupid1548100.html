Reports for UTM Product Category : Source Host Based Usage Report : Reports by Application Group, Destination (Upload) and User (Upload)
Reports by Application Group, Destination (Upload) and User (Upload)
To view the following reports of a particular user, destination and application group go to Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Destinations (Upload) widget →Destination →Top Users (Upload) widget →User.
•
Top Application (Upload)
•
Top Host (Upload)
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Application (Upload) widget
Widget report displays amount of data uploaded and number of connections per application by the selected destination, user and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Destinations (Upload) widget →Destination →Top Users (Upload) widget →User.
 
Bar graph displays amount of data uploaded by each application while tabular report contains following information:
•
Application: Displays name of the application as defined in Cyberoam iView. If application is not defined in Cyberoam iView then this field will display application identifier as combination of protocol and port number. To define unidentified application and group unassigned application please refer to Add Custom Application
•
Connections: Number of connections to the application
•
Bytes: Amount of data uploaded
Top Host (Upload) widget
Widget report displays amount of data uploaded and number of connections per host by the selected user, destination and application group.
 
View the report from Reports → Source Host Based Usage → Top Application Groups → Application Group → Top Destinations (Upload) widget →Destination →Top Users (Upload) widget →User.
 
Bar graph displays amount of data uploaded by each host while tabular report contains following information:
•
Host: IP address of the host
•
Connections: Number of connections to the host
•
Bytes: Amount of data uploaded
