Reports for UTM Product Category : Web Usage : Reports by User Group, Host and Application
Reports by User Group, Host and Application
To view the following granular reports of a particular user group, host and application, go to Reports → Web Usage → Top Web User Groups → User Group → Top Hosts widget → Host → Top Applications widget and drill down by clicking the application name hyperlink in the table.
•
Top Categories
•
Top Contents
The granular reports page displays multiple reports in the widgets which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date . Report date can be changed from the top most row of the page.
Top Categories widget
Widget report displays amount of data transferred and number of connections per category by the selected application, host and user group.
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Hosts widget → Host → Top Applications widget → Application.
 
Bar graph displays amount of data transferred by each category while tabular report contains following information:
•
Category: Displays name of the category as defined in monitored device. If category is not defined in the monitored device then this field will display ‘None’ at place of category name
•
Connections: Number of connections to the category
•
Bytes: Amount of data transferred through the category
 
To view the list of domains from where the data is transferred through the selected application, host and user group, drill down by clicking the application name hyperlink in the table.
Top Contents widget
Widget report displays amount of data transferred and number of connections per content type by the selected application, host and user group.
 
View the report from Reports → Web Usage → Top Web User Groups → User Group → Top Hosts widget → Host → Top Applications widget → Application.
 
Bar graph displays amount of data transferred through each content type while tabular report contains following information:
•
Content: Type of the content e.g. text, audio, video etc
•
Connections: Number of connections to the content type
•
Bytes: Amount of data transferred
 
To view the list of domains through which the data is transferred by the selected content type, application, host and user group, drill down by clicking the content type hyperlink in the table.
