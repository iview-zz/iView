Reports for EPS Product Category : USB Control : Reports by USB Serial Number and User
Reports by USB Serial Number and User
 
To view the following granular reports of a particular user and USB serial number, go to Reports → USB Control → Top USB Serial Number → USB Serial Number → Top Users widget and drill down by clicking user in the graph or the hyperlink in the table:
•
Top Host IP Addresses
•
Action Taken
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Host IP Addresses widget
Widget report displays number of connections per host by the selected USB serial number and user.
 
View the report from Reports → USB Control → Top USB Serial Number→ USB Serial Number→ Top Users widget→ User.
 
Bar graph displays number of connections per host while tabular report contains following information:
•
Host IP Address: IP Address of the host.
•
Connections: Number of connections to the host
 
To view the list of actions taken for the selected host, user and USB serial number, click the host in the graph or hyperlink in the table.
Action Taken widget
Widget report displays number of connection per action by the selected USB serial number and user.
 
View the report from Reports → USB Control → Top USB Serial Number→ USB Serial Number→ Top Users widget→ User.
 
Bar graph displays number of connections per action while tabular report contains following information:
•
Action Taken: Action taken by the EPS product. Possible actions
o
Allowed
o
Blocked
•
Connections: Number of connections to the action
 
To view the list of host IP addresses for the selected user, action and USB serial number, click the action taken in the graph or hyperlink in the table.
