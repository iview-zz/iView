Date wise Break-up by User Group and User
Report displays date wise break-up of surfing time.
 
View the report from Reports→ Internet Usage → Top Surfing Time → Top User Groups widget → User Group→Top Users → User Name.
 
Bar graph displays date wise surfing time while tabular report contains following information:
•
Date: Time stamp in YYYY-MM-DD HH:MM:SS format
•
Used Time: Total surfing time in HH:MM:SS format
 
To view detail report for the selected date, user and user group, drill down by clicking the Date Name hyperlink in the table.
