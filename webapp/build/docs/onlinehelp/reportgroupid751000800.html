Reports for Access Gateway Product Category : Net Capture Report : Reports by Destination Port and Destination
Reports by Destination Port and Destination
To view the following granular reports of a particular destination port and destination, go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget and drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Users
•
Top Clientips
•
Top Liveips
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Users widget
Widget report displays number of connections per user by the selected destination port and destination.
.
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget→ Destination.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If user name is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user.
•
Connections: Number of connections
 
To view the following granular reports for selected destination port, destination and user drill down by clicking the username in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
Top Clientips widget
Widget report displays number of connections per client IP by the selected destination and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget→ Destination.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the list of MAC addresses for selected client IP, destination and destination port, drill down by clicking the client IP address in the graph or hyperlink in the table
Top Liveips widget
Widget report displays number of connections per live IP by the selected destination and destination port.
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Destinations widget→ Destination.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the following granular reports for selected destination port, destination and live IP drill down by clicking the live IP in the graph or hyperlink in the table.
•
Top Clientip
•
Top Users
