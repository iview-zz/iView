Reports for UTM Product Category : Blocked Web Attempts : Top Denied Web Hosts
Top Denied Web Hosts
Report displays a list of top denied Web hosts based on the categories and number of blocked sites users tried to access through that hosts.
 
View the report from Reports → Blocked Web Attempts →Top Denied Web Hosts.
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays list of hosts while tabular report contains following information:
•
Host: Name of the Host
•
Connections: Number of Connections
 
To view the following granular reports of a particular host, drill down by clicking the bar against host name in the graph or the host name hyperlink in the table:
•
Top Categories
•
Top Domains
•
Top URLs
•
Top Users
•
Top Applications
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
