Reports by Mail Virus and Application
To view the following granular reports of a particular mail virus and application go to Reports → Virus → Mail Viruses → Virus → Top Applications widget and drill down by clicking the application in the graph or hyperlink in the table.
•
Top Senders
•
Top Recipients
•
Top Sender Hosts
•
Top Receiver Host
•
Top Users
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
 
Each widget displays report in graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
Top Senders widget
Widget report displays number of connections per sender by the selected mail virus and application.
 
View report from Reports → Virus → Mail Viruses → Virus → Top Applications widget → Application.
 
Bar graph displays number of connections per sender while tabular report contains following information:
•
Sender: Email address of the sender
•
Connections: Number of connections to the sender
 
To view the sender details for the selected sender, application and mail virus, drill down by clicking the sender in the graph or hyperlink in the table.
Top Recipients widget
Widget report displays number of connections per recipient by the selected mail virus and application.
 
View report from Reports → Virus → Mail Viruses → Virus → Top Applications widget → Application.
 
Bar graph displays number of connections per recipient while tabular report contains following information:
•
Recipient: Email address of the recipient
•
Connections: Number of connections to the recipient
 
To view the recipient details for the selected sender, application and mail virus, drill down by clicking the recipient in the graph or hyperlink in the table.
Top Sender Hosts widget
Widget report displays number of connections per host (sender) by the selected mail virus and application.
 
View report from Reports → Virus → Mail Viruses → Virus → Top Applications widget → Application.
 
Bar graph displays number of connections per host(sender) while tabular report contains following information:
•
Sender Host: IP address of the sender host
•
Connections: Number of connections to the sender host
 
To view the sender host details for the selected sender host, application and mail virus, drill down by clicking the sender host in the graph or hyperlink in the table.
Top Receiver Hosts widget
Widget report displays number of connections per host (receiver) by the selected mail virus and application.
 
View report from Reports → Virus → Mail Viruses → Virus → Top Applications widget → Application.
 
Bar graph displays number of connections per host(sender) while tabular report contains following information:
•
Receiver Host: IP address of the receiver host
•
Connections: Number of connections to the receiver host
 
To view the receiver host details for the selected receiver host, application and mail virus, drill down by clicking the receiver host in the graph or hyperlink in the table.
Top Users widget
Widget report displays number of connections per user by the selected mail virus and application.
 
View report from Reports → Virus → Mail Viruses → Virus → Top Applications widget → Application.
 
Bar graph displays number of connections per user while tabular report contains following information:
•
User: Username of the user as defined in the monitored device. If username is not defined in the monitored device then it will be considered as traffic generated by ‘Unknown’ user
•
Connections: Number of connections to the application
 
To view the user details for the selected user, application and mail virus, drill down by clicking the username in the graph or hyperlink in the table.
