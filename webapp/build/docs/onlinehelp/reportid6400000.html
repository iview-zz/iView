Reports for UTM Product Category : FTP Usage : Top FTP Users (Download)
Top FTP Users (Download)
Report displays the list of top users who have generated maximum traffic by downloading data through FTP protocol with number of connections.
 
View the report from Reports → FTP Usage Reports→Top FTP Users (Download).
 
Report is displayed using a graph as well as in tabular format.
By default, the report is displayed for the current date and all the devices. Report date or devices can be changed from the top most row of the page.
 
Bar graph displays amount of data downloaded by each user while tabular report contains following information:
•
User: Name of the user who downloaded the files. N/A will be displayed instead of username if username is not defined in the monitored device. If more than one such user exists then traffic details of all such users will be grouped and displayed under N/A.
•
Connections: Number of Connections
•
Bytes: Amount of data transferred
 
To view the following granular reports of a particular user, drill down by clicking the bar against user name in the graph or the user name hyperlink in the table:
•
Top Files
•
Top Hosts
•
Top Servers
 
The granular reports page displays multiple reports in the widgets, which can again be drilled down to view the detailed report.
