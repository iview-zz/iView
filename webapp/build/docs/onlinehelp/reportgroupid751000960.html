Reports by Destination Port and User
To view the following granular reports of a particular destination port and user, go to Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget and drill down by clicking the user name in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
•
Top MAC Addresses
•
Top Destinations
 
The granular reports page displays multiple reports in the widgets form, which can again be drilled down to view the detailed report.
Top Clientips widget
Widget report displays number of connections per client IP by the selected user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ User.
 
Bar graph displays number of connections per client IP while tabular report contains following information:
•
Client IP: IP address of the client machine
•
Connections: Number of connections per client IP address
 
To view the following granular reports for selected destination port, user and client IP drill down by clicking the client in the graph or hyperlink in the table:
•
Top MAC Addresses
•
Top Destinations
Top Liveips widget
Widget report displays number of connections per live IP by the selected user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ user.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
Live IP: Live IP address
•
Connections: Number of connections per live IP address
 
To view the following granular reports for selected destination port, user and live IP drill down by clicking the live IP in the graph or hyperlink in the table.
•
Top Clientips
•
Top Destinations
Top MAC Addresses widget
Widget report displays number of connections per MAC address by the selected user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ user.
 
Bar graph displays number of connections per live IP while tabular report contains following information:
•
MAC Address: MAC address of the machine
•
Connections: Number of connections to the MAC address
 
To view the list of client IPs for selected MAC address, user and destination port, drill down by clicking the MAC address in the graph or hyperlink in the table.
Top Destinations widget
Widget report displays number of connections per destination by the selected user and destination port.
 
View the report from Reports → Net Capture → Top Destination Ports → Destination Port → Top Users widget→ user.
 
Bar graph displays number of connections per destination while tabular report contains following information:
•
Destination: IP address of the destination
•
Connections: Number of connections to the destination
 
To view the following granular reports for selected destination port, user and destination drill down by clicking the destination in the graph or hyperlink in the table.
•
Top Clientips
•
Top Liveips
